AutoScript TEM Toolkit

AutoScript TEM Toolkit is a Python package that provides data processing capabilities in selected areas of electron microscopy. This package is a part of the AutoScript software suite. For more information, see other parts of this reference manual, in particular the Microscope API package.

The toolkit contains four modules. Image-template matchers can be found in the module Template matchers. Help for computer vision tasks is provided through the Vision module. User interface related functions can be found in the User interface module. The Context managers module allows for automatic resource management.
