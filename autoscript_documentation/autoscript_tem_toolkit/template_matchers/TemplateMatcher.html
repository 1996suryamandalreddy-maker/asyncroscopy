TemplateMatcher

This is a base class for all image template matchers compliant with AutoScript TEM Toolkit.

Methods

match() This method matches the given template image within the given image.

Remarks

Users are encouraged to derive from this class and implement match() method in order to create a custom template matcher compliant with AutoScript Toolkit.

Examples

The following example shows how a custom template matcher can be implemented:


        class CustomTemplateMatcher(TemplateMatcher):
            def match(self, image: AdornedImage, template: AdornedImage) -> ImageMatch:
                # Add a custom match implementation here
                return ImageMatch(center=Point(0, 0))
