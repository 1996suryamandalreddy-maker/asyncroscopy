beam_modulator_context

Start the beam modulation inside the with block by calling start_modulation() and blanks it using stop_modulation() upon exit from the with block.

Syntax

beam_modulator_context(beam_modulator, duty_cycle, frequency)

Parameters

beam_modulator BeamModulator Microscope BeamModulator object
duty_cycle duty_cycle A fraction of one period in which a signal is active. Ranges from 0 to 1.
frequency frequency The frequency of the modulation in Hz.

Examples


        beam_modulator = microscope.optics.blanker.beam_modulator
        
        # Start beam pulse width modulation with the given duty cycle and frequency
        with beam_modulator_context(beam_modulator, duty_cycle=0.2, frequency=1e6):
            print(beam_modulator.is_modulation_on)  # True

        # Modulation stopped, stop_modulation() called upon exiting the context manager
        print(beam_modulator.is_modulation_on)  # False
