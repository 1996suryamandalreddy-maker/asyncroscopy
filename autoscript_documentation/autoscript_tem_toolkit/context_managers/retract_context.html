retract_context

The retract context manager withdraws the given mechanism from the microscope by calling retract() and inserts it again by calling insert(). The resource is retracted inside the context manager - i.e. retract_context.

Syntax

retract_context(retractable)

Parameters

retractable CameraDetector | ScanningDetector | Slit | Screen | ApertureMechanism A retractable object with a retract() and an insert() method.

Examples


        from autoscript_tem_toolkit.context_managers import retract_context

        slit = microscope.optics.energy_filter.slit
        slit.width.value = microscope.optics.energy_filter.slit.width.limits.min
        
        with retract_context(slit):
            unfiltered_image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 1024, 1)
        
        filtered_image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 1024, 1)
