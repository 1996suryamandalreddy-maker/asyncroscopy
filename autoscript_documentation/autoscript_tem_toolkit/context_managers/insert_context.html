insert_context

The insert context manager positions the given mechanism inside the microscope by calling insert() and retracts it again by calling retract(). The resource is inserted inside the context manager - i.e. insert_context.

Syntax

insert_context(insertable)

Parameters

insertable CameraDetector | ScanningDetector | Slit | Screen | ApertureMechanism An insertable object with an insert() and a retract() method.

Examples


        from autoscript_tem_toolkit.context_managers import insert_context

        camera = microscope.detectors.get_camera_detector(CameraType.BM_CETA)
        print(camera.insertion_state)

        with insert_context(camera):
            print(camera.insertion_state)
            image = microscope.acquisition.acquire_camera_image(camera.name, 1024, 1.0)
            plot.imshow(image.data, cmap='gray')
            plot.show()

        print(camera.insertion_state)
