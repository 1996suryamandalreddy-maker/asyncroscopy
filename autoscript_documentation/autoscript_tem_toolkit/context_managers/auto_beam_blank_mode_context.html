auto_beam_blank_mode_context

Returns the set auto beam blank mode to the default value after the with code block terminates by calling reset_to_default().

Syntax

auto_beam_blank_mode_context(mode)

Parameters

mode Mode Auto beam blank mode object.

Examples


        mode = microscope.optics.blanker.auto_beam_blank.mode

        with auto_beam_blank_mode_context(mode, AutoBeamBlankMode.OFF):
            # Auto beam blank set to Off inside the with block
            stem_image = microscope.acquisition.acquire_stem_image(DetectorType.HAADF, 1024, 1e-6)
            ...
        
        # On with block exit reset_to_default() is called, default value is restored.
