blanker_context

Unblanks the beam blanker inside the with block by calling blanker.unblank() and blanks it using blanker.blank() upon exit from the with block.

Syntax

blanker_context(blanker)

Parameters

blanker Blanker Microscope blanker object

Examples


        # blanked
        
        with blanker_context(microscope.optics.blanker):
            # unblanked
            images = microscope.acquisition.acquire_stem_images_advanced(settings)
            ...
        
        # blanked again
