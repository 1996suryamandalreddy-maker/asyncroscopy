Context managers

Methods

blanker_context() Unblanks and re-blanks the beam.
insert_context() Inserts a given insertable and retracts after the with code block terminates.
retract_context() Retracts a given retractable and re-inserts after the with code block terminates.
auto_beam_blank_mode_context() Resets set mode to default after the with code block terminates.
beam_modulator_context() Resets modulation settings to default after the with code block terminates.

Remarks

This module provides a set of context managers to automatically manage a given resource using Python's with statement. The given resource is managed by automatically calling an initilization function on entry to the with code-block and a clean-up function when the with block terminates. The cleanup function is called even if the code inside raises an exception.

Examples

The following example demonstrates context manager syntax on the microscope's beam blanker. 


        # blanked here

        with blanker_context(microscope.optics.blanker): # calls .unblank() here
            # unblanked
            ...

        # blanked, .blank() was called
