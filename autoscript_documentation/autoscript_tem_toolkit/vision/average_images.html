average_images

Averages the pixel values of multiple images.

Syntax

average_images(images)

Parameters

images List[AdornedImage] List of images to average.

Return value

AdornedImage A new image with the averaged pixel values.

Remarks

The images must have the same dimensions, encoding, and bit depth. Only grayscale images are supported. Each pixel that exceeds the maximum value of the data type is set to the maximum value.

The given images are not modified. The result is a new image with the same dimensions, encoding, and bit depth as the input images. The result doesn't contain any metadata.

Examples

The following example shows how to create average image from the STEM segment ring:


        # Configure the BF-S detector and enable the whole inner ring (4 segments)
        detector = microscope.detectors.get_scanning_detector(DetectorType.BF_S)
        detector.set_enabled_segments(DetectorSegmentType.INNER_RING)
        segments = detector.get_enabled_segments()

        # Run the acquisition and average the recieved images
        inner_ring_images = microscope.acquisition.acquire_stem_segment_images(segments, ImageSize.PRESET_512, 1e-6)
        inner_ring_average = vision_toolkit.average_images(inner_ring_images)

        # Show the result
        pyplot.imshow(inner_ring_average.data, cmap="gray")
        pyplot.show()
