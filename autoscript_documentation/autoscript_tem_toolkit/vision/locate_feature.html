locate_feature

Finds the position of the specified template image in an image.

Syntax


        locate_feature(searched_image, feature_template, [template_matcher, original_feature_center])

Parameters

searched_image AdornedImage Image to be searched for feature.
feature_template AdornedImage Template of the feature to be located.
template_matcher TemplateMatcher Template matcher to be used when matching template against the searched image. See the Remarks section for more information.
original_feature_center Point or a point-like object. Center of the original feature location, in pixels. The information is used for shift calculation.

Return value

FeatureLocation A structure with details of the feature location.

Remarks

The optional template_matcher parameter can provide an object derived from the TemplateMatcher class. If not provided, the default template matcher OpencvTemplateMatcher is used. For more information, see the documentation for the Template matchers module.

The optional original_feature_center parameter serves as input for calculating the image shift. If provided, the resulting FeatureLocation structure will contain the feature shift data. When not provided, only the current feature position is returned.

The result contains properties in meters (center_in_meters, shift_in_meters) only if the given image contains pixel size metadata.

Examples

The following example shows how to use the locate feature functionality with various template matchers:


        # Acquire a full frame image and prepare a feature template
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)
        template = vision_toolkit.cut_image(image, relative_center=[0.40, 0.40], relative_size=[0.20, 0.20])

        # Locate the feature in the original image using the default (OpenCV based) template matcher
        opencv_location = vision_toolkit.locate_feature(image, template)

        # Locate the feature in the original image using the Histogram of Gradients template matcher
        hog_location = vision_toolkit.locate_feature(image, template, HogMatcher(microscope))

        # Inspect the results
        opencv_location.print_all_information()
        hog_location.print_all_information()

        # Plot the results using the plot_match function
        ui_toolkit.plot_match(image, template, opencv_location.center_in_pixels)
        ui_toolkit.plot_match(image, template, hog_location.center_in_pixels)
