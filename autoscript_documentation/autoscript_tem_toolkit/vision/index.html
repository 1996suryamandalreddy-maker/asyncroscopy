Vision

This module provides aid for computer vision tasks.

Classes

FeatureLocation This class contains various information about a feature located in an image.

Methods

add_scale_bar() Adds a scale bar to the lower right corner of the image.
average_images() Averages the pixel values of multiple images.
cut_image() Cuts out a portion of an image according to the specified target area definition.
draw_point() Draws a point in the given image.
draw_points() Draws multiple points in the given image.
draw_rectangle() Draws a rectangle in the given image.
draw_rectangles() Draws multiple rectangles in the given image.
locate_feature() Finds the position of the specified template image in an image.
plot_match() Display a searched image and a feature located in it in a popup window.
plot_image() Display and label an acquired image in a popup window.
plot_images() Display and label acquired images in a popup window.
plot_spectrum() Opens a pop-up window displaying a plot of the provided spectrum.
plot_spectra() Opens a pop-up window displaying a plot where the provided spectra are overlaid in a single figure.
select_point() Select a point of interest (pixel) by clicking on the given image.
select_points() Select multiple points of interest (pixels) by clicking on the given image.
select_rectangle() Select an area of interest by dragging a rectangle on the given image.
select_rectangles() Select multiple areas of interest by dragging rectangles on the given image.
stretch_histogram() Normalizes and stretches the histogram of the given grayscale image. Removes outliers if the threshold is provided.

Remarks

This module uses the template_matchers module for image template matching purposes.

Examples

Import vision package from AutoScript Toolkit:


        import autoscript_tem_toolkit.vision as vision_toolkit
