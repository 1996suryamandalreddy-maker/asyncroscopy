draw_points

Draws multiple points in the given image.

Syntax

draw_points(image, points)

Parameters

image AdornedImage Input image.
points List[Point | list | tuple] Points coordinates.

Return value

AdornedImage A new image.

Remarks

The points are drawn in sequence, meaning the later ones may overlap or cover the earlier points.

Examples

The following example shows how to draw multiple points of interest in the acquired image:


        # Acquire a full frame image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)

        # Draw points of interest
        new_image = vision_toolkit.draw_points(image, [(50, 50), (100, 100), (250, 250)])
