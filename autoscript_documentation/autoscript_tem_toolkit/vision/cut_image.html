cut_image

Cuts out a portion of an image according to the specified target area definition.

Syntax

cut_image(image, [top_left, bottom_right, center, size, relative_center, relative_size])

Parameters

image AdornedImage or numpy.ndarray Image to cut out a part from.
top_left Point or a point-like object. Top left point of the cut area, in pixels.
bottom_right Point or a point-like object. Bottom right point of the cut area, in pixels.
center Point or a point-like object. Center of the cut area, in pixels.
size Point or a point-like object. Size of the cut area, in pixels.
relative_center Point or a point-like object. Center of the cut area, in relative coordinates in the range [0, 1].
relative_size Point or a point-like object. Size of the cut area, in relative coordinates in the range [0, 1].

Return value

AdornedImage or numpy.ndarray Cut taken from the image in the same format as the input image.

Remarks

Please note that only the following combinations of parameters are valid: top left and bottom right, center and size, or relative center and relative size.

The method can accept both AdornedImage and numpy.ndarray for image parameter. The cut is then provided in the same form.

Examples

The following example shows cutting the image with various parameter combinations:


        # Acquire a full frame image and prepare a feature template
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)

        # Cutting the image with the top left and bottom right parameters
        # The result is a top left quarter of the image
        cut_result = vision_toolkit.cut_image(image, top_left=Point(0, 0), bottom_right=Point(2048, 2048))

        # Cutting the image with the center and size parameters
        # The result is a 100x100 px crop centered around the [256, 256] coordinates
        cut_result = vision_toolkit.cut_image(image, center=Point(256, 256), size=Point(100, 100))

        # Cutting the image with the relative center and relative size parameters
        # The result is a half of the image taken from the center
        cut_result = vision_toolkit.cut_image(image, relative_center=Point(.5, .5), relative_size=Point(.5, .5))
