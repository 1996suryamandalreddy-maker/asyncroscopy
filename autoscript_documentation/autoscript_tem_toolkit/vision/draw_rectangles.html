draw_rectangles

Draws multiple rectangles in the given image.

Syntax

draw_rectangles(image, rectangles)

Parameters

image AdornedImage Input image
rectangles List[Rectangle | list | tuple] Rectangle coordinates.

Return value

AdornedImage A new image.

Remarks

The rectangles are drawn in sequence, meaning the later ones may overlap or cover the earlier rectangles.

Examples

The following example shows how to draw multiple areas of interest in the acquired image:


        # Acquire a full frame image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)

        # Draw areas of interest
        new_image = vision_toolkit.draw_rectangles(image, [(50, 50, 120, 50), (15, 20, 20, 20)])
