plot_spectrum

Opens a pop-up window displaying a plot of the provided spectrum, given as an AdornedSpectrum instance.
[The method is deprecated and will be removed in version 1.19]

Syntax

plot_spectrum(spectrum, label)

Parameters

spectrum AdornedSpectrum Spectrum to be plotted.
label [Optional] str Label of the plot. If None, the label is automatically generated.

Remarks

This method has been moved to the user_interface module.

The function converts the raw data to primary electron counts and scale and offset corrected intensity values. For this conversion to be possible, the spectrum has to have valid metadata. If the metadata is not valid, an error is raised.

The plot is interactive and allows the user to zoom in and out, pan, and save the plot as an image file.

Examples

The following example shows how to plot acquired EDS spectrum with a custom label:


        # Get the first EDS detector
        eds_detector_name = microscope.detectors.eds_detectors[0]
        eds_detector = microscope.detectors.get_eds_detector(eds_detector_name)

        # Configure the acquisition
        settings = EdsAcquisitionSettings()
        settings.eds_detector = eds_detector_name
        settings.dispersion = eds_detector.dispersions[-1]
        settings.shaping_time = eds_detector.shaping_times[-1]
        settings.exposure_time = 3e-3
        settings.exposure_time_type = ExposureTimeType.LIVE_TIME

        # Run the spectrum acquisition
        spectrum = microscope.analysis.eds.acquire_spectrum(settings)

        # Plot the spectrum
        vision_toolkit.plot_spectrum(spectrum, "EDS spectrum")
