draw_rectangle

Draws a rectangle in the given image.

Syntax

draw_rectangle(image, rectangle)

Parameters

image AdornedImage Input image
rectangle Rectangle | list | tuple Rectangle coordinates.

Return value

AdornedImage A new image.

Remarks

All combinations of encoding and bit depth valid for AdornedImage are supported. The input image has minimum size of 256x256 pixels. The function returns a copy of the image, the original object is not changed.

Examples

The following example shows how to draw an area of interest in the acquired image:


        # Acquire a full frame image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)

        # Draw an area of interest
        new_image = vision_toolkit.draw_rectangle(image, (50, 50, 120, 50))
