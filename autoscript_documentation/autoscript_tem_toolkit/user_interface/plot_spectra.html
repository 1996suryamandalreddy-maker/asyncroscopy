plot_spectra

Opens a pop-up window displaying a plot where the provided spectra, given as AdornedSpectrum instances, are overlaid in a single figure.

Syntax

plot_spectra(spectra, labels)

Parameters

spectra List[AdornedSpectrum] Spectra to be plotted.
labels [Optional] List[str] Labels of the plots. If None, the labels are automatically generated.

Remarks

The function converts the raw data to primary electron counts and scale and offset corrected intensity values. For this conversion to be possible, the spectra have to have valid metadata. If the metadata are not valid, an error is raised. The number of labels can't be greater than the number of spectra, if any labels are missing, they are automatically generated.

The plot is interactive and allows the user to zoom in and out, pan, and save the plot as an image file.

Examples

The following example shows how to plot acquired EDS spectra with custom labels:


        # Get the first EDS detector
        eds_detector_name = microscope.detectors.eds_detectors[0]
        eds_detector = microscope.detectors.get_eds_detector(eds_detector_name)

        spectra = []
        labels = []

        # Acquire three spectra with different dispersions and shaping times
        for i in range(3):
            # Configure the acquisition
            settings = EdsAcquisitionSettings()
            settings.eds_detector = eds_detector_name
            settings.dispersion = eds_detector.dispersions[i]
            settings.shaping_time = eds_detector.shaping_times[i]
            settings.exposure_time = 3e-3
            settings.exposure_time_type = ExposureTimeType.LIVE_TIME

            # Run the spectrum acquisition
            spectra.append(microscope.analysis.eds.acquire_spectrum(settings))
            labels.append(f"EDS spectrum {i + 1}")

        # Plot the spectra
        ui_toolkit.plot_spectra(spectra, labels)

The next example shows how to plot acquired EELS spectra:


        # EELS acquisition settings with multiple exposures
        settings = EelsAcquisitionSettings(
            eels_detector=EelsDetectorType.ZEBRA,
            eels_exposure_settings=[
                EelsExposureSettings(exposure_time=1),
                EelsExposureSettings(exposure_time=5),
                EelsExposureSettings(exposure_time=10)
        ])

        # Run the acquisition
        spectra = microscope.analysis.eels.acquire_spectra(settings)

        # Plot the spectra with default labels
        ui_toolkit.plot_spectra(spectra)
