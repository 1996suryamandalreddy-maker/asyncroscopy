select_rectangle

Select an area of interest by dragging a rectangle on the given image. It can use either pixel or relative coordinates.

Syntax

select_rectangle(image)

Parameters

image AdornedImage Input image
coordinate_system [Optional] RegionCoordinateSystem Coordinate system to use when selecting the area. Pixels by default.

Return value

[Optional] Rectangle Selected area, if it was selected. None otherwise.

Remarks

The function displays the provided image using Matplotlib and prompts the user to drag a rectangular area on it. Once the user completes dragging the rectangle, the image closes, and the function returns the rectangle's coordinates. The coordinates can be either in pixels or relative to the image size, depending on the coordinate system parameter.
