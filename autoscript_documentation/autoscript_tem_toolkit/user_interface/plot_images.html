plot_images

Opens a pop-up window displaying a plot of the provided images, given as AdornedImage instances. The images are rendered using a grayscale palette.

Syntax

plot_images(images, labels)

Parameters

images List[AdornedImage] Images to be plotted.
labels [Optional] List[str] Labels of the plots. If None, the labels are automatically generated.

Remarks

The number of labels can't be greater than the number of spectra, if any labels are missing, they are automatically generated.

The plot is interactive and allows the user to zoom in and out, pan, and save the plot as an image file.

Examples

The following example shows how to plot acquired images with a custom label:


        # Acquire 5 full frame images
        images = microscope.acquisition.acquire_camera_series(CameraType.BM_CETA, ImageSize.PRESET_4096, exposure_time=1, length=5)

        # Display the images
        ui_toolkit.plot_images(images, ["1", "2", "3", "4", "5"])
