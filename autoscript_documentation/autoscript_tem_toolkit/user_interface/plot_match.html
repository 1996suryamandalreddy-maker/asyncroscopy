plot_match

Display a searched image and a feature located in it in a popup window.

Syntax

plot_match(image, template, match_center)

Parameters

image AdornedImage Searched image in which a feature was located.
template AdornedImage Template image used for feature location.
match_center Point or a point-like object. Point referring to the center of the feature located within the searched image, in pixels.

Remarks

The function can be used together with the locate_feature() function to debug or check the results.

See also

locate_feature()
