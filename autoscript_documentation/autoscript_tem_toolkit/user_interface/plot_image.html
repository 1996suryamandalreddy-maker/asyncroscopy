plot_image

Opens a pop-up window displaying a plot of the provided image, given as an AdornedImage instance. The image is rendered using a grayscale palette.

Syntax

plot_image(image, label)

Parameters

image AdornedImage Image to be plotted.
label [Optional] str Label of the plot. If None, the label is automatically generated.

Remarks

The plot is interactive and allows the user to zoom in and out, pan, and save the plot as an image file.

Examples

The following example shows how to plot an acquired image with a custom label:


        # Acquire a full frame image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, exposure_time=1)

        # Display the image
        ui_toolkit.plot_image(image, "BM-CETA full frame image")
