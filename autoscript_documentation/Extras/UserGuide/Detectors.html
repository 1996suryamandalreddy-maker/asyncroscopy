User guide: Detectors

This chapter gives an introduction to controlling the detectors with AutoScript. For the complete reference of detector calls, see detectors section of the reference manual.

Before proceeding with the following code snippets, make sure to initialize AutoScript.

Main screen

The main screen is the main fluorescent screen, which is also used by the SmartCam or Flucam.

The screen can be inserted, retracted and the current can be measured.

To print the screen current in pA: 


        microscope.detectors.screen.insert()
        print(str(microscope.detectors.screen.measure_current() * 1E12) + " pA")
        
        > 1149.4356973629922 pA

Camera detectors

Available camera detectors are exposed as a list of camera names. The names can be used in acquisition calls to select a camera. 


        print(microscope.detectors.camera_detectors)
        
        > ['BM-Ceta', 'Flucam']

Scanning detectors

Available scanning detectors are exposed as a list of scanning detector names.

The example below show how to gain access to the BF-S detector object, set the gain in dB and get the collection angle range: 


        print(microscope.detectors.scanning_detectors)

        > ['BF-S', 'DF-S', 'HAADF']

        bfs = microscope.detectors.get_scanning_detector("BF-S")
        bfs.gain = 0.5

        print(bfs.collection_angle_limits)

        > Limits(min=0,max=0.2)

EDS detectors

Available EDS detectors are exposed as a list of names. 


        available_detectors = microscope.detectors.eds_detectors
        print(available_detectors)

EELS detectors

Available EELS detectors are exposed as a list of names. 


        available_detectors = microscope.detectors.eels_detectors
        print(available_detectors)
