User guide: Optics

This chapter provides an introduction to optics control with AutoScript. For the complete reference of optics calls, see the Optics section of the reference manual.

Magnification

TEM

In TEM mode, magnifications and camera lengths are exposed as lists in optics.magnification and optics.camera_length. Magnifications and camera lengths have a nominal value, calibrated value, rotation and a label. The label is a complete description of the camera length or magnification and can be used to set the projector directly. 


        print(microscope.optics.magnification.value.label)
        print(microscope.optics.magnification.value.calibrated)
        print(microscope.optics.magnification.value.nominal)
        print(microscope.optics.magnification.value.rotation)

        > SA 8600 x
        > 9030.0
        > 8609.428
        > 4.71238898038469

To print labels of all magnifications: 


        for mag in microscope.optics.magnification.available_values:
        print(mag.label)

        > Mi 2050 x 
        > Mi 2600 x 
        > Mi 3300 x 
        > SA 4300 x 
        > SA 5500 x 
        ...

To set the projector from a label:


        microscope.optics.set_projector("SA 5500 x")

STEM

In STEM mode, the field of view can be controlled. The displayed magnification is by convention 0.1 / field of view. To set a field of view of 0.5 µm (or 20 000 x): 


        microscope.optics.scan_field_of_view = 0.5e-6

If necessary, the field of view will be clipped to the valid range.

Focus

The focus object can be used to adjust the focus and defocus of the microscope. The (de)focus value is given in meters in STEM or imaging mode, and is unitless in diffraction mode.

Setting the focus to 0.0 is equivalent to setting the eucentric focus: 


        microscope.optics.focus.value = 0.0

You can get and set the defocus as well.

The function reset_defocus() sets defocus to zero without changing the objective lens strength.

Apertures

The apertures can be controlled by interacting with the aperture mechanisms:


        microscope.optics.aperture_mechanisms.SA.insert()

Depending on the type of microscope, not all mechanisms may be present. To list all apertures in a particular mechanism: 


        for aperture in microscope.optics.aperture_mechanisms.SA.apertures:
            print(aperture.diameter)
        
        > 10.0
        > 40.0
        > 200.0
        > 800.0

To select the 800 µm aperture, select number three from the list (counting from zero):


        microscope.optics.aperture_mechanisms.SA.aperture = microscope.optics.aperture_mechanisms.SA.apertures[3]
