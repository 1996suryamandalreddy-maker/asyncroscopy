AutoScript TEM user guide

In this part of the reference manual you can explore the AutoScript functions by means of short example code snippets.

AutoScript must first be initialized before it can be used in a Python script. The following chapters explain how to import AutoScript packages and start an AutoScript client:

  • Client startup
