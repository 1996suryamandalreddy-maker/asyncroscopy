User guide: Client startup

This chapter shows how to start an AutoScript client and connect it to a microscope.

Before proceeding with code snippets below, import AutoScript packages using the following code:


    from autoscript_tem_microscope_client import TemMicroscopeClient
    from autoscript_tem_microscope_client.enumerations import *
    from autoscript_tem_microscope_client.structures import *

First, create a microscope client object:


    microscope = TemMicroscopeClient()

Connect the client to the AutoScript server running on the microscope computer with an IP address of 192.168.0.1:


    microscope.connect("192.168.0.1")

or connect the client to the server at the local computer:


    microscope.connect("localhost")

By default, the client attempts to connect to the standard address 192.168.0.1:


    microscope.connect()

Connect the client to the server with certain host name, e.g. MPC-TALOS-A2:


    microscope.connect("MPC-TALOS-A1")

To disconnect the client:


    microscope.disconnect()
