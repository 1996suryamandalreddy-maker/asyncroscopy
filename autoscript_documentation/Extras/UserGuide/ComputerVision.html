User guide: Vision

This chapter shows how to utilize vision capabilities of AutoScript TEM Toolkit add-on.

Before proceeding with the code snippets listed below, initialize AutoScript TEM with the following code. Do not forget to set the server address or host name (parameters of the connect() function) according to your environment. 


        from autoscript_tem_microscope_client import TemMicroscopeClient
        from autoscript_tem_microscope_client.enumerations import *
        from autoscript_tem_microscope_client.structures import *

        microscope = TemMicroscopeClient()
        microscope.connect("localhost")

Import vision package from AutoScript Toolkit:


        import autoscript_tem_toolkit.vision as vision_toolkit

Cut out image region using relative coordinates.


        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)

        region = vision_toolkit.cut_image(image, relative_center=[0.40, 0.40], relative_size=[0.20, 0.20])

        ui_toolkit.plot_image(image)
        ui_toolkit.plot_image(region)

Cut out image region using pixel coordinates:


        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)

        region1 = vision_toolkit.cut_image(image, center=[140, 96], size=[120, 120])
        region2 = vision_toolkit.cut_image(image, top_left=[80, 36], bottom_right=[200, 156])

        ui_toolkit.plot_image(image)
        ui_toolkit.plot_image(region1)
        ui_toolkit.plot_image(region2)

Locate feature in the image using default template matcher and inspect the result:


        # Acquire a full frame image and cut out a template from it
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)
        template = vision_toolkit.cut_image(image, relative_center=[0.40, 0.40], relative_size=[0.20, 0.20])

        # Locate feature from the template in the original image
        location = vision_toolkit.locate_feature(image, template)
        location.print_all_information()

        ui_toolkit.plot_match(image, template, location.center_in_pixels)

Locate feature in the image using specialized HOG (histogram of gradients) template matcher and inspect the result:


        from autoscript_tem_toolkit.template_matchers import *

        # Acquire a full frame image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_4096, 1)
        
        # Acquire an image with a reduced size
        template = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_256, 1)
        
        # Locate feature from the template in the original image
        hog_matcher = HogMatcher(microscope)
        location = vision_toolkit.locate_feature(image, template, hog_matcher)
        location.print_all_information()

        ui_toolkit.plot_match(image, template, location.center_in_pixels)
