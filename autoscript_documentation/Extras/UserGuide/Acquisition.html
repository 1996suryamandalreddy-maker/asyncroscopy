User guide: Image acquisition

This chapter shows examples of how to acquire images in AutoScript. It is assumed that the imports are done and the object 'microscope' has been connected. See Getting Started for details.

STEM Images

To acquire a STEM image from the HAADF detector of 1k x 1k pixels with a dwell time of 1 µs: 


        microscope.optics.optical_mode = OpticalMode.STEM
        haadf_image = microscope.acquisition.acquire_stem_image(DetectorType.HAADF, 1024, 1e-6)

The image that is returned is an 'Adorned Image'. It contains both the image data and metadata.

The image can be saved as a TIFF file by calling haadf_image.save(filename).

To acquire STEM images from multiple detectors at once, acquire_stem_images() should be called which returns a list of images, in the same order as the list of detectors was specified in the first parameter: 


        images = microscope.acquisition.acquire_stem_images([DetectorType.HAADF, DetectorType.BF], 1024, 1e-6)

Camera images

To acquire a 4k image with an exposure time of 1s, you can use acquire_camera_image() function:


        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 4096, 1.0)

Acquiring series

Image series can only be taken with cameras. The entire series must fit in memory and therefore the number of images (frames) is limited to a maximum of 64.

To acquire series of 32 4k images with the Ceta camera, with 2s exposure time each: 


        images = microscope.acquisition.acquire_camera_series(CameraType.BM_CETA, 4096, 2.0, 32)
