Getting started

This section will walk you through the process of creating and executing a test project with the AutoScript software.

It is assumed that the product has been installed in the recommended Remote scripting configuration.

On the Microscope computer:

  1. Start the microscope server.

  2. Prepare the system for TEM image acquisition

    1. The vacuum system and electron beam gun are in an operational state.
    2. Specimen is in a position suitable for imaging.

  3. Check the taskbar tray to ensure that the AutoScript Server icon is blue.

    A blue icon indicates that the AutoScript Server is running and that all prerequisites for executing scripts are met. If you don't see the icon, launch the AutoScript Server from the Start menu by navigating to Thermo Scientific AutoScript > AutoScript Server.

On the Support computer

(or more specifically on the computer where the AutoScript Client software is installed.)

  1. Open PyCharm Community Edition.

  2. If PyCharm is started for the first time, then:

    1. Accept the JetBrains Privacy Policy: Tick I confirm… and select Continue.
    2. At the data sharing options, select Don't send.

  3. In the PyCharm welcome screen, select New Project.

  4. In Location, enter the folder name for the new project: "Test". This will also be the name of the new project.

    Keep the Python interpreter option set to the preselected "Python 3.11 (AutoScript)" option.

    Press Create.

    When creating the first project in PyCharm with the specific interpreter settings, it scans Python packages installed with that interpreter. This process is called indexing. During indexing, the editor's functionality is limited, and scripts cannot be executed.

    Wait for the indexing to finish. This may take up to 15 minutes.

  5. In the Project panel, right-click on the Test project and select "New > AutoScript File with Examples".

  6. Enter the filename for the new script: "test". The script file will be saved as test.py.

  7. In the script, locate the function

    microscope.connect()

    This function, when called with no parameters, will attempt to contact the AutoScript server at the default microscope computer IP address 192.168.0.1. If you intend to run the script locally on the microscope PC (you are in Local scripting configuration), modify the command to:

    microscope.connect("localhost")

  8. In the Project panel, right-click on test.py and select Run test.

  9. After closing the window with the acquired image, the "Example script finished successfully" appears in the output panel.
