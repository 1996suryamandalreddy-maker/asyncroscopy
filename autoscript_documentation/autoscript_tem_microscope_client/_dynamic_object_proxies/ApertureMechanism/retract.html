ApertureMechanism.retract

Retracts this aperture mechanism.

Syntax

retract()

Remarks

No aperture is selected after the aperture mechanism is retracted. Mechanism must be enabled and retractable to retract it.

Examples

Retract the SA aperture mechanism:


        microscope.optics.aperture_mechanisms.SA.retract()

Retracts aperture mechanism if it is retractable:


        if microscope.optics.aperture_mechanisms.C1.is_retractable:
            microscope.optics.aperture_mechanisms.C1.retract()
            print(microscope.optics.aperture_mechanisms.C1.insertion_state)
        else:
            print("Aperture mechanism is not retractable.")
