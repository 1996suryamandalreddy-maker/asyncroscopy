ApertureMechanism

Contains properties and enables control of an aperture mechanism.

Properties

aperture Gets or sets the selected aperture on this mechanism.
apertures [Read only] Returns available apertures on this mechanism.
insertion_state [Read only] The insertion state of the aperture mechanism. Maps to ApertureMechanismInsertionState enumeration.
is_enabled [Read only] Whether this aperture mechanism is currently enabled.
is_retractable [Read only] Whether this aperture mechanism is a retractable device.
position Gets or sets the X and Y position of the selected aperture. 2D vector in meters (x, y).

Methods

disable() Disables this aperture mechanism.
enable() Enables this aperture mechanism.
insert() Inserts this aperture mechanism.
reset_positions() Resets positions of all apertures on this mechanism to default values from microscope alignment.
retract() Retracts this aperture mechanism.

Remarks

An aperture mechanism is a representation of a physical motorized device that contains individual apertures (holes punched in a thin metal sheet). If the mechanism is retractable, it can be inserted into the column and retracted. Non-retractable mechanisms are always inserted.

  • The position property positions the aperture mechanism to a new location. This new location is associated with the currently selected aperture and can be recovered by re-selecting the aperture. The position property can therefore be thought of as a preset for the currently selected aperture.
  • The reset_positions() method restores these presets for all apertures in a mechanism to their default values. A given position cannot be reset individually.

See aperture_mechanisms and get_mechanism for information on how to create aperture mechanism dynamic objects.
