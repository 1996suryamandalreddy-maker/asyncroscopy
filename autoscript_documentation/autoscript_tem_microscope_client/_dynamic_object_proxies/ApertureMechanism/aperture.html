ApertureMechanism.aperture

Gets or sets the selected aperture on this mechanism.

Data type

Aperture

Remarks

An aperture has a pre-defined position where it is located. This location can be modified by changing the value of the position property with the given aperture selected. You can change the selected aperture only when the aperture mechanism is enabled. Setting the aperture property will insert the aperture mechanism that contains it into the column (if retracted). None is returned if no aperture is selected.

Examples

Select the second aperture on the SA aperture mechanism and inspect its name, type and diameter:


        # Select the second aperture
        apertures = microscope.optics.aperture_mechanisms.SA.apertures
        microscope.optics.aperture_mechanisms.SA.aperture = apertures[1]
        
        # Read and print aperture properties
        aperture = microscope.optics.aperture_mechanisms.SA.aperture
        print("Name: ", aperture.name)
        print("Type: ", aperture.type)
        print("Diameter:", aperture.diameter)
