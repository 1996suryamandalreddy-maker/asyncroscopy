ApertureMechanism.insert

Inserts this aperture mechanism.

Syntax

insert()

Remarks

  • When you insert the aperture mechanism, an aperture that was used before retraction is automatically selected. If the mechanism is inserted for the first time, the last aperture is selected.
  • The mechanism must be enabled and retractable to allow insertion.

Examples

Insert the SA aperture mechanism:


        microscope.optics.aperture_mechanisms.SA.insert()

Inserts the C1 aperture mechanism if it is retractable:


        if microscope.optics.aperture_mechanisms.C1.is_retractable:
            microscope.optics.aperture_mechanisms.C1.insert()
            print(microscope.optics.aperture_mechanisms.C1.insertion_state)
        else:
            print("Aperture mechanism is not retractable.")
