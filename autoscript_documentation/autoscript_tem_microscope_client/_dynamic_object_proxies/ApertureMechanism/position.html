ApertureMechanism.position

Gets or sets the X and Y position of the selected aperture. 2D vector in meters (x, y).

Data type

Get: Point
Set: Point | list | tuple

Remarks

Each aperture has a default position that is determined during microscope alignment. Use reset_positions() to restore positions to default values or to recover from an invalid aperture position.

When setting the value, the coordinates of the given point can be None. That means the position of this axis will remain the same. Getting the position can also return None, meaning that no aperture is currently selected.

Mechanism must be enabled and inserted to set or get the position.

Examples

Find out current position of the selected aperture in the C3 mechanism. C3 is assumed to be inserted and enabled.


        position = microscope.optics.aperture_mechanisms.C3.position
        print(position)

Move selected aperture by 1µm in X:


        # Get current position
        old_position = microscope.optics.aperture_mechanisms.C3.position
        print(old_position)

        # Move by 1um in X
        new_position = Point(old_position.x + 1e-6, old_position.y)
        microscope.optics.aperture_mechanisms.C3.position = new_position

        # Move by 1um in Y (value of X is None, that means no movement on this axis)
        new_position = Point(y=old_position.y + 1e-6)
        microscope.optics.aperture_mechanisms.C3.position = new_position

        # Print new position
        print(microscope.optics.aperture_mechanisms.C3.position)

A list or tuple can also be used to change the position:


        # Using a list to modify the position
        microscope.optics.aperture_mechanisms.C3.position = [2e-6, 2e-6]

        # Using a tuple to modify the position
        microscope.optics.aperture_mechanisms.C3.position = (1e-6, 1e-6)
