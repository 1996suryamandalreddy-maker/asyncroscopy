ApertureMechanism.is_enabled

Whether this aperture mechanism is currently enabled.

Data type

[Read only] bool

Examples

Find out if the C2 aperture mechanism is enabled:


        print(microscope.optics.aperture_mechanisms.C2.is_enabled)

Disable and enable the SA aperture mechanism and print the current state:


        print("SA is enabled:", microscope.optics.aperture_mechanisms.SA.is_enabled)
        
        microscope.optics.aperture_mechanisms.SA.disable()
        print("SA is enabled:", microscope.optics.aperture_mechanisms.SA.is_enabled)
        
        microscope.optics.aperture_mechanisms.SA.enable()
        print("SA is enabled:", microscope.optics.aperture_mechanisms.SA.is_enabled)
