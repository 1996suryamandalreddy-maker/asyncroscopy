ApertureMechanism.disable

Disables this aperture mechanism.

Syntax

disable()

Remarks

If the aperture mechanism is already disabled, does nothing.

Examples

Disable the C1 aperture mechanism and check the status:


        microscope.optics.aperture_mechanisms.C1.disable()
        print(microscope.optics.aperture_mechanisms.C1.is_enabled)
