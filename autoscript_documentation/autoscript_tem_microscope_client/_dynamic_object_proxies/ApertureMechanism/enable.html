ApertureMechanism.enable

Enables this aperture mechanism.

Syntax

enable()

Remarks

If the aperture mechanism is already enabled, does nothing.

Examples

Enable the objective aperture mechanism and check the status:


        microscope.optics.aperture_mechanisms.objective.enable()
        print(microscope.optics.aperture_mechanisms.objective.is_enabled)
