ApertureMechanism.apertures

Returns available apertures on this mechanism.

Data type

[Read only] List[Aperture]

Examples

Enumerate available apertures on the C1 aperture mechanism:


        apertures = microscope.optics.aperture_mechanisms.C1.apertures
        
        for aperture in apertures:
            print("Type:", aperture.type)
            print("Diameter:", aperture.diameter)
