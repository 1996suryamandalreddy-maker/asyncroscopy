ApertureMechanism.reset_positions

Resets positions of all apertures on this mechanism to default values from microscope alignment.

Syntax

reset_positions()

Remarks

Mechanism must be enabled to reset the position. reset_positions can be used to recover the aperture from an invalid position.

Examples


        # Move selected aperture to some position
        microscope.optics.aperture_mechanisms.C3.position = Point(0.002, 0.002)
        print("Selected aperture position:", microscope.optics.aperture_mechanisms.C3.position)
        
        # Reset positions of all apertures to defaults
        microscope.optics.aperture_mechanisms.C3.reset_positions()
        
        # Print the selected aperture position again
        print("Selected aperture position:", microscope.optics.aperture_mechanisms.C3.position)
