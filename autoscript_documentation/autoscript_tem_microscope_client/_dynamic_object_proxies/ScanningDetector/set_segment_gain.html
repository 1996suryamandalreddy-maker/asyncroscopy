ScanningDetector.set_segment_gain

Sets gain on a given segment. Accepts a single segment name or a list of segment names. Enumeration DetectorSegmentType can be used.

Syntax

set_segment_gain(segment_name, value)

Parameters

segment_name str | List[str]
value float

Remarks

The function can act on a single segment or multiple segments. It accepts the input parameter in several forms:

  • A string representing a single segment, such as "Inner1" or "Segment1".
  • A string representing a group of segments, e.g. "InnerRing" or "All".
  • A list of segment names, e.g. ["Inner1", "Inner2"] or ["Segment1", "Segment2"].

In any case, it is recommended to use the DetectorSegmentType enumeration instead of strings to avoid typos.

Examples

The following examples show several ways of setting the segment gain:


        # Get the BF-S detector object
        detector = microscope.detectors.get_scanning_detector(DetectorType.BF_S)

        # Set gain to value 5 on all segments
        detector.set_segment_gain(DetectorSegmentType.ALL, 5)

        # Set gain to value 5 on outer ring segments; do not change gain of other segments
        detector.set_segment_gain(DetectorSegmentType.OUTER_RING, 5)

        # Set gain to value 5 on explicitly listed segments; do not change gain of other segments
        detector.set_segment_gain([DetectorSegmentType.INNER1, DetectorSegmentType.OUTER1], 5)

It is also possible to access the gain property directly on the segment object.
Note that this approach doesn't allow for batch processing (acting on multiple segments at once). 


        # Get the DF4 detector object
        detector = microscope.detectors.get_scanning_detector(DetectorType.DF4)
        
        # Set the first segment gain
        segment = detector.get_segment(DetectorSegmentType.SEGMENT1)
        segment.gain = 5
