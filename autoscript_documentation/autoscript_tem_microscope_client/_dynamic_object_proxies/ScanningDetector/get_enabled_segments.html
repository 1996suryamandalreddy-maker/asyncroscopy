ScanningDetector.get_enabled_segments

Provides a list of enabled segments.

Syntax

get_enabled_segments()

Return value

List[DetectorSegment]

Remarks

This function can be used to create a list of segments for the image acquisition function acquisition.acquire_stem_segment_images() after the segments have been selected for acquisition by calling ScanningDetector.set_enabled_segments().

The resulting list can be combined with similar lists obtained from other detectors.

Examples

Enable inner ring segments on BF-S and DF-S detectors and combine them to a single list:


        # Enable inner ring segments on the BF-S detector (4 segments in total)
        bf_s = microscope.detectors.get_scanning_detector(DetectorType.BF_S)
        bf_s.set_enabled_segments(DetectorSegmentType.INNER_RING)

        # Enable inner ring segments on the DF-S detector (4 segments in total)
        df_s = microscope.detectors.get_scanning_detector(DetectorType.DF_S)
        df_s.set_enabled_segments(DetectorSegmentType.INNER_RING)

        # 4 + 4 segments
        enabled_segments = bf_s.get_enabled_segments() + df_s.get_enabled_segments()

Enabling segments on the DF4 detector:


        # Enable SEGMENT1 detector
        df4 = microscope.detectors.get_scanning_detector(DetectorType.DF4)
        df4.set_enabled_segments(DetectorSegmentType.SEGMENT1)

        # Enabling INNER_RING or INNER1 segments fails, DF4 detectors don't have rings
        # df4.set_enabled_segments(DetectorSegmentType.INNER_RING)
        # df4.set_enabled_segments(DetectorSegmentType.INNER1)
