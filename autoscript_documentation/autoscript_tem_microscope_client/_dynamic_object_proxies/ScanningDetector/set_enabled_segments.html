ScanningDetector.set_enabled_segments

Enables a given segment and disables all others. Accepts a single segment name or a list of segment names. Enumeration DetectorSegmentType can be used.

Syntax

set_enabled_segments(segment_name)

Parameters

segment_name str | List[str]

Remarks

The function can act on a single segment or multiple segments. It accepts the input parameter in several forms:

  • A string representing a single segment, such as "Inner1" or "Segment1".
  • A string representing a group of segments, e.g. "InnerRing" or "All".
  • A list of segment names, e.g. ["Inner1", "Inner2"] or ["Segment1", "Segment2"].

In any case, it is recommended to use the DetectorSegmentType enumeration instead of strings to avoid typos.

Examples

The following examples show several ways in which segments can be enabled:


        # Get BF-S detector object.
        # BF-S is an 8-segment detector divided into two rings, inner and outer. Both rings have 4 segments.
        detector = microscope.detectors.get_scanning_detector(DetectorType.BF_S)

        # Enable all segments
        detector.set_enabled_segments(DetectorSegmentType.ALL)

        # Enable a single segment, disable all others
        detector.set_enabled_segments(DetectorSegmentType.OUTER1)

        # Enable the first two segments of the inner ring, disable all others
        detector.set_enabled_segments([DetectorSegmentType.INNER1, DetectorSegmentType.INNER2])

        # Enable inner ring (INNER1, INNER2, INNER3 and INNER4) and disable all other segments (OUTER_RING).
        detector.set_enabled_segments(DetectorSegmentType.INNER_RING)

The function is also applicable to detectors without segments. Such detectors are assumed to have a virtual segment SINGLE. Enabling this segment is necessary if you want to include the detector in the segment acquisition task with other detectors, e.g. acquire images from DF-S segments and HAADF detector simultaneously. 


        # Get the HAADF detector object 
        haadf = microscope.detectors.get_scanning_detector(DetectorType.HAADF)

        # Enable its virtual segment "Single"
        haadf.set_enabled_segments(DetectorSegmentType.SINGLE)
        
        # Get the DF-S detector object and enable its outer ring segments
        df_s = microscope.detectors.get_scanning_detector(DetectorType.DF_S)
        df_s.set_enabled_segments(DetectorSegmentType.OUTER_RING)
        
        # Combine all enabled segments into one list (len=5) 
        segments = df_s.get_enabled_segments() + haadf.get_enabled_segments()

        # Acquire images from all segments simultaneously 
        images = microscope.acquisition.acquire_stem_segment_images(segments, 1024, 25e-9)

        # 5 images returned

It is also possible to access the is_enabled property directly on the segment object. Note that this approach doesn't allow for batch processing (acting on multiple segments at once). 


        # Get the DF4 detector object
        detector = microscope.detectors.get_scanning_detector(DetectorType.DF4)
        
        # Enable its first segment
        segment = detector.get_segment(DetectorSegmentType.SEGMENT1)
        segment.is_enabled = True
