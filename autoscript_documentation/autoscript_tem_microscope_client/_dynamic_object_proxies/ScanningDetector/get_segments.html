ScanningDetector.get_segments

Provides a list of segments by their names. By default returns all available segments. Enumeration DetectorSegmentType can be used.

Syntax

get_segments()
get_segments(segment_names)

Parameters

segment_names str | List[str]

Return value

List[DetectorSegment]

Remarks

The parameter-less function returns all segments that are on the detector.

Segment name or a list of segment names can be passed, group segments are 'unpacked' and translated into their members (e.g. ALL is translated into all available segments). The objects are returned in the same order as the names specified in the input list.

Examples

Enumerate all segments on the DF-S detector:


        # Get DF-S detector object
        detector = microscope.detectors.get_scanning_detector(DetectorType.DF_S)

        # Get a list of segments on the detector and inspect their names
        segments = detector.get_segments()  # 8 objects returned

        # Same as
        # segments = detector.get_segments(DetectorSegmentType.ALL)

        for segment in segments:
            print(segment.name)

Get segments that form the inner ring of DF-S:


        # Get DF-S detector object
        detector = microscope.detectors.get_scanning_detector(DetectorType.DF_S)

        # Get inner ring segments
        inner_ring_segments = detector.get_segments(
            [DetectorSegmentType.INNER1, DetectorSegmentType.INNER2,
             DetectorSegmentType.INNER3, DetectorSegmentType.INNER4])  # 4 objects returned

        # Same as
        # inner_ring_segments = detector.get_segments(DetectorSegmentType.INNER_RING)

        for segment in inner_ring_segments:
            print(segment.name)
