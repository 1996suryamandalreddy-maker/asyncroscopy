ScanningDetector

Contains properties and enables control of a scanning (STEM) detector.

Properties

collection_angle_limits [Read only] The range of the collection angles (annular detectors) in radians.
display_name [Read only] The commercial name of the detector.
gain The gain of the detector. The unit depends on the detector type.
gain_limits [Read only] The range of valid gain values. The unit depends on the detector type.
insertion_state [Read only] The insertion state of the detector. Maps to DetectorInsertionState enumeration.
is_insertable [Read only] Whether the detector can be inserted/retracted.
name [Read only] The identifier of the detector.
offset The offset of the detector. The unit depends on the detector type.
offset_limits [Read only] The range of valid offset values. The unit depends on the detector type.
relative_gain Gain value relative to its range. The range is from 0 to 1.
relative_offset Offset value relative to its range. The range is from 0 to 1.

Methods

get_enabled_segments() Provides a list of enabled segments.
get_segment() Returns an object representing one detector segment. Enumeration DetectorSegmentType can be used.
get_segments() Provides a list of segments by their names. By default returns all available segments. Enumeration DetectorSegmentType can be used.
insert() Inserts the detector.
reset_segments() Disables all segments and resets their gain and offset to default values.
retract() Retracts the detector.
set_enabled_segments() Enables a given segment and disables all others. Accepts a single segment name or a list of segment names. Enumeration DetectorSegmentType can be used.
set_segment_gain() Sets gain on a given segment. Accepts a single segment name or a list of segment names. Enumeration DetectorSegmentType can be used.
set_segment_offset() Sets offset on a given segment. Accepts a single segment name or a list of segment names. Enumeration DetectorSegmentType can be used.

Remarks

Scanning detectors are uniquely identifiable by name. The enumeration DetectorType contains all supported scanning detectors.

Examples

The following example shows how to obtain and work with the BF-S detector and it's first inner segment:


        detector = microscope.detectors.get_scanning_detector(DetectorType.BF_S)
        print(detector.name)
        print(detector.insertion_state)

        segment = detector.get_segment(DetectorSegmentType.INNER1)
        print(segment.detector_name)
        print(segment.name)
        print(segment.is_enabled)
        print(segment.gain)
        print(segment.offset)
