EdsDetector.get_segments

Provides a list of segments by their names. By default returns all available segments. Enumeration EdsDetectorSegmentType can be used.

Syntax

get_segments()
get_segments(segment_names)

Parameters

segment_names str | List[str]

Return value

List[EdsDetectorSegment]

Remarks

The parameter-less function returns all segments that are on the detector.

Segment name or a list of segment names can be passed, group segments are 'unpacked' and translated into their members (e.g. ALL is translated into all available segments). The objects are returned in the same order as the names specified in the input list.

Examples

Enumerate all segments on the SuperX detector:


        # Get SuperX detector object
        detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)

        # Get a list of segments on the detector and inspect their names
        segments = detector.get_segments()  # 4 objects returned

        # Same as
        # segments = detector.get_segments(EdsDetectorSegmentType.ALL)

        for segment in segments:
            print(segment.name)

Get specified segments of the SuperX detector:


        # Get SuperX detector object
        detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)

        # Get a list of specific segments
        segments = detector.get_segments([EdsDetectorType.SEGMENT1, EdsDetectorType.SEGMENT2])  # 2 objects returned

        # Same as
        # inner_ring_segments = detector.get_segments(DetectorSegmentType.INNER_RING)

        for segment in inner_ring_segments:
            print(segment.name)
