EdsDetector

Contains properties and enables control of a EDS detector.

Properties

bin_counts [Read only] List of all supported energy bin counts.
dispersions [Read only] List of all supported dispersions in eV.
display_name [Read only] The commercial name of the detector.
insertion_state [Read only] The insertion state of the detector. Maps to DetectorInsertionState enumeration.
is_insertable [Read only] Whether the detector can be inserted/retracted.
name [Read only] The identifier of the detector.
shaping_times [Read only] List of all supported shaping times in seconds.

Methods

get_enabled_segments() Provides a list of enabled segments.
get_segment() Returns an object representing one detector segment. Enumeration EdsDetectorSegmentType can be used.
get_segments() Provides a list of segments by their names. By default returns all available segments. Enumeration EdsDetectorSegmentType can be used.
insert() Inserts the detector.
reset_segments() Disables all segments.
retract() Retracts the detector.
set_enabled_segments() Enables a given segment and disables all others. Accepts a single segment name or a list of segment names. Enumeration EdsDetectorSegmentType can be used.

Remarks

EDS detectors are uniquely identifiable by name. The enumeration EdsDetectorType contains all supported EDS detectors.

Examples

The following example shows how to obtain and work with an EDS detector:


        # Get SuperX detector object
        detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)

        # Get a list of segments on the detector and inspect their names
        segments = detector.get_segments()  # 4 objects returned
        for segment in segments:
            print(segment.name)

        # Insert the detector (if already inserted, do nothing)
        detector.insert()
        print(detector.insertion_state)
        
        # Retract the detector (if already retracted, do nothing)
        detector.retract()
        print(detector.insertion_state)
