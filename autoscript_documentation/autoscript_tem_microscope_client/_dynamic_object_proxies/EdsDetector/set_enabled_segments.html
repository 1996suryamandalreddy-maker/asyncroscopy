EdsDetector.set_enabled_segments

Enables a given segment and disables all others. Accepts a single segment name or a list of segment names. Enumeration EdsDetectorSegmentType can be used.

Syntax

set_enabled_segments(segment_name)

Parameters

segment_name str | List[str]

Remarks

The function can act on a single segment or multiple segments. It accepts the input parameter in several forms:

  • A string representing a single segment, such as "Segment1".
  • A string representing a group of segments, e.g. "All".
  • A list of segment names, e.g. ["Segment1", "Segment2"].

In any case, it is recommended to use the EdsDetectorSegmentType enumeration instead of strings to avoid typos.

Examples

The following examples show several ways in which segments can be enabled:


        # Get SuperX detector object
        detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)

        # Enable all segments
        detector.set_enabled_segments(EdsDetectorSegmentType.ALL)

        # Enable a single segment, disable all others
        detector.set_enabled_segments(EdsDetectorSegmentType.SEGMENT1)

        # Enable the first two segments, disable all others
        detector.set_enabled_segments([EdsDetectorSegmentType.SEGMENT1, EdsDetectorSegmentType.SEGMENT2])

It is also possible to access the is_enabled property directly on the segment object. Note that this approach doesn't allow for batch processing (acting on multiple segments at once). 


        # Get the SuperX detector object
        detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)
        
        # Enable its first segment
        segment1 = detector.get_segment(EdsDetectorSegmentType.SEGMENT1)
        segment1.is_enabled = True
