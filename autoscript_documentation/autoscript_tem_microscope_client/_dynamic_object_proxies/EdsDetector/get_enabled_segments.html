EdsDetector.get_enabled_segments

Provides a list of enabled segments.

Syntax

get_enabled_segments()

Return value

List[EdsDetectorSegment]

Remarks

This function can be used to create a list of segments for the spectrum acquisition function acquire_spectrum(). after the segments have been selected for acquisition by calling EdsDetector.set_enabled_segments().

Examples

Enable all segments on SuperX detector:


        # Enable all segments on the SuperX detector (4 segments in total)
        super_x = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)
        super_x.set_enabled_segments(EdsDetectorSegmentType.ALL)
        
        # 4 segments
        enabled_segments = super_x.get_enabled_segments()

See also
