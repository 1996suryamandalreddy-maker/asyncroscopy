EelsDetector

Contains properties of an EELS detector.

Properties

display_name [Read only] The commercial name of the detector.
maximum_bin_count [Read only] Returns the maximum number of bins supported by the detector.
name [Read only] The identifier of the detector.

Examples

The following example shows how get and work with an EELS detector:


        # Get the Zebra detector object
        detector = microscope.detectors.get_eels_detector(EelsDetectorType.ZEBRA)
        
        # Print detector name
        print(detector.name)
        
        # Print detector display name
        print(detector.display_name)
        
        # Inspect detector state and properties
        print(detector.maximum_bin_count)
