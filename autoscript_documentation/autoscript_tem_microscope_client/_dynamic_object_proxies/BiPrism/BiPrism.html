BiPrism

Contains properties and enables control of a bi-prism.

Properties

is_enabled [Read only] Whether the bi-prism is currently enabled.
name [Read only] The name of the bi-prism.
rotation [Read only] The rotation control of the bi-prism.
voltage [Read only] The voltage control of the bi-prism.

Methods

recover() Recover the bi-prism from error state.

Remarks

Enabling bi-prism

To enable the bi-prism, holography mode is_holography_mode_on has to be enabled and bi-prism aperture has to be set as aperture in desired ApertureMechanism and inserted.

Examples

Setting the rotation by degrees


        import math
        biprism.rotation.set(math.radians(45))

Setting the max voltage


        biprism.voltage.value = biprism.voltage.limits.max

Selecting bi-prism aperture enables control of bi-prism


        microscope.optics.is_holography_mode_on = True

        aperture_mechanism = microscope.optics.aperture_mechanisms.get_mechanism("C3")

        for aperture in aperture_mechanism.apertures:
            if aperture.type == ApertureType.BI_PRISM:
                aperture_mechanism.aperture = aperture

        is_enabled =  microscope.optics.aperture_mechanisms.bi_prisms.get_bi_prism("C3").is_enabled
        print(f"Bi-prism is enabled: {is_enabled}")
