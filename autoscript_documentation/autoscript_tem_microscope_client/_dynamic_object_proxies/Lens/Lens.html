Lens

Contains properties of a lens.

Properties

limits_raw [Read only] Minimum and maximum raw output value.
name [Read only] Name of the Lens. Uses values from LensType.
value Output value as a ratio of the raw output and the limit value.
value_raw Raw output as a 2D vector in optical units (x, y).

Remarks

The excitation of each lens can be controlled using two different properties.

  • The first one is a ratio value. Its allowed range is -1 to 1 and corresponds to the limits_raw min to max values. The value describes the degree to which the lens is excited, with a value of -1 indicating that the lens is excited to the limits_raw.min. Value of 0 indicates that the lens is not excited at all and value of 1 indicates excitation to the limits_raw.max.
  • The second property to control the lens excitation is value_raw measured in optical units. The allowed range is in this case defined by limits_raw property.
  • Both value and value_raw properties are dependent on each other, so updating one of them will result in a corresponding update to the other.
  • Raw value properties (value_raw, limits_raw) are not available for the X lens type. Instead, specific alignments for the X lens type are provided (see x alignment page).

About the Free Control mode

Free control mode functionality is licensed under separate Free Control feature license. Accessing the functionality without the proper license will result in an error.

Adjusting any lens value triggers the system to enter free control mode. In this mode, the mathematical model governing the optics is deactivated, allowing users direct control over all lenses. The accuracy of the optics model relies on calibrations to assess the system's state. Directly altering any lens parameter renders the calibrations ineffective in predicting the model's state. Once the system transitions to free control mode, the values derived from the optical model are no longer accurate (such as optical modes, magnification, camera length, etc.).

Storing the state of the optical model before entering free control mode is possible through PEOUI with FEG registers and alignments.

The system reverts to calibrated values when the state of the optical model changes, which can occur due to any currently active application on the microscope. In AutoScript, modifying the following parameters deactivates free control mode:

  • Optical mode
  • Probe mode
  • Projector mode
  • Objective lens mode
  • Illumination mode
  • Is EFTEM on
  • Magnification
  • Camera length

Simply reading the values does not disable free control mode; however, as mentioned earlier, the values may be inaccurate. Free control mode deactivates even if the same value is set.

Examples


        # retrieve the C1 lens
        lens = microscope.optics.lenses.get_lens(LensType.C1)
        
        # read the value_raw limits
        print(lens.limits_raw)
        
        # set the value_raw property
        lens.value_raw = 0.125
        
        # read both value and value_raw
        print(lens.value)
        print(lens.value_raw)
