AdornedSpectrumImage

Handle to the spectrum image result existing on the server.

Properties

acquisition_start_date_time [Read only] Date and time of the start of acquisition in ISO 8601 format (YYYY-MM-DDThh:mm:ss).

Methods

save() Transfers the spectrum image file to the client and saves it to disk.

Remarks

This object represents a spectrum image file that resides on the MPC disk. It can be sent to the client over the network using the save() method.

The file is valid on the MPC for about 24 hours and is then removed.
