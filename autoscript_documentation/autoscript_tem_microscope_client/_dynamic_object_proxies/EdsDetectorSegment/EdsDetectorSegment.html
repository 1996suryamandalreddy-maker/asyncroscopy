EdsDetectorSegment

Represents a single segment on a EdsDetector.

Properties

dead_time_fraction [Read only] Current dead time fraction.
detector_name [Read only] Name of the detector containing this segment. Uses values from EdsDetectorType.
input_count_rate [Read only] Current input count rate in counts/second.
is_enabled Whether this segment is currently enabled.
name [Read only] Name of the segment. Uses values from EdsDetectorSegmentType.
output_count_rate [Read only] Current output count rate in counts/second.
