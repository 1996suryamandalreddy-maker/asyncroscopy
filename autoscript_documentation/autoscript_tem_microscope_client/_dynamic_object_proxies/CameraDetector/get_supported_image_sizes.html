CameraDetector.get_supported_image_sizes

Returns the list of supported image sizes.

Syntax

get_supported_image_sizes()

Return value

List[CameraImageSize]

Remarks

Returns a list of supported image sizes, broken down into the square size of the image in pixels and the fixed readout area. These settings can be used for all standard camera acquisitions. The list is ordered by image size, starting with the smallest.

Examples

The example shows how the function can be used to acquire all supported sizes from the camera:


        camera = self.detectors.get_camera_detector(CameraType.BM_CETA)

        # Run acquisition for each supported image size
        for image_size in camera.get_supported_image_sizes():
            # Create acquisition settings
            settings = CameraAcquisitionSettings(
                camera.name,
                image_size.size,
                exposure_time=1e-6,
                fixed_readout_area=image_size.fixed_readout_area)

            # Acquire image
            image = self.acquisition.acquire_camera_image_advanced(settings)
