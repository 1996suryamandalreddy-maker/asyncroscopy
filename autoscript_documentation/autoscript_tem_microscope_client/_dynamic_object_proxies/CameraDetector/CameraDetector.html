CameraDetector

Contains properties of a camera detector.

Properties

display_name [Read only] The commercial name of the detector.
exposure_time_limits [Read only] Minimum and maximum exposure times in seconds. Flucam only.
insertion_state [Read only] The insertion state of the detector. Maps to DetectorInsertionState enumeration.
is_dose_sensitive [Read only] Whether the camera is dose sensitive.
is_insertable [Read only] Whether the detector can be inserted/retracted.
is_operational [Read only] Whether the camera is operational.
name [Read only] The identifier of the detector.
pixel_size [Read only] Pixel size as a 2D vector in meters (x, y).

Methods

get_supported_image_sizes() Returns the list of supported image sizes.
insert() Inserts the detector.
retract() Retracts the detector.

Remarks

Camera detectors are uniquely identifiable by name. The enumeration CameraType contains all supported scanning detectors.
