AdornedImage.load

Loads an image from a file. Supported formats: tiff, png, bmp.

Syntax

load(path)

Parameters

path File path from which to load the image.

Return value

AdornedImage Loaded image.

Remarks

TIFF is a preferred image format because it allows metadata to be stored with the image. However, the function can also read BMP and PNG image formats. For those two formats, the member metadata is left uninitialized.

PNG images with the 16-bit bit depth are loaded as 32 bit.

AutoScript needs read access for the file to open it. All locks are released after load method finishes (no resource release or close required).

Examples

Load TIFF file from disk:


        image = AdornedImage.load("c:\\image.tiff")
