AdornedImage.thumbnail

The thumbnail of the original image.

Data type

[Read only] AdornedImage

Remarks

The thumbnail is recreated each time the property is accessed to account for possible changes to the image data. You may want to cache the thumbnail in a variable for performace critical applications.

The thumbnail is created by resizing the original image in each dimension to a maximum of 512 pixels, maintaining the aspect ratio where possible. The minimum size in each dimension is 1 pixel, so extremely large aspect ratios will distort the image.
