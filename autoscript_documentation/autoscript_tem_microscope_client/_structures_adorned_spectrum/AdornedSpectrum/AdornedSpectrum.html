AdornedSpectrum

Contains EDS spectrum data and metadata.

Items

encoding [Read only] str Encoding of the spectrum. Uses values from enumeration SpectrumEncoding.
data [Read only] Numpy.ndarray Provides access to data in a form of numpy.ndarray.
metadata [Optional] AdornedImageMetadata Metadata describing the EDS spectrum.

Remarks

The structure contains both the numpy array representing the spectrum and the metadata. It is used as a result by EDS acquisition functions, but can also be created manually by specifying a numpy array and a metadata object. Both arguments are optional, the numpy array must be one-dimensional and contain only unsigned integer values (see SpectrumEncoding for all supported formats).

The object construction uses deep copy internally for the metadata object to avoid overwriting it later. This way, multiple spectra do not share the same metadata object.

Examples

The following example shows how to create an AdornedSpectrum object filled with random data:


        import numpy as np

        # Creates a random spectrum with 4096 bins (the array must be casted to unsigned integer type)
        empty_metadata = AdornedImageMetadata()
        random_spectrum = AdornedSpectrum(np.random.randint(low=0, high=1024, size=4096).astype(np.uint32), empty_metadata)
