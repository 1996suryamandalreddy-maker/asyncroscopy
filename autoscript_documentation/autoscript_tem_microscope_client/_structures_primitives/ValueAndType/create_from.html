ValueAndType.create_from

Creates a new instance of the structure from a collection object.

Syntax

create_from(value)

Parameters

value

Return value

ValueAndType

Remarks

The collection object can be either a list or a tuple with up to two values, or another instance of ValueAndType. The function always creates a new object.

Examples


        ValueAndType.create_from(["100", "int"])                # created from a list
        ValueAndType.create_from(("100", "int"))                # created from a tuple
        ValueAndType.create_from(ValueAndType("100", "int"))    # created from another ValueAndType
