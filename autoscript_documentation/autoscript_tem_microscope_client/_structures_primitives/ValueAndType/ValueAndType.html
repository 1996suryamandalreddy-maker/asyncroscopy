ValueAndType

Defines a value with its type.

Items

value [Optional] str
type [Optional] str

Methods

create_from() Creates a new instance of the structure from a collection object.

Overloaded operators

Indexer Values are indexed in following order: value, type.
Equality A value-and-type-like object is supported (indexable object with a length of 2).

Examples

Overloaded operator examples:


        value_and_type_a = ValueAndType("100", "int")
        value_and_type_b = ValueAndType("3.14", "float")

        print(value_and_type_a[0])                   # "100"
        print(value_and_type_a == value_and_type_b)  # False
