StageVelocity

Defines a stage velocity.

Items

x [Optional] float X axis in meters per second.
y [Optional] float Y axis in meters per second.
z [Optional] float Z axis in meters per second.
a [Optional] float Alpha axis in radians per second.
b [Optional] float Beta axis in radians per second.

Methods

create_from() Creates a new instance of the structure from a collection object.

Overloaded operators

Indexer Lower or upper case names, values from StageAxis enumerations, or integers can be used as an index: 'x' == 'X' == StageAxis.X == 0. Integer indexes have following order: x, y, z, a, b.
Addition A scalar value or a velocity-like object is supported (indexable object with up to 5 values).
Subtraction A scalar value or a velocity-like object is supported (indexable object with up to 5 values).
Multiplication A scalar value is supported.
Division A scalar value is supported.
Floor division A scalar value is supported.
Equality A velocity-like object is supported (indexable object with up to 5 values). Tests approximate equality with maximum tolerance of 1e-6.

Examples

Following example shows different ways of creating stage velocity:


        import math
        
        StageVelocity(1e-6, 2e-6, 3e-6)  # x = 1e-6 m/s, y = 2e-6 m/s, z = 3e-6 m/s
        StageVelocity(z=4e-6)  # z = 4e-6 m/s, other axes are None

        # Conversion from degrees to radians
        StageVelocity(a=math.radians(5))  # alpha = 5 deg/s

        # Get current piezo stage jogging velocity and update it
        velocity = microscope.specimen.stage.jogging_velocity
        velocity.x = 5e-6
        velocity.y = 6e-6

Overloaded operator examples:


        velocity_a = StageVelocity(2, 3, 4, 5, 6)
        velocity_b = StageVelocity(1, 2, 3, 4, 5)

        print(velocity_a[2])            # 4
        print(velocity_a[StageAxis.Y])  # 3
        print(velocity_a["b"])          # 6
        print(velocity_a + velocity_b)  # StageVelocity(3, 5, 7, 9, 11)
        print(velocity_a + 5)           # StageVelocity(7, 8, 9, 10, 11)
        print(velocity_a - velocity_b)  # StageVelocity(1, 1, 1, 1, 1)
        print(velocity_a - 3)           # StageVelocity(-1, 0, 1, 2, 3)
        print(velocity_a * 2)           # StageVelocity(4, 6, 8, 10, 12)
        print(velocity_a / 2)           # StageVelocity(1, 1.5, 2, 2.5, 3)
        print(velocity_a // 3)          # StageVelocity(0, 1, 1, 1, 2)
        print(velocity_a == velocity_b) # False
