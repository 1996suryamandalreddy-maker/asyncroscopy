StageVelocity.create_from

Creates a new instance of the structure from a collection object.

Syntax

create_from(value)

Parameters

value

Return value

StageVelocity

Remarks

The collection object can be either a list or a tuple with up to five values, or another instance of StageVelocity. The function always creates a new object.

Examples


        # created from a list
        StageVelocity.create_from([0, 1, 2, 3, 4])

        # created from a tuple
        StageVelocity.create_from((0, 1, 2, 3, 4))

        # created from another StageVelocity
        StageVelocity.create_from(StageVelocity(0, 1, 2, 3, 4))
