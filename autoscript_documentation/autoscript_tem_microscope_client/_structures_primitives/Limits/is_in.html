Limits.is_in

Checks if the given value falls into the limits. Tests approximate equality with maximum tolerance of 1e-9.

Syntax

is_in(value)

Parameters

value

Return value

bool

Remarks

None values function as open intervals.

Examples


        limits = Limits(1, 5)

        # interval is closed
        print(limits.is_in(1))  # True
        print(limits.is_in(5))  # True

        # acceptable tolerance is 1e-9
        print(limits.is_in(1 - 2e-10))  # True
        print(limits.is_in(5 + 2e-10))  # True

        # None values function as open intervals
        open_limits = Limits(max=1)  # min is None
        print(open_limits.is_in(2))  # False
        print(open_limits.is_in(0))  # True
        print(open_limits.is_in(-1000))  # True
