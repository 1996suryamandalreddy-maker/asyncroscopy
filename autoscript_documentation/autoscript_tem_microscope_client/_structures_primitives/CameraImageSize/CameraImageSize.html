CameraImageSize

Represents a size of a camera image with square pixel value and a fixed readout area.

Items

size [Read only] int Square image size in pixels.
fixed_readout_area [Read only] str Readout area of the sensor. Uses FixedReadoutArea.

Remarks

The structure defines a combination of image size and camera readout area. Together they implicitly define binning as well. For example, a camera with a full size of 4096x4096 pixels and values CameraImageSize(size=2048, fixed_readout_area=FixedReadoutArea.FULL) has a binning of 2x2.

See the following table for an example of how the image size, readout area and binning are related for a Ceta camera:

Image size Readout area Sampling / Binning
4096 Full 1
2048 Full 2
1024 Full 4
512 Full 8
2048 Half 1
1024 Half 2
512 Half 4
256 Half 8
1024 Quarter 1
512 Quarter 2
256 Quarter 4

Acquisition functions without the readout area parameter (e.g. acquire_camera_image()) will use the full readout area by default. Sizes outside the range given by the chosen readout area are not accepted. Sizes within the range but not equal to any specific value are rounded to the nearest supported size.
