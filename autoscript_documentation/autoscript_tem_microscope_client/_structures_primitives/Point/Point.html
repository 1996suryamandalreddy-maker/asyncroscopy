Point

Defines the X and Y coordinates of a point in 2D space.

Items

x [Optional] float X coordinate of the point.
y [Optional] float Y coordinate of the point.

Methods

angle() Returns the angle of the point in radians.
create_from() Creates a new instance of the structure from a collection object.
magnitude() Returns the magnitude of the point which is the distance from 0, 0.

Overloaded operators

Indexer Values are indexed in following order: x, y.
Addition A scalar value or a point-like object is supported (indexable object with a length of 2).
Subtraction A scalar value or a point-like object is supported (indexable object with a length of 2).
Multiplication A scalar value or a point-like object is supported (indexable object with a length of 2). Returns dot product in case of point-like object.
Division A scalar value is supported.
Floor division A scalar value is supported.
Equality A point-like object is supported (indexable object with a length of 2). Tests approximate equality with maximum tolerance of 1e-9.

Examples

Overloaded operator examples:


        point_a = Point(2, 2)
        point_b = Point(1, 1)

        print(point_a[0])          # 2
        print(point_a + point_b)   # Point(3, 3)
        print(point_a + 5)         # Point(7, 7)
        print(point_a - point_b)   # Point(1, 1)
        print(point_a - 3)         # Point(-1, -1)
        print(point_a * 2)         # Point(4, 4)
        print(point_a * point_b)   # 4 (dot product)
        print(point_a / 2)         # Point(1, 1)
        print(point_a // 3)        # Point(0, 0)
        print(point_a == point_b)  # False
