Point.create_from

Creates a new instance of the structure from a collection object.

Syntax

create_from(value)

Parameters

value

Return value

Point

Remarks

The collection object can be either a list or a tuple with up to two values, or another instance of Point. The function always creates a new object.

Examples


        Point.create_from([0, 1])       # created from a list
        Point.create_from((0, 1))       # created from a tuple
        Point.create_from(Point(1, 1))  # created from another Point
