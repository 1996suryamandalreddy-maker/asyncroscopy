Point.magnitude

Returns the magnitude of the point which is the distance from 0, 0.

Syntax

magnitude()

Return value

float

Remarks

Error is thrown if one of the point coordinates is None.

Examples


        print(Point(0, 1).magnitude())  # 1
        print(Point(1, 0).magnitude())  # 1
        print(Point(4, 3).magnitude())  # 5
