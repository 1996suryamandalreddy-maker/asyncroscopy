Rectangle.create_from

Creates a new instance of the structure from a collection object.

Syntax

create_from(value)

Parameters

value

Return value

Rectangle

Remarks

The collection object can be either a list or a tuple with up to four values, or another instance of Rectangle. The function always creates a new object.

Examples


        Rectangle.create_from([0, 1, 2, 3])           # created from a list
        Rectangle.create_from((0, 1, 2, 3))           # created from a tuple
        Rectangle.create_from(Rectangle(1, 1, 2, 3))  # created from another Rectangle
