Rectangle

Defines a rectangular shape.

Items

left [Optional] float The left side of a rectangle.
top [Optional] float The top side of a rectangle.
width [Optional] float The width of a rectangle.
height [Optional] float The height of a rectangle.

Methods

create_from() Creates a new instance of the structure from a collection object.

Overloaded operators

Indexer Values are indexed in following order: left, top, width, height.
Addition A scalar value or a rectangle-like object is supported (indexable object with a length of 4).
Subtraction A scalar value or a rectangle-like object is supported (indexable object with a length of 4).
Multiplication A scalar value is supported.
Division A scalar value is supported.
Floor division A scalar value is supported.
Equality A rectangle-like object is supported (indexable object with a length of 4). Tests approximate equality with maximum tolerance of 1e-9.

Examples

Overloaded operator examples:


        rectangle_a = Rectangle(2, 2, 2, 2)
        rectangle_b = Rectangle(1, 1, 1, 1)

        print(rectangle_a[0])               # 2
        print(rectangle_a + rectangle_b)    # Rectangle(3, 3, 3, 3)
        print(rectangle_a + 5)              # Rectangle(7, 7, 7, 7)
        print(rectangle_a - rectangle_b)    # Rectangle(1, 1, 1, 1)
        print(rectangle_a - 3)              # Rectangle(-1, -1, -1, -1)
        print(rectangle_a * 2)              # Rectangle(4, 4, 4, 4)
        print(rectangle_a / 2)              # Rectangle(1, 1, 1, 1)
        print(rectangle_a // 3)             # Rectangle(0, 0, 0, 0)
        print(rectangle_a == rectangle_b)   # False
