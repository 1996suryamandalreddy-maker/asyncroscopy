StagePosition

Defines a stage position.

Items

x [Optional] float X coordinate in meters.
y [Optional] float Y coordinate in meters.
z [Optional] float Z coordinate in meters.
a [Optional] float Alpha coordinate in radians.
b [Optional] float Beta coordinate in radians.

Methods

create_from() Creates a new instance of the structure from a collection object.

Overloaded operators

Indexer Lower or upper case names, values from StageAxis enumerations, or integers can be used as an index: 'x' == 'X' == StageAxis.X == 0. Integer indexes have following order: x, y, z, a, b.
Addition A scalar value or a position-like object is supported (indexable object with up to 5 values).
Subtraction A scalar value or a position-like object is supported (indexable object with up to 5 values).
Multiplication A scalar value is supported.
Division A scalar value is supported.
Floor division A scalar value is supported.
Equality A position-like object is supported (indexable object with up to 5 values). Tests approximate equality with maximum tolerance of 1e-6.

Examples

Following example shows different ways of creating stage position:


        import math
        
        StagePosition(0.001, 0.002, 0.003, 0.4, 0.5)  # x = 0.001 m, y = 0.002 m, z = 0.003 m, alpha = 0.4 rad, beta = 0.5 rad
        StagePosition(z=0.003)  # z = 0.003 m, other axes are None

        # Conversion from degrees to radians
        StagePosition(a=math.radians(5))  # alpha = 5 deg

        # Get current stage position and update it
        position = microscope.specimen.stage.position
        position.x = 0.005
        position.y = 0.006

Overloaded operator examples:


        position_a = StagePosition(2, 3, 4, 5, 6)
        position_b = StagePosition(1, 2, 3, 4, 5)

        print(position_a[2])            # 4
        print(position_a[StageAxis.Y])  # 3
        print(position_a["b"])          # 6
        print(position_a + position_b)  # StagePosition(3, 5, 7, 9, 11)
        print(position_a + 5)           # StagePosition(7, 8, 9, 10, 11)
        print(position_a - position_b)  # StagePosition(1, 1, 1, 1, 1)
        print(position_a - 3)           # StagePosition(-1, 0, 1, 2, 3)
        print(position_a * 2)           # StagePosition(4, 6, 8, 10, 12)
        print(position_a / 2)           # StagePosition(1, 1.5, 2, 2.5, 3)
        print(position_a // 3)          # StagePosition(0, 1, 1, 1, 2)
        print(position_a == position_b) # False
