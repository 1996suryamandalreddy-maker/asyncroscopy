RunBeamTiltAutoFocusSettings

Contains settings for the beam tilt automatic focus routine.

Items

camera_detector str The name of the detector. Enumeration CameraType can be used.
size [Optional] int Square image size in pixels. Default is 1024. Enumeration ImageSize can be used.
exposure_time [Optional] float The exposure time in seconds. Default is 1.
focus_method [Optional] str Focus correction method to be used. Maps to enumeration FocusMethod. Default is OBJECTIVE_LENS.
auto_stigmate [Optional] bool Whether the procedure should optimize astigmatism or not. False by default.
use_three_image_method [Optional] bool Whether the procedure should use 3 images to be less sensitive to drift. False by default.
maximum_focus_change [Optional] float A new iteration is started as long as the applied focus change is larger than this value. Default is 0.25e-6.
maximum_iterations [Optional] int Maximum number of iterations that is applied. Default is 4.
maximum_astigmatism_change [Optional] float A new iteration is started as long as the applied astigmatism change is larger than this value. Default is 0.1e-6.
iterate_to_defocus [Optional] float Iterate to defocus value in meters. Makes the procedure more robust for samples where the contrast at focus is too low. Default is 0.

Remarks

All attributes without specified default values are required.
