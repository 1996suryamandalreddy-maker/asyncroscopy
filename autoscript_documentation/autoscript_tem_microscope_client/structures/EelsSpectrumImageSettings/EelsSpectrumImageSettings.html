EelsSpectrumImageSettings

Contains settings for EELS spectrum image acquisition.

Items

eels_detector str The name of the EELS detector. Enumeration EelsDetectorType can be used.
eels_exposure_settings List[EelsExposureSettings] Exposure settings for the acquisition. Must contain exactly one element.
dwell_time float Scan dwell time in seconds.
size [Optional] int Square image size in pixels. Enumeration ImageSize can be used.
region [Optional] Region Acquisition area to be scanned.
scanning_detectors [Optional] List[str] The name(s) of the STEM detector(s). Enumeration DetectorType can be used.
scanning_detector_segments [Optional] List[DetectorSegment] The detector segment(s) used for the acquisition.

Remarks

The minimum dwell time value is 100µs. The dwell time must also be higher than the sum of exposure times (plus the necessary overhead depending on the number of spectra and HW limitations).

See the following table for some minimum dwell time examples:

Sum of exposure times [s] Minimum dwell time [s]
1e-6 1e-4
1e-5 1e-4
1e-4 1.3e-4
1e-3 1.03e-3
1e-2 1.01e-2
1e-1 1.01e-1
