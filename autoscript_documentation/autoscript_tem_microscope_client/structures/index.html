Data structures

Data structures are simple objects for transporting data. They usually contain some helper functions and/or overloaded operators for convenience. Structures are often used to combine multiple arguments or return values (e.g. CameraAcquisitionSettings), or to represent data formats (e.g. AdornedImage, EmdFile).

Structures can be created on the client as well as on the server. When the structure is passed as a parameter to the function call, the entire structure is serialized and sent to the server. When accepting structures from the server, some structures may not be fully deserialized until their data is accessed, i.e., lazy deserialization.
