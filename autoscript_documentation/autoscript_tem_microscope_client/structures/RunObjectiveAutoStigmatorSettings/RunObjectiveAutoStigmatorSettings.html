RunObjectiveAutoStigmatorSettings

Contains settings for the objective auto stigmator routine.

Items

camera_detector str The name of the detector. Enumeration CameraType can be used.
size [Optional] int Square image size in pixels. Default is 1024. Enumeration ImageSize can be used.
exposure_time [Optional] float The exposure time in seconds. Default is 1.
maximum_iterations [Optional] int Maximum number of iterations that is applied. Default is 10.
maximum_remaining_astigmatism [Optional] float Maximum remaining astigmatism in meters. Default is 5e-9.
defocus_step [Optional] float Defocus step in meters. Default is -1e-6.
maximum_anisotropy [Optional] float Maximum anisotropy. Default is 2.
maximum_astigmatism_fit_error [Optional] float Maximum astigmatism fit error. Default is 0.9.
tile_size [Optional] int Tile size. Default is 512.
is_thon_ring_mask_image_crossed_out_when_fitting_fails [Optional] bool Whether the Thon ring mask image is crossed out when fitting fails. True by default.

Remarks

All attributes without specified default values are required.
