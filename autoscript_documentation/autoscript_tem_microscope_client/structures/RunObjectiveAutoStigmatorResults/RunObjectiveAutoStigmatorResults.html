RunObjectiveAutoStigmatorResults

Contains result data from the objective auto stigmator routine.

Items

astigmatism [Read only] float Measured objective astigmatism in meters.
astigmatism_orientation [Read only] float Measured objective astigmatism angle in radians.
astigmatism_fit_error [Read only] float A value between 0 and 1 as a measure for quality: 0 = perfect fit, 1 = poor fit.
is_reliable [Read only] bool Indication whether the result is reliable.
