RunStemAutoFocusSettings

Contains settings for the automatic focus routine in STEM optical mode.

Items

scanning_detector str The name of the detector. Enumeration DetectorType can be used.
size [Optional] int Square image size in pixels. Default is 1024. Enumeration ImageSize can be used.
dwell_time [Optional] float The dwell time in seconds. Default is 1e-6.
auto_stigmate [Optional] bool Whether the procedure should optimize stigmation or not. False by default.
accuracy_level [Optional] str Specifies level of accuracy. Enumeration AccuracyLevel can be used. Default is COARSE.
reset_beam_blanker_afterwards [Optional] bool If true, beam blanker will will be set to its previous status after the task completes. If false, beam will be left blanked. False by default.

Remarks

All attributes without specified default values are required.

See also

auto_functions.run_stem_auto_focus()
