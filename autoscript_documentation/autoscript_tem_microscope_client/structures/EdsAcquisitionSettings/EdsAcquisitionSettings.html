EdsAcquisitionSettings

Contains settings for single spectrum EDS acquisition.

Items

eds_detector str The name of the EDS detector. Enumeration EdsDetectorType can be used.
dispersion float Dispersion in eV.
shaping_time float Shaping time in seconds.
exposure_time float Exposure time in seconds.
exposure_time_type str Exposure time type. Enumeration ExposureTimeType can be used.
detector_segments [Optional] List[EdsDetectorSegment] The list of enabled detector segments. If not provided, all segments are assumed.

See also
