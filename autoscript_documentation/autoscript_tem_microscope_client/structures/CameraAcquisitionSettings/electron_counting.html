CameraAcquisitionSettings.electron_counting

Whether or not an electron counted image is produced (counting detectors only). True by default.

Data type

[Optional] bool

Remarks

The parameter applies only to counting cameras (Falcon). For other camera types, the acquisition will return an error if this parameter is set to both true and false (leave the parameter as it is or set it to None).

If the parameter is set to false, the counts are integrated and can no longer be related to electrons.
