CameraAcquisitionSettings

Contains settings for image acquisition from a camera detector.

Items

camera_detector str The name of the detector. Enumeration CameraType can be used.
size int Square image size in pixels. Enumeration ImageSize can be used.
exposure_time float The exposure time in seconds.
fixed_readout_area [Optional] str Readout area of the sensor. Enumeration FixedReadoutArea can be used. FULL by default.
frame_combining [Optional] int The number of sub-frames the camera will combine (Ceta only). Default value is 1.
electron_counting [Optional] bool Whether or not an electron counted image is produced (counting detectors only). True by default.

Remarks

This structure contains a set of advanced settings for camera acquisition. It is used as an argument for several acquisition functions.
