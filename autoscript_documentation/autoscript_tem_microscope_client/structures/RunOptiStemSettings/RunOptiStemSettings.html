RunOptiStemSettings

Contains settings for the OptiStem routine.

Items

method [Optional] str Specifies what OptiStemMethod should be used. The default value is C1A1.
dwell_time [Optional] float STEM dwell time in seconds. The default value depends on method.
cutoff_in_pixels [Optional] int Interval between any two adjacent samples - Nyquist interval in pixels. The default value is 4 pixels. This parameter is not used for B2A2 method and will result in exception if set.
cutoff_spatial_frequency [Optional] float Amount of cycles per meter. The default value is 2.85e+9. This parameter is not used for B2A2 method and will result in exception if set.
