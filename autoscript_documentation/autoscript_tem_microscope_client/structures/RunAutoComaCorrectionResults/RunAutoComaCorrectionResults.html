RunAutoComaCorrectionResults

Contains result data from the auto coma correction routine.

Items

astigmatism [Read only] float Measured objective astigmatism in meters.
astigmatism_orientation [Read only] float Measured objective astigmatism angle in radians.
threefold_astigmatism [Read only] float Measured threefold objective astigmatism in meters.
threefold_astigmatism_orientation [Read only] float Measured threefold objective astigmatism angle in radians.
coma [Read only] float Measured coma in meters.
coma_orientation [Read only] float Measured coma angle in meters.
defocus [Read only] float Measured defocus in meters.
tableau_fit_error [Read only] float A value between 0 and 1 as a measure for quality: 0 = perfect fit, 1 = poor fit.
is_reliable [Read only] bool Indication whether the result is reliable.
