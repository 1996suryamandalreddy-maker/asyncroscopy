EdsSpectrumImageSettings.save_eds_event_stream

Defines whether EDS data events are to be saved. True by default.

Data type

[Optional] bool

Remarks

Disabling this option will prevent EDS data events from being stored individually in the output file. As a result, the file size is significantly reduced, allowing for faster transportation to the client. However, the data will only be accessible in a compressed format using proprietary compression algorithm. This means that only Velox can be used for further analysis of the acquired data. Additionally, certain analysis features, such as frame removal, will no longer be possible without the individual data events.

When EDS events are enabled, one summed spectrum per segment (stream) is also saved to the EMD file.
