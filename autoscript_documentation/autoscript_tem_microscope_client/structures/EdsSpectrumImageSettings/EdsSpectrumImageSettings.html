EdsSpectrumImageSettings

Contains settings for EDS spectrum image acquisition.

Items

eds_detector str The name of the EDS detector. Enumeration EdsDetectorType can be used.
dispersion float Dispersion in eV.
shaping_time float Shaping time in seconds.
dwell_time float Scan dwell time in seconds.
size [Optional] int Square image size in pixels. Enumeration ImageSize can be used.
region [Optional] Region Acquisition area to be scanned.
scanning_detectors [Optional] List[str] The name(s) of the STEM detector(s). Enumeration DetectorType can be used.
scanning_detector_segments [Optional] List[DetectorSegment] The detector segment(s) used for the acquisition.
eds_detector_segments [Optional] List[EdsDetectorSegment] The list of enabled EDS detector segments. If not provided, all segments are assumed.
save_eds_event_stream [Optional] bool Defines whether EDS data events are to be saved. True by default.

Remarks

This structure contains a set of settings used for the EDS spectrum image acquisition. Not all settings are intercompatible. A valid Settings structure must adhere to the following criteria:

  • dwell_time, shaping_time, and dispersion must be specified.
  • Either the size or the region must be specified. If both are specified, the region must fit into the size dimensions. If the size is not specified, the maximum scan size of the system is assumed.
  • region has to contain exactly one shape.
  • Either scanning_detectors or scanning_detector_segments must be specified. If both are given, they cannot overlap. In other words, segments from a detector listed in scanning detectors can't be used.
  • eds_detector must be specified, configuring the EDS segments is optional.

Scan region

When a region is specified, both the scanning area and scanning time are reduced. This technique can be utilized for optimization purposes. Additionally, the resulting images will have a smaller size, as they will be cropped to match the specified region area.
