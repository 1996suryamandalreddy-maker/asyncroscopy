MoveToPixelSettings.use_safe_stage_moves

Whether the stage should be moved safely by setting the tilt to zero before moving the stage to avoid pole touches. True by default.

Data type

[Optional] bool

Remarks

The function uses safe stage moves by default, but in some cases, it may be necessary to use unsafe moves because the safe move may cause some shift of the specimen. It is recommended to use this option with caution.
