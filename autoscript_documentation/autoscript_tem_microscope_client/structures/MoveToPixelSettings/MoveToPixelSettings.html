MoveToPixelSettings

Additional settings for move to pixel functionality.

Items

image_size [Optional] int Square image size in pixels. Enumeration ImageSize can be used. Default value is the maximum size supported by the camera (with the given fixed readout area in mind), or the maximum supported scanning resolution.
fixed_readout_area [Optional] str Readout area used to acquire the image (camera acquisition only). Enumeration FixedReadoutArea can be used. FULL by default.
method [Optional] str Method of the move to pixel. Enumeration MoveToPixelMethod can be used. Default value is STAGE.
use_safe_stage_moves [Optional] bool Whether the stage should be moved safely by setting the tilt to zero before moving the stage to avoid pole touches. True by default.
