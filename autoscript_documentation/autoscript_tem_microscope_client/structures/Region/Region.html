Region

Defines a generic region shape.

Items

coordinate_system [Optional] str Specifies Region's unit. Values from enumeration RegionCoordinateSystem are used.
rectangle Get: [Optional] Rectangle
Set: [Optional] Rectangle | list | tuple		 A Rectangular region.

Remarks

Region is a generic shape that can be used to define a scanning area. Exactly one of the items must be specified. Currently, only rectangles are supported, but it is expected that the options will expand in the future.

Precision

  • If PIXELS coordinates are used and float values are provided, the rectangle values are floored.
  • If RELATIVE coordinates are used, the actual size is calculated as relative_value * size. The result is then floored. If the size is not specified, the maximum scan size is assumed.
