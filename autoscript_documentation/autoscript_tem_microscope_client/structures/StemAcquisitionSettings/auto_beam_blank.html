StemAcquisitionSettings.auto_beam_blank

Automatic beam blanking. False by default.
[The property is deprecated and will be removed in version 1.21]

Data type

[Optional] bool

Remarks

Systems equipped with the NanoPulser hardware may utilize the auto_beam_blank.mode setting to set automatic blanking. The auto_beam_blank context manager can be used instead of setting the auto_beam_blank property or the blank() and unblank() methods may be called manually.

Examples


        from autoscript_tem_toolkit.context_managers import blanker_context

        settings = StemAcquisitionSettings(
            1e-6,
            detector_types=[DetectorType.HAADF, DetectorType.BF_S],
            size=1024)

        with blanker_context(microscope.optics.blanker):
            images = microscope.acquisition.acquire_stem_images_advanced(settings)
