StemAcquisitionSettings

Contains advanced settings for scanning detector image acquisition.

Items

dwell_time float STEM dwell time in seconds.
detector_types [Optional] List[str] The name(s) of the detector(s) used for the acquisition. Enumeration DetectorType can be used.
detector_segments [Optional] List[DetectorSegment] The detector segment(s) used for the acquisition.
region [Optional] Region Acquisition area to be scanned. Use Region.
size [Optional] int Square image size in pixels. Enumeration ImageSize can be used.
auto_beam_blank [Optional] bool Automatic beam blanking. False by default.
[The property is deprecated and will be removed in version 1.21]

Remarks

This structure contains a set of settings used for the advanced stem acquisition. Not all settings are intercompatible. A valid Settings structure must adhere to the following criteria:

  • dwell_time must be specified.
  • Either size or region must be specified. If both are specified, the region must fit into the size dimensions. If the size is not specified, the maximum scan size of the system is assumed.
  • region has to contain exactly one shape.
  • Either detector_types or detector_segments must be specified. If both are given, they cannot overlap. In other words, segments from a detector listed in detector types can't be used.

Beware that auto beam blanking is False by default.

When a region is specified, both the scanning area and scanning time are reduced. This technique can be utilized for optimization purposes. Additionally, the resulting images will have a smaller size, as they will be cropped to match the specified region area.
