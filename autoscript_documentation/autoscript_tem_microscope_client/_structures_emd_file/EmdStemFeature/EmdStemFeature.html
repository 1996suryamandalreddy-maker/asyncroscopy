EmdStemFeature

Items

images List[AdornedImage] List of AdornedImage objects representing the frames acquired by the detector.

Remarks

Each AdornedImage needs to satisfy metadata requirements for the EMD file to be valid. These metadata are already provided when acquiring images using the AutoScript functionality (e.g., acquire_stem_images). So you can directly save the images to an EMD file. If any required metada are missing, exception is thrown when saving the EMD file.

Examples

To save EmdStemFeature to an EMD file, you can use the static function . 


        stem_image = microscope.acquisition.acquire_stem_image(DetectorType.HAADF, 1024, 1e-6)

        # Create an emd file and save the image to it
        EmdFile.create('C:\\stem_image.emd', EmdStemFeature([stem_image])).close()
