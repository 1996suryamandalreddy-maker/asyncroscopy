EmdFile.read_all_images

Reads all images from the Emd file and converts them to AdornedImage.

Syntax

read_all_images()

Return value

List[AdornedImage] List of images as AdornedImage objects.

Remarks

The method parses the Emd file and reads all images with supported pixel formats. Those are uint8, uint16, int16, uint32 and int32.
An exception is not thrown if the file contains an unsupported image.

Examples


        # Open emd file and read all images
        with EmdFile.open(r"C:\emd_file.emd") as emd_file:
            images = emd_file.read_all_images()
            
            # Print xml metadata from all images
            for image in images:
                print(image.metadata.metadata_as_xml)
