EmdFile.convert

Converts an Emd experiment file to a new format version.

Syntax

convert(input_path, output_path)

Parameters

input_path str Path to the Emd file.
output_path str Path to the output Emd file.

Remarks

The function raises an error if the output file already exists.

Examples

The following example shows how to convert an EMD file to a new format and open it:


        old_file = r"C:\old_emd_file.emd"
        new_file = r"C:\new_emd_file.emd"

        EmdFile.convert(old_file, new_file)
        EmdFile.open(new_file)
