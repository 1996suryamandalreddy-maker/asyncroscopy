EmdFile.close

Closes the Emd file and releases all resources.

Syntax

close()

Remarks

After opening the file, AutoScript keeps it locked until the script ends or the file is closed. The close() function can be called explicitly, or the with statement can be used.

Examples

The following example shows implicit resource releasing using the with statement:


        # Open emd file and read all images
        with EmdFile.open(r"C:\emd_file.emd") as emd_file:
            images = emd_file.read_all_images()
            
        # The file is closed automatically when the with statement ends

The next example demonstrates releasing the file explicitly:


        # Open emd file and read all images
        emd_file = EmdFile.open(r"C:\emd_file.emd")
        images = emd_file.read_all_images()
        
        # Close the file by calling the close method
        emd_file.close()

Attempting to read the file multiple times at once will raise an exception:


        path = r"C:\emd_file.emd"
        
        emd_file1 = EmdFile(path)
        emd_file2 = EmdFile(path)  # Raises an error, emd_file1 is not closed
