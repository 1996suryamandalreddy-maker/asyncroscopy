EmdFile.open

Opens an Emd experiment file.

Syntax

open(path)
open(path, read_only)

Parameters

path str Path to the Emd file.
read_only bool Indicates whether the file should be opened in read-only mode. False by default

Return value

EmdFile Opened Emd file as EmdFile object.

Remarks

After opening the file, AutoScript will keep it locked until the script ends or until the file is closed explicitly. See close() for more information.

See the Emd file format page or the convert() function for more information about supported Emd file formats.

Examples

To open an Emd file you can use the static function open() or a constructor. The two options are equivalent. 


        path = r"C:\emd_file.emd"

        # Open emd file with open()
        emd_file = EmdFile.open(path)

        # Open emd file with constructor
        emd_file = EmdFile(path)

To open an emd file in read-only mode to prevent accidental changes, use the read_only parameter. 


        path = r"C:\emd_file.emd"

        # Open emd file in read-only mode with open()
        emd_file = EmdFile.open(path, read_only=True)

        # Open emd file in read-only mode with constructor
        emd_file = EmdFile(path, read_only=True)
