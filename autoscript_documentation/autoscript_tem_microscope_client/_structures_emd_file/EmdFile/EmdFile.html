EmdFile

Describes one Emd experiment file.

Methods

close() Closes the Emd file and releases all resources.
convert() Converts an Emd experiment file to a new format version.
create() Creates a new Emd experiment file.
open() Opens an Emd experiment file.
read_all_images() Reads all images from the Emd file and converts them to AdornedImage.

Remarks

The Emd file holds experiment data from Velox or Metrios applications. It has *.emd file extension. AutoScript can read images from the file and convert them to AdornedImage objects.

Note that not all Emd file formats are supported. The format evolves with new versions of Velox and Metrios applications. It is being optimized for real-time data storage use cases, and its structure is being extended to accommodate new data types. In AutoScript, support for older Emd formats is limited. See the Emd file format page or the convert() function for more information.

The current functionality is limited to reading and writing images and metadata from the Emd file. See snippets for examples of how to read spectra from the Emd file.

Examples

To open an Emd file you can use the static function open() or a constructor. The two options are equivalent.


        path = r"C:\emd_file.emd"
        
        # Open emd file with open()
        emd_file = EmdFile.open(path)
        
        # Open emd file with constructor
        emd_file = EmdFile(path)
