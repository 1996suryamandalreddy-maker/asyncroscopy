auto_functions

Enables execution of the microscope autofunctions.

Methods

run_auto_coma_correction() Performs automatic coma correction. Available only in TEM mode.
run_beam_tilt_auto_focus() Runs a routine to automatically focus using the beam tilt method. Available only in TEM mode.
run_objective_auto_stigmator() Performs automatic defocus and objective astigmatism correction. Available only in TEM mode.
run_opti_stem() Runs OptiStem corrections with the specified parameters. Only available in STEM mode.
run_stem_auto_focus() Runs a routine to automatically focus in STEM optical mode.

Remarks

Functions in this group on AutoScript API are considered experimental.

The function signatures are not guaranteed to be stable across AutoScript product versions. Behavior of the functions may vary depending on TEM system type and server software version. Both function signatures and behavior may evolve as new, optimized and more robust autofunctions are developed.

Each autofunction is designed to operate under specific optical conditions and with specific sample types. The functions may fail under other conditions or with other samples. For troubleshooting purposes each autofunction provides a return value that contains diagnostic information.

Autofunctions do not respect the configured auto beam blank mode, may manipulate the blanker manually and not return to the initial state which may therefore expose the sample for longer periods of time. Some autofunction's preconditions require the beam to be unblanked in which case the auto beam blank has to be set to OFF.
