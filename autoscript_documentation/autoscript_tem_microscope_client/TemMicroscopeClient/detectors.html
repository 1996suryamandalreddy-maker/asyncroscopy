detectors

Exposes the detectors that are available and can be controlled by scripting.

Classes

screen The main fluorescent screen of the microscope.

Properties

camera_detectors [Read only] Returns the list of available camera detectors on the microscope. Uses values from CameraType.
eds_detectors [Read only] Returns the list of available EDS detectors on the microscope. Uses values from EdsDetectorType.
eels_detectors [Read only] Returns the list of available EELS detectors on the microscope. Uses values from EelsDetectorType.
scanning_detectors [Read only] Returns the list of available scanning detectors on the microscope. Uses values from DetectorType.

Methods

get_camera_detector() Retrieves a camera detector by name.
get_eds_detector() Retrieves an EDS detector by name.
get_eels_detector() Retrieves an EELS detector by name.
get_scanning_detector() Retrieves a scanning detector by name.

Examples

Acquire an image from all cameras on the microscope and save the images: 

    for camera in microscope.detectors.camera_detectors:
        image = microscope.acquisition.acquire_camera_image(camera, 1024, 1e-3)
        image.save("C:/test/" + camera)
