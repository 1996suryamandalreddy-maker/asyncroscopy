optics

The microscope optical settings.

Classes

acceleration_voltage Provides access to electron beam acceleration voltage.
aperture_mechanisms The collection of aperture mechanisms on the microscope.
beam_stopper Provides access to the beam stopper device.
blanker Blanker group contains controls responsible for controlling the electron beam.
camera_length Provides access to the camera lengths of the microscope.
deflectors Provides access to deflector lens.
energy_filter Provides access to energy filter (imaging filter) settings.
focusing Allows you to control the focusing mechanisms of the microscope.
lenses Provides a direct access to the lenses of the microscope.
magnification Provides access to the TEM magnifications of the microscope.
monochromator Provides access to the monochromator.
stigmators Provides access to stigmators.

Properties

beam_diameter The beam diameter in meters. Only available on systems with three condenser lenses.
convergence_angle The beam convergence angle in radians. Only available on systems with three condenser lenses.
defocus Defocus value is focus with an offset (in meters).
[The property is deprecated and will be removed in version 1.17]
focus Focus value of the main imaging lens (in meters).
[The property is deprecated and will be removed in version 1.17]
illumination_mode The illumination mode of the condenser system. Maps to IlluminationMode enumeration. Only available on systems with three condenser lenses.
intensity Intensity value (not available in STEM mode).
is_accelerator_on [Read only] Returns true if the high voltage accelerator is on.
is_beam_blanked [Read only] Returns true when the beam blanker is active.
[The property is deprecated and will be removed in version 1.21]
is_eftem_on The EFTEM (Energy Filtered TEM) mode.
is_holography_mode_on The holography mode.
is_xl_mode_on The X Lens mode.
objective_lens_mode The objective lens mode. Maps to enumeration ObjectiveLensMode.
optical_mode The optical mode of the microscope. Maps to OpticalMode enumeration.
paused_scan_beam_position The position of the beam in the full scan area when the scan is paused. Range: -1 to +1 for x and y.
probe_mode The probe mode of the condenser system. Maps to ProbeMode enumeration.
projector_mode The mode of the projector. Maps to ProjectorMode enumeration.
scan_field_of_view The field of view of the scan in meters. Applicable only in STEM mode.
scan_rotation The rotation of the scan in radians. Applicable only in STEM mode.
spot_size_index Index of currently selected spot size.

Methods

blank() Blanks the beam. When blanked, the sample is not exposed to the beam.
[The method is deprecated and will be removed in version 1.21]
reset_defocus() [The method is deprecated and will be removed in version 1.17]
set_projector() Sets the projector to magnification or camera length using information encoded in a text label.
unblank() Unblanks the beam. When unblanked, the sample is exposed to the beam.
[The method is deprecated and will be removed in version 1.21]
