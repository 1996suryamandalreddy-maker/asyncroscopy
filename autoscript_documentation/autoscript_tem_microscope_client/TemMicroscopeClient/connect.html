TemMicroscopeClient.connect

Connects the AutoScript client to the AutoScript server.

Syntax

connect()
connect(host, port)

Remarks

Repeated connections are not allowed. To connect to another host, the disconect function must be called first.

Examples

The following code snippet shows how to initialize TemMicroscopeClient and connect it to a running instance of the AutoScript server.
Calling the connect method without parameters will use the default microscope computer address of 192.168.0.1. 


        # Create TemMicroscopeClient and connect it to the server running at 192.168.0.1
        microscope = TemMicroscopeClient()
        microscope.connect()

A custom IP address can be provided to the connect method:


        # Connect to the server with a custom IP address of 10.140.15.1
        microscope.connect("10.140.15.1")

Custom host names are also supported:


        # Connect to the server running at a computer with the hostname "TALOS": 
        microscope.connect("TALOS")

If the server is running on a non-standard port, it must be specified:


        # Connect to the server at IP address 10.140.15.1 and port number 6500
        microscope.connect("10.140.15.1", 6500)

The Connect method also accepts arrays of host addresses:


        # Connect to the server at address 10.140.15.1. If the connection fails, try localhost and 192.168.0.1 consecutively.
        microscope.connect(["10.140.15.1", "localhost", "192.168.0.1"])

Each IP address in the array can also have its own port number (lengths of both arrays must be equal):


        # Connect to the server at address 10.140.15.1:7500. If the connection fails, try localhost:4000
        microscope.connect(["10.140.15.1", "localhost"], [7500, 4000])
