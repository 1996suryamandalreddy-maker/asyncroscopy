AdornedImageMetadataScanSettings

Image metadata structure for scan settings values.

Items

scan_rotation [Optional] float Scan rotation, in radians.
line_integration_count [Optional] int The average number of lines before proceeding to the next line.
line_interlacing [Optional] int The number of partial frames to get one total frame.
mains_lock_on [Optional] bool If on, the image line or frame scan frequency is synchronized with the power supply frequency.
scan_size Get: [Optional] Point
Set: [Optional] Point | list | tuple		 Total size of the frame in pixels.
acquisition_time [Optional] float Total time (duration) taken to acquire the image/result, in seconds.
dwell_time [Optional] float Time to scan one pixel, in seconds.
frame_time [Optional] float Time to scan one frame, including flyback times, in seconds.
line_time [Optional] float Time to scan one line, including flyback time, in seconds.
scan_area Get: [Optional] Rectangle
Set: [Optional] Rectangle | list | tuple		 Rectangular area of the scan, in pixels.
