AdornedImageMetadata.get_custom_property

Retrieves custom metadata entries from the metadata object

Syntax

get_custom_property(name)
get_custom_property(name, scope)

Parameters

name
scope

Return value

AdornedImageMetadataCustomProperty

Remarks

Retrieves a custom metadata entry by its name from a specified scope. By default searches the empty string scope "" as this is where most custom metadata are saved to. If the entry does not exist, raises a ValueError.

Examples


        beam_modulator = microscope.optics.blanker.beam_modulator

        with beam_modulator_context(beam_modulator, duty_cycle=0.2, frequency=1e6):
            # Acquire image 
            image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 1024, 1)

            # Get the duty cycle custom property from the default scope of the image metadata
            duty_cycle_property = image.metadata.get_custom_property("Optics.BeamModulator.DutyCycle")
            if duty_cycle_property.type == "double":
                print(f"Duty cycle: {duty_cycle_property.value}")
                duty_cycle = float(duty_cycle_property.value)
