AdornedImageMetadata

A structure representing metadata of AdornedImage.

Items

core [Optional] AdornedImageMetadataCore

metadata_as_xml [Read only] str Provides all available metadata in single XML file.

Methods

create_from_xml() Creates new metadata structure from an XML string.
get_custom_property() Retrieves custom metadata entries from the metadata object

Remarks

The metadata format is designed to be flexible and extensible, supporting multiple types of systems (TEM, SEM, FIB, etc.). Its naming convention and overall design are intentionally generic to accommodate this variety. Extensibility is achieved through custom properties elements, which can contain any data (see AdornedImageMetadataCustomProperty), or custom sections, which can contain any XML element (see AdornedImageMetadataCustomSection).

AutoScript presents the metadata as an object tree, though the underlying format is XML. The XML format is used for serialization and saving metadata into an image, as well as for transferring metadata between client and server. Serialization can also be explicitly triggered by calling the metadata_as_xml property.

Since the metadata format is used across multiple ThermoFisher systems and applications, AutoScript is not the owner of the format. This also explains slight inconsistencies in the naming conventions between the AutoScript API and the metadata format. Any AutoScript-specific metadata are saved in custom properties or custom sections.

Examples

All parts of metadata objects can be modified:


        # Acquire new image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 1024, 1)

        # Modify image metadata object
        image.metadata.stage_settings.stage_position = AdornedImageMetadataStagePosition(0, 0, 0, 0, 0)
        image.metadata.binary_result.image_size = Point(1024, 1024)
        image.metadata.acquisition.acquisition_date_time = "2021-08-03T20:00:00+01:00"
        
        # Print modified metadata xml
        print(image.metadata.metadata_as_xml)
        
        # Save image to tiff (with modified metadata)
        image.save(r"C:\image_with_modified_metadata.tiff")

Metadata objects can also be created from scratch:


        metadata = AdornedImageMetadata()

        core = AdornedImageMetadataCore()
        core.application_software = "AutoScript"
        core.application_software_version = "1.2"
        metadata.core = core

        binary_result = AdornedImageMetadataBinaryResult(image_size=Point(1024, 1024), bits_per_pixel=16)
        metadata.binary_result = binary_result

        # Print created metadata as xml
        print(metadata.metadata_as_xml)
