AdornedImageMetadataBinaryResult

Image metadata structure for binary result values.

Items

acquisition_unit [Optional] str The basic unit of acquisition (Pixel, CameraImage, Spectrum).
bits_per_pixel [Optional] int Number of bits per pixel.
black_level [Optional] float Luminosity value in the image to display as the darkest level.
composition_type [Optional] str The method used to compose the acquisition units (Single, Series, Rectangle).
detector [Optional] str The name of the detector used for acquisition.
detector_index [Optional] int The index of the detector in the list of detectors.
digital_brightness [Optional] float Brightness applied to the image after acquisition and before saving.
digital_contrast [Optional] float Contrast applied to the image after acquisition and before saving.
encoding [Optional] str Encoding type of the image or spectrum (Unsigned, Signed, Float, RGB, BGR, Complex).
filter_frame_count [Optional] int Number of source frames combined using the averaging or integration filter.
filter_type [Optional] str Type of filter used (Average, Integrate, DriftCorrectedFrameIntegration, RecursiveAverage).
gamma [Optional] float Image adjustment gamma setting.
intensity_offset [Optional] float The value added to the scaled input value to determine actual intensity.
intensity_scale [Optional] float Multiplication factor to convert intensity values to other physical measurements if they do not represent intensity.

pixel_value_maximum [Optional] float Computed maximum numeric value of pixel values in the unprocessed image.

pixel_value_mean [Optional] float Computed average (mean) numeric value of pixel values in the unprocessed (grayscale) image.
pixel_value_standard_deviation [Optional] float Computed standard deviation of pixel values in the unprocessed image.
recursive_filter_coefficient [Optional] float User-selected value that describes the filtering.
