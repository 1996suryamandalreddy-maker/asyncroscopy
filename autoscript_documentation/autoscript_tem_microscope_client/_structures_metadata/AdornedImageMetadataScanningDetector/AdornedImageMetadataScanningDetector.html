AdornedImageMetadataScanningDetector

Image metadata structure representing a scanning detector.

Items

detector_name [Optional] str String indicating the type of detector.
detector_type [Optional] str Unique identifier for the detector model that can be used to cross-reference external data such as quantification properties.
enabled [Optional] bool Indicates if the detector is switched on, which may influence the optics.
inserted [Optional] bool Indicates if the detector is inserted.
mix_contribution [Optional] float The amount this detector's signal contributes to the final result. -1 means inverted. Can be a proportion. If there is only one signal, it would be 1.0.
signal [Optional] str A setting on the detector to help determine the difference between backscatter and secondary electrons in the ETD detector, for example.
segments [Optional] str Mathematical expression of segment contributions where the coefficient is assumed to be 1.0 unless otherwise noted.
gain [Optional] float Detector gain in decibels.
offset [Optional] float Detector offset in volts, amperes, or native units.
enhanced_gain [Optional] float Additional signal boost after gain and offset are applied, in decibels.
grid_voltage [Optional] float Voltage of the grid on the ETD/CDEM/ICE in volts.
suction_tube_voltage [Optional] float Voltage of the suction tube on the TLD in volts.
front_end_voltage [Optional] float Voltage of the ESEM/CDEM gaseous detector front end in volts.
converter_electrode_voltage [Optional] float Voltage of the converter electrode of the ICE detector in volts.
scintillator_voltage [Optional] float Voltage on the scintillator for ETD/ICE in volts.
contrast_normalized [Optional] float Contrast voltage converted to decibels then normalized to a 0-100 scale.
brightness_normalized [Optional] float Value applied to the detector inducing brightness then converted to a 0-100 scale.
collection_angle_range [Optional] Limits Collection angle range in radians.
custom_properties [Optional] List[AdornedImageMetadataCustomProperties]
custom_sections [Optional] List[AdornedImageMetadataCustomSection]
