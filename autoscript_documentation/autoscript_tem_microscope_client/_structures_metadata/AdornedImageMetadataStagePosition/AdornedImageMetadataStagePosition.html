AdornedImageMetadataStagePosition

Image metadata structure representing a stage position.

Items

x [Optional] float X coordinate in meters.
y [Optional] float Y coordinate in meters.
z [Optional] float Z coordinate in meters.
a [Optional] float Alpha angle in radians.
b [Optional] float Beta angle in radians.
r [Optional] float Rotation angle in radians.

Methods

create_from() Creates a new instance of the structure from a collection object.

Overloaded operators

Indexer Lower or upper case names, values from StageAxis enumerations, or integers can be used as an index: 'x' == 'X' == StageAxis.X == 0. Integer indexes have following order: x, y, z, a, b, r.
Addition A scalar value or a position-like object is supported (indexable object with up to 6 values).
Subtraction A scalar value or a position-like object is supported (indexable object with up to 6 values).
Multiplication A scalar value is supported.
Division A scalar value is supported.
Floor division A scalar value is supported.
Equality A position-like object is supported (indexable object with up to 6 values). Tests approximate equality with maximum tolerance of 1e-6.

Remarks

This structure inherits from StagePosition structure so it can be used instead. For conversion in the opposite direction, use create_from().

Tilt values should follow an inclusive interval of -3.15 to 3.15.

Examples

Example of using AdornedImageMetadataStagePosition instead of StagePosition:


        # grab stage position from the image metadata
        stage_position = image.metadata.stage_settings.stage_position
        # move the stage to the position
        microscope.specimen.stage.absolute_move(stage_position)

Overloaded operator examples:


        position_a = AdornedImageMetadataStagePosition(2, 3, 4, 5, 6, 7)
        position_b = AdornedImageMetadataStagePosition(1, 2, 3, 4, 5, 6)

        print(position_a[2])            # 4
        print(position_a[StageAxis.Y]   # 3
        print(position_a["r"]           # 7
        print(position_a + position_b)  # AdornedImageMetadataStagePosition(3, 5, 7, 9, 11, 13)
        print(position_a + 5)           # AdornedImageMetadataStagePosition(7, 8, 9, 10, 11, 12)
        print(position_a - position_b)  # AdornedImageMetadataStagePosition(1, 1, 1, 1, 1, 1)
        print(position_a - 3)           # AdornedImageMetadataStagePosition(-1, 0, 1, 2, 3, 4)
        print(position_a * 2)           # AdornedImageMetadataStagePosition(4, 6, 8, 10, 12, 14)
        print(position_a / 2)           # AdornedImageMetadataStagePosition(1, 1.5, 2, 2.5, 3, 3.5)
        print(position_a // 3)          # AdornedImageMetadataStagePosition(0, 1, 1, 1, 2, 2)
        print(position_a == position_b) # False
