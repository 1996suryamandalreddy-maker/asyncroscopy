AdornedImageMetadataStagePosition.create_from

Creates a new instance of the structure from a collection object.

Syntax

create_from(value)

Parameters

value

Return value

AdornedImageMetadataStagePosition

Remarks

The collection object can be either a list or a tuple with up to six values, or another instance of AdornedImageMetadataStagePosition or StagePosition. The function always creates a new object.

Examples


        # created from a list with a length of six
        AdornedImageMetadataStagePosition.create_from([0, 1, 2, 3, 4, 5])

        # created from a tuple with a length of five (r is None)
        AdornedImageMetadataStagePosition.create_from((0, 1, 2, 3, 4))

        # created from a StagePosition (r is None)
        AdornedImageMetadataStagePosition.create_from(StagePosition(1, 2, 3, 4, 5))

        # created from another AdornedImageMetadataStagePosition
        AdornedImageMetadataStagePosition.create_from(AdornedImageMetadataStagePosition(0, 1, 2, 3, 4, 5))
