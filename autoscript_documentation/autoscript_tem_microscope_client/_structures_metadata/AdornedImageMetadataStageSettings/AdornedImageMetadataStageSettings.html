AdornedImageMetadataStageSettings

Image metadata structure for stage settings values.

Items

holder_temperature [Optional] float Holder temperature, in Kelvin.
holder_type [Optional] str Identification of the type of specimen holder used.
sample_loader [Optional] str The type of sample loader mounted on the stage (None, MultiLoader, AutoLoader, AutoLoaderCryo).
stage_position Get: [Optional] AdornedImageMetadataStagePosition
Set: [Optional] AdornedImageMetadataStagePosition | list | tuple		 The position of the stage.
