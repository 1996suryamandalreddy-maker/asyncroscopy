AdornedImageMetadataCore

Image metadata structure for core metadata values.

Items

application_computer_name [Optional] str Represents the computer that the application software is running on, which could be the same or different from the acquisition and control software.
application_software [Optional] str The application the user interacts with to set up the acquisition and save the image.
application_software_version [Optional] str A 4-part version number of the application (#.#.#.#).
comment [Optional] str User-provided description or comment.
file_date_time [Optional] str Date and time when the result file is created.
file_name [Optional] str Filename and extension (but not path) of the pixel content (binary data) described by this metadata.
guid [Optional] str Unique identifier of the image.
md5_checksum [Optional] bytes MD5 hash of the binary result data.
parent_guid [Optional] str Unique identifier of the image from which this data was derived.
private_checksum [Optional] bytes Vendor-specific checksum.
user_id [Optional] str The user ID or username of the user running the process that acquired the image.
