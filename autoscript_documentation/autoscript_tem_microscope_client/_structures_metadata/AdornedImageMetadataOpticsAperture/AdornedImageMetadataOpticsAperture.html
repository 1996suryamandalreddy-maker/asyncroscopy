AdornedImageMetadataOpticsAperture

Image metadata structure representing an aperture.

Items

diameter [Optional] float Diameter, in meters.
index [Optional] int Numeric indicator of the selected aperture.
mechanism_type [Optional] str The type of aperture mechanism (Unknown, Manual, Motorized).
name [Optional] str The name of the aperture (C1, C2, C3, Objective, SelectedArea, BDA, BAA).
number [Optional] int The sequence number of the aperture mechanisms, starting at 1 for the topmost aperture mechanism in the column.
type [Optional] str Type of aperture (Unknown, Circular, Biprism, EnergySlit, FaradayCup, PhasePlate).
position_offset Get: [Optional] Point
Set: [Optional] Point | list | tuple		 Position offset from the align position, in meters.
