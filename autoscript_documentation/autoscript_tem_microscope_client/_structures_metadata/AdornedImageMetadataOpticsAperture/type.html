Type of aperture (Unknown, Circular, Biprism, EnergySlit, FaradayCup, PhasePlate).

Data type

[Optional] str

Remarks

Type is an enumeration. The value should be equal to one of the following:

  • Slit
  • Cicular
  • Biprism
  • Retracted
  • None
  • EnergySlit
  • FaradayCup
  • Unknown
