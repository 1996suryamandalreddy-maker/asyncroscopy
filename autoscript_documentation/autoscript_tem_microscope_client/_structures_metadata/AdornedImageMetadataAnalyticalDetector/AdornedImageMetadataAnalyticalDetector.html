AdornedImageMetadataAnalyticalDetector

Image metadata structure representing an analytical detector.

Items

detector_name [Optional] str String indicating the type of detector.
detector_type [Optional] str Unique identifier of the detector model.
enabled [Optional] bool Indicates if the detector is switched on, which may influence the optics.
inserted [Optional] bool Indicates if the detector is inserted.
mix_contribution [Optional] float The amount this detector's signal contributes to the final result. 1 means only one signal, -1 means inverted.
elevation_angle [Optional] float The angle between the center of the detector and the horizontal plane, in radians.
azimuth_angle [Optional] float The azimuthal angle between the center of the detector and the primary tilt axis, in radians.
collection_angle [Optional] float The solid angle subtended by the active area of the detector at the sample, in radians.
dispersion [Optional] float Dispersion setting indicating electronvolts per channel, or energy range per channel.
pulse_process_time [Optional] float Time, in seconds, to shape a pulse during which no other pulses can be accepted.
real_time [Optional] float Total time, in seconds, for the acquisition which includes live time and dead time.
live_time [Optional] float The effective acquisition time, in seconds. Real time minus dead time.
input_count_rate [Optional] float Number of photons per second (hertz) detected during input.
output_count_rate [Optional] float Number of photons per second (hertz) counted.
binning Get: [Optional] Point
Set: [Optional] Point | list | tuple		 Number of pixels (in X, Y respectively) whose combined signal produces one pixel output.
shutters Get: [Optional] List[ValueAndType]
Set: [Optional] List[ValueAndType | list | tuple]		 List indicating which shutter types were in effect for detection.
shutter_state [Optional] str The state of the mechanical shutter.
offset_energy [Optional] float The start energy of bin zero, in volts.
electronics_noise [Optional] float The noise in the spectrum caused by the electronics, in electronvolts. A typical value is 30.
custom_properties [Optional] List[AdornedImageMetadataCustomProperties]
custom_sections [Optional] List[AdornedImageMetadataCustomSection]
