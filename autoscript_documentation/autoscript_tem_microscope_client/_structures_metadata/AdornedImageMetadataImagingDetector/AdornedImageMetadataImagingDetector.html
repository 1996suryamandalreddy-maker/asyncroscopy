AdornedImageMetadataImagingDetector

Image metadata structure representing an imaging detector.

Items

detector_name [Optional] str String indicating the type of detector.
detector_type [Optional] str Unique identifier of the detector model.
enabled [Optional] bool Indicates if the detector is switched on, which may influence the optics.
inserted [Optional] bool Indicates whether the detector is inserted.
exposure_mode [Optional] str Type of exposure for imaging detector (Single, Double).
gain [Optional] float Gain of the CCD camera in decibels.
binning Get: [Optional] Point
Set: [Optional] Point | list | tuple		 Number of pixels (in X, Y respectively) whose combined signal produces one pixel output.
readout_area Get: [Optional] Rectangle
Set: [Optional] Rectangle | list | tuple		 Readout area of the CCD in pixels.
exposure_time [Optional] float Total time in seconds to expose one frame.
exposure_time_2 [Optional] float Second exposure time for dual exposure, in seconds.
pre_exposure_time [Optional] float Pre-exposure time, in seconds.
shutters Get: [Optional] List[ValueAndType]
Set: [Optional] List[ValueAndType | list | tuple]		 List indicating which shutter types were in effect for detection.
dark_gain_correction_type [Optional] str The type of dark gain correction applied (None, Dark, DarkGain).
pre_exposure_pause_time [Optional] float Time between pre-exposure and exposure, in seconds.
scintillator_pixel_size Get: [Optional] Point
Set: [Optional] Point | list | tuple		 The size of the CCD pixels at the scintillator, in meters.
post_magnification [Optional] float The post-magnification of the CCD camera with respect to the nominal magnification plane.
custom_properties [Optional] List[AdornedImageMetadataCustomProperties]
custom_sections [Optional] List[AdornedImageMetadataCustomSection]
