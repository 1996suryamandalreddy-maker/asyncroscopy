source.field_emission_gun.flash

Performs low temperature flashing unless high temperature is advised and sets flashing post action.

Syntax

flash()
flash(post_action)

Parameters

post_action str Post action which is performed after flashing. Enumeration FlashingPostAction can be used

Remarks

If post action is not set, either the default action RESTORE_OPTIMAL_BEAM_CURRENT or the last set post action is used.

See also

is_flashing_advised
