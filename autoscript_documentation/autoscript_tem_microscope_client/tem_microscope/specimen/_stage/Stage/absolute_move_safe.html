specimen.stage.absolute_move_safe

Moves the stage to a new position. To avoid pole touches the tilt is first set to zero.

Syntax

absolute_move_safe(position)

Parameters

position StagePosition | list | tuple The absolute position to which the stage should move.

Remarks

Axes with value None will not be affected by the move. The stage must be in a ready state for the move to be successful. If the stage is being manipulated by another application or process, or the stage is currently jogging, the error message 'Stage not ready after X ms' is returned. Property is_moving can be checked to make sure that the stage is not busy.

The function checks the limits of all affected axes before moving. You can use the get_axis_limits() function to get the limits.

Examples


        # Move X to 250 micrometers, Y to 300 micrometers and Z to 200 micrometers safely, do not move other axes
        microscope.specimen.stage.absolute_move_safe(StagePosition(x=250e-6, y=300e-6, z=200e-6))

        # Use a list to move X, Y, Z axes safely to the given coordinates
        microscope.specimen.stage.absolute_move_safe([250e-6, 300e-6, 200e-6])
