specimen.stage.start_jogging

Starts jogging at a given velocity. This function is asynchronous and returns immediately.

Syntax

start_jogging(jogging_velocity)

Parameters

jogging_velocity StageVelocity | list | tuple The velocity of the stage movement.

Remarks

The jogging vector can be set up in a single start_jogging call, or in multiple consecutive calls (e.g., one call per axis). If the jogging velocity is specified more than once for a single axis, the value is reset and only the last velocity is used.

While the stage is jogging, it cannot be manipulated by other moves. The jog needs to finish or be stopped first (see stop_jogging()). No errors are raised if the jog is finished by reaching the end of the stage.

Examples

The following example creates a simple jog with a given velocity vector and terminates it after a specified time has elapsed:


        import time

        # Start jogging with X, Y axes
        microscope.specimen.stage.start_jogging(StageVelocity(x=2.4e-10, y=4e-10))

        print(microscope.specimen.stage.is_moving) # True

        # Sleep for 10 seconds
        time.sleep(10)

        # Stop jogging on all axes
        microscope.specimen.stage.stop_jogging()

        print(microscope.specimen.stage.is_moving) # False

The next example shows that jogging velocity could be specified multiple times:


        # Start jogging with X axis
        microscope.specimen.stage.start_jogging(StageVelocity(x=2e-10))

        # Start jogging with Y axis, the X axis jogging is still going
        microscope.specimen.stage.start_jogging(StageVelocity(y=4e-10))

        # Reset X axis velocity with a new value
        microscope.specimen.stage.start_jogging(StageVelocity(x=3e-10))

A list or tuple can be used to specify the stage velocity:


        # Use a list to set the velocity on the X axis
        microscope.specimen.stage.start_jogging([1e-10])

        # Use a tuple to set the velocity on the X, Y, Z axes
        microscope.specimen.stage.start_jogging((1e-10, 1e-10, 1e-10))
