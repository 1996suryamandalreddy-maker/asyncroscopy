specimen.stage.relative_move_safe

Moves the stage relatively to the current stage position. To avoid pole touches the tilt is first set to zero.

Syntax

relative_move_safe(relative_position)

Parameters

relative_position StagePosition | list | tuple The delta position by which the stage should move.

Remarks

Axes with value None will not be affected by the move. The stage must be in a ready state for the move to be successful. If the stage is being manipulated by another application or process, or the stage is currently jogging, the error message 'Stage not ready after X ms' is returned. Property is_moving can be checked to make sure that the stage is not busy.

The function checks the limits of all affected axes before moving. You can use the get_axis_limits() function to get the limits.

Examples

Relative move moves the axis by addition. The following example moves x axis by 200 micrometers safely in two steps: 


        print(microscope.specimen.stage.position)  # StagePosition(x=0, y=0, z=0, a=0)

        # Step one, move by 100 micrometers in X
        microscope.specimen.stage.relative_move_safe(StagePosition(x=100e-6))  
        print(microscope.specimen.stage.position)  # StagePosition(x=100e-6, y=0, z=0, a=0)

        # Step two, move by 100 micrometers in X
        microscope.specimen.stage.relative_move_safe(StagePosition(x=100e-6))
        print(microscope.specimen.stage.position)  # StagePosition(x=200e-6, y=0, z=0, a=0)

Relative move can also move multiple axes at the same time. Tilt axes use radians, math.radians() function helps to convert from degrees: 


        import math

        # Move X, Y, Z by 100 micrometers and alpha and beta tilt by 5 degrees safely
        delta_position = StagePosition(x=100e-6, y=100e-6, z=100e-6, a=math.radians(5), b=math.radians(5))
        microscope.specimen.stage.relative_move_safe(delta_position)

A list or tuple can be used to specify the stage position:


        # Use a list to move X axis by the given coordinates
        microscope.specimen.stage.relative_move_safe([100e-6])

        # Use a tuple to move X, Y, Z axes by the given coordinates
        microscope.specimen.stage.relative_move_safe((100e-6, 100e-6, 100e-6))
