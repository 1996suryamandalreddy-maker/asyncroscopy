specimen.stage.get_axis_limits

Returns limits of a stage axis in meters or radians.

Syntax

get_axis_limits(axis)

Parameters

axis str The axis name. Enumeration StageAxis can be used.

Return value

Limits The Limits object containing limits of the specified stage axis.

Remarks

Only one axis can be specified at a time. The name of the axis is case insensitive, i.e. 'x', 'X', 'StageAxis.X' are all allowed.

Examples


        # Get limits for Y axis
        limits = microscope.specimen.stage.get_axis_limits(StageAxis.Y)
        
        # Print min and max values for Y axis
        print(f"min: {limits.min}, max: {limits.max}")
