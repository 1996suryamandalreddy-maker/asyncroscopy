specimen.stage.is_moving

Returns true if the stage is currently moving, jogging, or wobbling. False otherwise.

Data type

[Read only] bool

Examples


        import time

        # Start jogging with X, Y axes
        microscope.specimen.stage.start_jogging(StageVelocity(x=2.4e-10, y=4e-10))

        # Print stage position each second until the stage stops moving
        while microscope.specimen.stage.is_moving:
            print(microscope.specimen.stage.position)
            time.sleep(1)
