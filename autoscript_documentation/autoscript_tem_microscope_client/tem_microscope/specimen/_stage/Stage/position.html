specimen.stage.position

Returns the stage position (X, Y, Z) in meters, A and B in radians.

Data type

[Read only] StagePosition

Examples

The following example shows how to obtain all the axes of the stage position:


        position = microscope.specimen.stage.position

        print("X:", position.x)
        print("Y:", position.y)
        print("Z:", position.z)
        
        print("A:", position.a)
        print("B:", position.b) # None for single tilt holders
