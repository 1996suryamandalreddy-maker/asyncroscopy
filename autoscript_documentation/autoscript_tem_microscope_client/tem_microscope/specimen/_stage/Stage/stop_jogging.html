specimen.stage.stop_jogging

Stops jogging on all axes.

Syntax

stop_jogging()

Remarks

The function stops all jogs currently running. If no axis is jogging, the function does nothing. The function does not stop other stage moves (absolute_move(), relative_move(), etc.).

Examples

The following example creates a simple jog with a given velocity vector and terminates it after a specified time has elapsed:


        import time

        # Start jogging with Z axis
        microscope.specimen.stage.start_jogging(StageVelocity(z=2e-5))

        print(microscope.specimen.stage.is_moving) # True

        # Sleep for 10 seconds
        time.sleep(10)

        # Stop jogging on all axes
        microscope.specimen.stage.stop_jogging()

        print(microscope.specimen.stage.is_moving) # False
