specimen.piezo_stage.stop

Stops all movement on all axes.

Syntax

stop()

Examples

The following example shows the use of stop() function to timeout a waiting loop:


        import time

        # Start jogging with Y axis
        microscope.specimen.piezo_stage.start_jogging(StageVelocity(y=2e-6))

        # Declare timeout and loop_tick in seconds
        timeout = 10
        loop_tick = 0.5

        while microscope.specimen.piezo_stage.is_moving:
            # Sleep for the duration of the loop_tick, decrement timeout value
            time.sleep(loop_tick)
            timeout -= loop_tick

            # If the timeout is reached, stop the jogging - it will end the loop
            if timeout <= 0:
                microscope.specimen.piezo_stage.stop()

        # The piezo stage is no longer moving if the loop ended
        print("Jogging finished")
