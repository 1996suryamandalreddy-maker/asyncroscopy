specimen.piezo_stage.is_moving

Returns true if the piezo stage is currently moving (jogging velocity is non-zero), false otherwise.

Data type

[Read only] bool

Remarks

The value is updated only every 200ms and is cached in the meantime.

Examples

The following example shows the use of is_moving function in a simple waiting loop:


        import time

        # Start jogging with X axis
        microscope.specimen.piezo_stage.start_jogging(StageVelocity(2e-6))

        while microscope.specimen.piezo_stage.is_moving:
            # Sleep for half a second
            time.sleep(0.5)

        # The piezo stage is no longer moving if the loop ended
        print("Jogging finished")
