specimen.piezo_stage.relative_move

Moves the piezo stage relatively to the current position.

Syntax

relative_move(relative_position)
relative_move(relative_position, max_velocity)

Parameters

relative_position StagePosition | list | tuple The delta position by which the piezo stage should move.
max_velocity StageVelocity | list | tuple Maximum velocity of the relative move, default value is 2e-6 m/s.

Remarks

Axes with value None will not be affected by the move. The function checks the limits of all affected axes before moving. Use get_axis_limits() function to get the limits. If the velocity is specified, the dimensions of the specified velocity and the position must match (i.e. if the position specifies the X and Y axes, the velocity must also specify the X and Y axes).

Unlike start_jogging(), the relative move function is not asynchronous and the function is not finished until the movement is complete.

Examples

The next example shows relative move with and without specified velocity:


        # Relative move with default velocity
        microscope.specimen.piezo_stage.relative_move(StagePosition(1e-6, 1e-6, 0.5e-6))

        # Print the changed position
        print(microscope.specimen.piezo_stage.position)

        # Relative move with specified velocity
        microscope.specimen.piezo_stage.relative_move(StagePosition(1e-6, 1e-6, 0.5e-6), StageVelocity(1e-6, 1e-6, 1e-6))

        # Print the changed position
        print(microscope.specimen.piezo_stage.position)

        # A list or tuple can be used to specify piezo stage position or velocity
        microscope.specimen.piezo_stage.relative_move([1e-6, 1e-6, 0.5e-6], (1e-6, 1e-6, 1e-6))
