specimen.piezo_stage.reset_position

Resets the piezo stage to the specified position while compensating the stage position at the same time.

Syntax

reset_position()
reset_position(reset_position)
reset_position(reset_position, max_velocity)

Parameters

reset_position StagePosition | list | tuple The piezo stage reset position coordinates. Default value is the center of the stage.
max_velocity StageVelocity | list | tuple Maximum velocity of the reset move, default value is 2e-6 m/s.

Remarks

The reset moves the piezo stage to the specified position while the corresponding stage axis remains in closed loop. This will cause the image to "wobble" as the stage attempts to compensate for the piezo movement. After the reset is complete, the specimen should be moved only minimally while the piezo axis has assumed a new position. If multiple axes are specified, the resets are performed sequentially. Unlike start_jogging(), the reset position function is not asynchronous and the function is not finished until the reset is complete.

The specified velocity may affect the accuracy of the reset movement. Tests have shown that a speed of 2µm/s (2e-6 m/s) is a fairly good default. Lower velocities may result in more accurate motion (and vice versa). The maximum speed is 100µs/s, it corresponds to the maximum speed of the stage.

Axes with value None will not be affected by the reset. The function checks the limits of all affected axes before moving. Use get_axis_limits() function to get the limits. If the velocity is specified, the dimensions of the specified velocity and the position must match (i.e. if the position specifies the X and Y axes, the velocity must also specify the X and Y axes).

The default position is in the center of the stage, i.e. each available axis is calculated as follows: (axis_limits.min + axis_limits.max) / 2.

Examples

The following example shows how to reset the piezo stage to minimal value on each axis:


        # Fetch limits of each axis
        x_limits = microscope.specimen.piezo_stage.get_axis_limits(StageAxis.X)
        y_limits = microscope.specimen.piezo_stage.get_axis_limits(StageAxis.Y)
        z_limits = microscope.specimen.piezo_stage.get_axis_limits(StageAxis.Z)

        # Reset piezo stage position to minimal values with default velocity
        microscope.specimen.piezo_stage.reset_position(StagePosition(x_limits.min, y_limits.min, z_limits.min))

The next example shows how the reset velocity can be specified:


        # Reset piezo stage position with non-default velocity
        microscope.specimen.piezo_stage.reset_position(StagePosition(x=1e-6, y=1e-6), StageVelocity(x=1e-5, y=1e-5))

        # A list or tuple can be used to specify piezo stage position or velocity
        microscope.specimen.piezo_stage.reset_position([1e-6, 1e-6], (1e-5, 1e-5))
