specimen.piezo_stage.start_jogging

Starts jogging at a given velocity. This function is asynchronous and returns immediately.

Syntax

start_jogging(jogging_velocity)

Parameters

jogging_velocity StageVelocity | list | tuple The velocity of the piezo stage movement.

Remarks

The jogging vector can be set up in a single start_jogging call, or in multiple consecutive calls (e.g., one call per axis). If the jogging velocity is specified more than once for a single axis, the value is reset and only the last velocity is used.

Calling reset_position() or relative_move() stops the current jogging process on affected axes. The jog can also be explicitly stopped by calling stop(). If the jogging is not stopped, it will be aborted after it reaches the end of the stage. In this case no error will be raised.

Examples

The following example creates a simple jogging job with a given velocity vector:


        import time

        # Start jogging with X, Y axes
        microscope.specimen.piezo_stage.start_jogging(StageVelocity(x=2e-6, y=4e-6))

        while microscope.specimen.piezo_stage.is_moving:
            # Sleep for 0.25s
            time.sleep(0.25)

            # Print current velocity and position
            print(microscope.specimen.piezo_stage.jogging_velocity)
            print(microscope.specimen.piezo_stage.position)

        # The piezo stage is no longer moving if the loop ended
        print("Jogging finished")

The next example shows that jogging velocity could be specified multiple times:


        # Start jogging with X axis
        microscope.specimen.piezo_stage.start_jogging(StageVelocity(x=2e-6))

        # Start jogging with Y axis, the X axis jogging is still going
        microscope.specimen.piezo_stage.start_jogging(StageVelocity(y=4e-6))

        # Reset X axis velocity with a new value
        microscope.specimen.piezo_stage.start_jogging(StageVelocity(x=3e-6))

A list or tuple can be used to specify piezo stage velocity:


        # Use a list to specify the velocity
        microscope.specimen.piezo_stage.start_jogging([2e-6])

        # Use a tuple to specify the velocity
        microscope.specimen.piezo_stage.start_jogging((2e-6, 4e-6))
