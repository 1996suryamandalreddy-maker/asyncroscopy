analysis.eds.acquire_spectrum

Acquires a single EDS spectrum.

Syntax

acquire_spectrum(settings)

Parameters

settings EdsAcquisitionSettings EDS settings structure.

Return value

AdornedSpectrum The object containing both the EDS spectrum data and metadata describing the system state.

Remarks

The single spectrum acquisition only contains spectral data (1D array of energy values), no images are acquired. Information on which EDS segments were enabled during the acquisition can be found in the custom section of the spectrum metadata.

Examples

The following example shows EDS acquisition with the Super-X detector:


        # Get the EDS detector object
        eds_detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)

        # Configure the EDS acquisition
        settings = EdsAcquisitionSettings()
        settings.eds_detector = eds_detector.name
        settings.dispersion = eds_detector.dispersions[0]
        settings.shaping_time = eds_detector.shaping_times[0]
        settings.exposure_time = 3e-3
        settings.exposure_time_type = ExposureTimeType.REAL_TIME

        spectrum = microscope.analysis.eds.acquire_spectrum(settings)

The next example shows the usage of detector segments:


        # Get the EDS detector object
        eds_detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)

        # Configure the EDS acquisition - same as in previous example
        settings = EdsAcquisitionSettings()
        # ...

        # Configure the detector segments, enable first two segments only
        eds_detector.set_enabled_segments([EdsDetectorSegmentType.SEGMENT1, EdsDetectorSegmentType.SEGMENT2])
        settings.detector_segments = eds_detector.get_enabled_segments()

        # There is still only one spectrum result, but it was captured with only two segments
        spectrum = microscope.analysis.eds.acquire_spectrum(settings)
