analysis.eds.acquire_spectrum_image

Acquires EDS spectrum image.

Syntax

acquire_spectrum_image(settings)

Parameters

settings EdsSpectrumImageSettings EDS spectrum image settings structure.

Return value

AdornedSpectrumImage Object handle that provides a way to download the acquired data to the client.

Remarks

Spectrum image acquisition is not available in offline mode.

File representation

The acquired handle is associated with a file on the MPC disk. The handle is valid only for a limited amount of time. In the AutoScript startup sequence, the acquired files are checked and all files older than 24 hours are deleted.

The save() method can be used to download the acquired data to the client. This method saves the spectrum image in a Velox EMD format. Information on the compatibility of the EMD file format can be found on the Extras/EMD file format page in the reference manual.

File contents

The data is hierarchically structured and uses the binary data format HDF5. The Python package h5py can be used to read the file directly. Alternatively, the Python package hyperspy or the Velox application can be used to read the file and/or further analyze the data.

The data is organized as follows:

  • Compressed spectrum cube + metadata

    • Acquired EDS events are compressed into a single cube using ThermoFisher's proprietary compression algorithm. Only Velox can be used to read and analyze the cube. Metadata have the same structrure as AdornedImageMetadata but are stored in JSON format.

    Information on which EDS segments were enabled during the acquisition can be found in the custom section of the cube metadata.

  • STEM images + metadata

    • All STEM frames are stored with the corresponding metadata. Metadata have the same structrure as AdornedImageMetadata but are stored in JSON format.

    A STEM detector can be specified either through a detector type (using scanning_detectors parameter) or through a list of segments (using scanning_detector_segments parameter). While it is possible to combine these two settings, it's important to note that a detector cannot be simultaneously defined by both its segment and its type. Each detector and segment individually capture an image.

  • EDS event streams + metadata

    • The EDS event stream has the form of (usually) uint16 values. Each value represents a number of counts detected for a particular pixel. The maximum value of the data type (usually 65535) is referred as gate pulse and signifies a change to the next pixel. Most multi-segment detectors collect a separate stream for each segment of the detector. Each frame of the event stream contains the corresponding metadata. Metadata have the same structrure as AdornedImageMetadata but are stored in JSON format.

    Saving the (uncompressed) EDS event stream takes up a lot of memory. Setting the value of save_eds_event_stream setting deactivates saving.

Performance and limitations

To minimize acquisition time, it is recommended to restrict the scan region to a maximum of either 512x512 or 1024x1024 pixels. In our laboratory testing, we successfully acquired 500 frames without any data loss or signs of frame number limitations caused by processing bottlenecks. It appears that the only constraint on the number of frames is the resulting file size.

Examples

The following example shows EDS spectrum image acquisition with Super-X and HAADF detectors:


        # Get the EDS detector object
        eds_detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)

        # Configure the EDS spectrum image acquisition
        settings = EdsSpectrumImageSettings(eds_detector=eds_detector.name,
                                            dispersion=eds_detector.dispersions[0],
                                            shaping_time=eds_detector.shaping_times[0],
                                            dwell_time=4e-6,
                                            size=ImageSize.PRESET_256,
                                            scanning_detectors=[DetectorType.HAADF])

        # Acquire the spectrum image
        eds_spectrum_image = microscope.analysis.eds.acquire_spectrum_image(settings)

        # Save the spectrum image to disk (the object is just a handle for the data, we need to transfer it to the client)
        eds_spectrum_image.save(r"C:\temp\spectrum_image.emd")

The next example shows the use of EDS detector segments:


        # Get the EDS detector object
        eds_detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)

        # Configure the spectrum image acquisition - same as in previous example
        settings = EdsSpectrumImageSettings()
        # ...

        # Configure the detector segments, enable first two segments only
        eds_detector.set_enabled_segments([EdsDetectorSegmentType.SEGMENT1, EdsDetectorSegmentType.SEGMENT2])
        settings.eds_detector_segments = eds_detector.get_enabled_segments()

        # Acquire the spectrum image
        eds_spectrum_image = microscope.analysis.eds.acquire_spectrum_image(settings)

The next example shows the use of STEM detector segments (two detectors combined) and a rectangular region:


        # Get the EDS detector object
        eds_detector = microscope.detectors.get_eds_detector(EdsDetectorType.ULTRA_X)

        # Get the BF-S detector and select inner ring segments
        bf_s = microscope.detectors.get_scanning_detector(DetectorType.BF_S)
        bf_s.set_enabled_segments(DetectorSegmentType.INNER_RING)

        # Get the DF-S detector and select outer ring segments
        df_s = microscope.detectors.get_scanning_detector(DetectorType.DF_S)
        df_s.set_enabled_segments(DetectorSegmentType.OUTER_RING)

        # Combine enabled segments
        segments = df_s.get_enabled_segments() + bf_s.get_enabled_segments()

        # Configure the rectangle region (75x100 pixels)
        region = Region(coordinate_system=RegionCoordinateSystem.PIXELS, rectangle=[0, 0, 75, 100])

        # Configure the EDS spectrum image acquisition
        settings = EdsSpectrumImageSettings(eds_detector=eds_detector.name,
                                            dispersion=eds_detector.dispersions[0],
                                            shaping_time=eds_detector.shaping_times[0],
                                            dwell_time=10e-6,
                                            region=region,
                                            scanning_detector_segments=segments)

        # Acquire the spectrum image - the resulting file contains 8 STEM images (1 per segment of the two detectors)
        # Each STEM image has a size of 75x100 pixels
        eds_spectrum_image = microscope.analysis.eds.acquire_spectrum_image(settings)
