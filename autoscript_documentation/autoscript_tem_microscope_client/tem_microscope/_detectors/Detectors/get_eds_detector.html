detectors.get_eds_detector

Retrieves an EDS detector by name.

Syntax

get_eds_detector(detector_name)

Parameters

detector_name str The name of the detector. Enumeration EdsDetectorType can be used.

Return value

EdsDetector The EDS detector.

Examples

The following example shows how to create an get an EDS detector object and some of it's properties:


        # Get SuperX detector object
        detector = microscope.detectors.get_eds_detector(EdsDetectorType.SUPER_X)
        
        # Print detector name
        print(detector.name)
        
        # Print detector display name
        print(detector.display_name)
        
        # Inspect detector state and properties
        print(detector.insertion_state)
        print(detector.dispersions)
        print(detector.shaping_times)
