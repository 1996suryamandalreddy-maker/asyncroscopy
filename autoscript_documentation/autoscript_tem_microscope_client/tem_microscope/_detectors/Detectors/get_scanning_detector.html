detectors.get_scanning_detector

Retrieves a scanning detector by name.

Syntax

get_scanning_detector(detector_name)

Parameters

detector_name str The name of the detector. Enumeration DetectorType can be used.

Return value

ScanningDetector The scanning detector.

Examples

The following example shows how to create a scanning detector object for all available scanning detectors:


        for detector_name in microscope.detectors.scanning_detectors:
            detector = microscope.detectors.get_scanning_detector(detector_name)
            
            # print the name of the detector
            print(detector.display_name)
