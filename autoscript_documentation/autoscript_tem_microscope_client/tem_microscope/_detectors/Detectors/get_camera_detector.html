detectors.get_camera_detector

Retrieves a camera detector by name.

Syntax

get_camera_detector(detector_name)

Parameters

detector_name str The name of the camera. Enumeration CameraType can be used.

Return value

CameraDetector The camera detector.

Examples

The following example shows how to create a camera detector object for all available camera detectors:


        for camera_name in microscope.detectors.camera_detectors:
            camera = microscope.detectors.get_camera_detector(camera_name)
            
            # print the name of the camera
            print(camera.display_name)

The following example shows how access the properties of a detector and it's methods:


        # Get a Ceta camera object
        camera = microscope.detectors.get_camera_detector(CameraType.BM_CETA)
        print(camera.name)
        
        # Insert the camera (if already inserted, do nothing)
        camera.insert()
        print(camera.insertion_state)
        
        # Retract the camera (if already retracted, do nothing)
        camera.retract()
        print(camera.insertion_state)
