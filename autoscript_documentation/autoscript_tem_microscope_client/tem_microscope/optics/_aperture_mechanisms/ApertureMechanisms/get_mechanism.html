optics.aperture_mechanisms.get_mechanism

Retrieves an aperture mechanism by its id.

Syntax

get_mechanism(mechanism_type)

Parameters

mechanism_type str

Return value

ApertureMechanism Handle to a given Aperture Mechanism object.

Remarks

The mechanism_type parameter should be one of the values returned by get_available(). If get_mechanism is called with an unsupported mechanism_type, an exception will be raised. Only motorized aperture mechanisms are supported.

See also
