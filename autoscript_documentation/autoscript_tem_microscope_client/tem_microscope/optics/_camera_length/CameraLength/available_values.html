optics.camera_length.available_values

Returns the full collection of camera lengths for the current optical state.

Data type

[Read only] List[CameraLengthData]

Remarks

It is recommended to fetch a new list of available values after every optical mode change (objective_lens_mode, projector_mode, is_eftem_on).

Returns an empty collection if the projector_mode is IMAGING.

Examples


        # Set all available camera lengths in a loop
        for available_camera_length in microscope.camera_length.available_values:
            microscope.camera_length.value = available_camera_length
            
            # Do some work per camera length
