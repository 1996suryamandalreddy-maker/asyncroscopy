optics.camera_length.increase

Increases the camera length by steps (may be negative, will be clipped if necessary).

Syntax

increase(steps)

Parameters

steps int

Return value

CameraLengthData

Remarks

The method is only available in DIFFRACTION.

Increase function iterates over available_values collection. If the value is larger it gets clipped and the function sets the last possible value. Negative values are supported as well and clipping works the same way.

Examples

Index clipping is explained on following example:


        # Set the last available camera length value
        microscope.camera_length.value = microscope.camera_length.available_values[-1]
        
        # Increasing the camera length does nothing, because the current camera length
        # is the last possible value
        microscope.camera_length.increase(1)
        microscope.camera_length.increase(100)
        
        # Negative steps value can be used to set the second-to-last value
        microscope.camera_length.increase(-1)
        
        # Set the first available camera length value
        microscope.camera_length.value = microscope.camera_length.available_values[0]
        
        # Decreasing the camera length does nothing, because the current camera length
        # is the first possible value
        microscope.camera_length.increase(-1)
        microscope.camera_length.increase(-100)
