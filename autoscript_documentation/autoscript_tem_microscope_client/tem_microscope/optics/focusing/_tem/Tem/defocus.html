optics.focusing.tem.defocus

Defocus value - focus with an offset (in meters).

Data type

float

Remarks

Defocus value of the current optical_mode. Changing defocus will also change focus and vice versa. Defocus is measured with respect to the origin that can be set by using reset_defocus().

Examples


    # Set defocus value to 8nm
    microscope.optics.focusing.tem.defocus = 8e-9
