optics.focusing.stem.objective.defocus

Defocus value - offset from the objective focus value (in meters).

Data type

float

Remarks

The total defocus is equal to the sum of objective and intensity focus.

Examples


        # Prepare optics
        microscope.optics.optical_mode = OpticalMode.STEM
        microscope.optics.objective_lens_mode = ObjectiveLensMode.HM
        
        # Set defocus
        microscope.optics.focusing.stem.intensity.defocus = 5e-9
        microscope.optics.focusing.stem.objective.defocus = 6e-9
        
        # Print defocus values
        intensity_defocus = microscope.optics.focusing.stem.intensity.defocus
        objective_defocus = microscope.optics.focusing.stem.objective.defocus
        print(f"Intensity defocus: {intensity_defocus}")
        print(f"Objective defocus: {objective_defocus}")
        print(f"Total defocus: {intensity_defocus + objective_defocus}")
