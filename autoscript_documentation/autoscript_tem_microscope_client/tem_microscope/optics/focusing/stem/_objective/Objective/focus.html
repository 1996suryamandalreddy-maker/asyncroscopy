optics.focusing.stem.objective.focus

Focus value of the objective lens (in meters).

Data type

float

Remarks

The total focus is equal to the sum of objective and intensity focus.

Examples


        # Prepare optics
        microscope.optics.optical_mode = OpticalMode.STEM
        microscope.optics.objective_lens_mode = ObjectiveLensMode.HM
        
        # Set the focus
        microscope.optics.focusing.stem.intensity.focus = 5e-9
        microscope.optics.focusing.stem.objective.focus = 6e-9
        
        # print focus values
        intensity_focus = microscope.optics.focusing.stem.intensity.focus
        objective_focus = microscope.optics.focusing.stem.objective.focus
        print(f"Intensity focus: {intensity_focus}")
        print(f"Objective focus: {objective_focus}")
        print(f"Total focus: {intensity_focus + objective_focus}")
