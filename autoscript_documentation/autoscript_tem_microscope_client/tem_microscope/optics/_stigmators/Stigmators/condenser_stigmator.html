optics.stigmators.condenser_stigmator

The condenser stigmator value as a 2D vector in optical units (x, y). The maximum value range for both x and y is influenced by optical conditions and approaches (-1, +1).

Data type

Get: Point
Set: Point | list | tuple

Remarks

The property throws an error on systems which do not have the condenser stigmator.

When setting the value, the coordinates of the given point can be None. That means the value of this axis will remain the same.

Examples


        # Set condenser stigmator to a new value
        microscope.optics.stigmators.condenser_stigmator = Point(0.25, 0.5)

        # Set condenser stigmator to a new value with a list
        microscope.optics.stigmators.condenser_stigmator = [0.25, 0.5]

        # Set condenser stigmator to a new value with a tuple
        microscope.optics.stigmators.condenser_stigmator = (0.25, 0.5)

        # Set condenser stigmator to a new y value, leave x value as is
        microscope.optics.stigmators.condenser_stigmator = Point(y=0.7)
