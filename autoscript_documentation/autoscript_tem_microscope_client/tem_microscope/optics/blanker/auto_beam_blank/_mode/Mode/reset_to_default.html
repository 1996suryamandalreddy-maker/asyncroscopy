optics.blanker.auto_beam_blank.mode.reset_to_default

Resets the Auto Beam Blank mode to its default value.

Syntax

reset_to_default()

Remarks

Auto beam blank mode is reset back to default even without explicitly calling reset_to_default() in under 30s after script after script completion.

The default mode differs based on system hardware configuration.

Examples

Calling reset to default restores the default mode. 


        beam_blank_mode = microscope.optics.blanker.auto_beam_blank.mode

        beam_blank_mode.value = AutoBeamBlankMode.OFF
        print(beam_blank_mode.value)  # OFF
        
        beam_blank_mode.reset_to_default()
        print(beam_blank_mode.value)  # BLANK_WHEN_IDLETEM

Reset to default is called implicitly when exiting the with block of the auto_beam_blank_mode_context context manager.
