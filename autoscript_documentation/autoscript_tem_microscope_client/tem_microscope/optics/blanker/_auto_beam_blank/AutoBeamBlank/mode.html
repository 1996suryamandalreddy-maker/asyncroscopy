optics.blanker.auto_beam_blank.mode

Controls the auto beam blank mode.

Properties

available_values [Read only] Returns a list of supported Auto Beam Blank modes. Maps to values of the AutoBeamBlankMode enumeration.
value Gets or sets the current Auto Beam Blank mode. Maps to values of the AutoBeamBlankMode enumeration.

Methods

reset_to_default() Resets the Auto Beam Blank mode to its default value.

Remarks

Default mode

Default auto beam blank mode differs based on the system's hardware configuration.

Auto beam blank mode is reset back to default even without explicitly calling reset_to_default() in under 30s after script after script completion.

Mode is a system-wide setting of the auto beam blanker that controls blanking behavior in multiple scenarios including before and after an image acquisition. It can be set to one of three modes. See AutoBeamBlankMode enumeration for more details.

Autofunctions may blank or unblank the beam irrespective of the configured auto beam blank mode value.
