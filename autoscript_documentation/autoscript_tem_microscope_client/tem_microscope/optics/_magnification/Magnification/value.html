optics.magnification.value

The current magnification. Accepts values from available_values property.

Data type

MagnificationData

Remarks

Allows a quick magnification change without affecting optics modes (objective_lens_mode, projector_mode). Use function set_projector() to change magnification and optics modes at the same time.

The property is only available in IMAGING. See available_values for list of available values.

Examples


        # Set the first available magnification value
        microscope.magnification.value = microscope.magnification.available_values[0]
    
        # Set the last available magnification value
        microscope.magnification.value = microscope.magnification.available_values[-1]
