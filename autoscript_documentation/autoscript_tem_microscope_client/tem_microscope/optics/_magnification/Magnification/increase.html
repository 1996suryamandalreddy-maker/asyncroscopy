optics.magnification.increase

Increases the magnification by steps (may be negative, will be clipped if necessary).

Syntax

increase(steps)

Parameters

steps int

Return value

MagnificationData

Remarks

The method is only available in IMAGING.

Increase function iterates over available_values collection. If the value is larger it gets clipped and the function sets the last possible value. Negative values are supported as well and clipping works the same way.

Examples

Index clipping is explained by the following example:


        # Set the last available magnification value
        microscope.magnification.value = microscope.magnification.available_values[-1]
        
        # Increasing the magnification does nothing, because the current magnification
        # is the last possible value
        microscope.magnification.increase(1)
        microscope.magnification.increase(100)
        
        # Negative steps value can be used to set the second-to-last value
        microscope.magnification.increase(-1)
        
        # Set the first available magnification value
        microscope.magnification.value = microscope.magnification.available_values[0]
        
        # Decreasing the magnification does nothing, because the current magnification
        # is the first possible value
        microscope.magnification.increase(-1)
        microscope.magnification.increase(-100)
