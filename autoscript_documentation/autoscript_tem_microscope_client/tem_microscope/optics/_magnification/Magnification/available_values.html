optics.magnification.available_values

Returns the full collection of magnifications for the current optical state.

Data type

[Read only] List[MagnificationData]

Remarks

It is recommended to fetch a new list of available values after every optical mode change (objective_lens_mode, projector_mode, is_eftem_on).

Returns an empty collection if the projector_mode is DIFFRACTION.

Examples


        # Set all available magnifications in a loop
        for available_magnification in microscope.magnification.available_values:
            microscope.magnification.value = available_magnification
            
            # Do some work per magnification
