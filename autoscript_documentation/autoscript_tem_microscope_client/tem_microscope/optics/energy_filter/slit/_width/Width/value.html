optics.energy_filter.slit.width.value

Gets/sets the value in eV.

Data type

float

Remarks

Only values within limits are supported.

Examples


        # Get the width limits
        limits = microscope.optics.energy_filter.slit.width.limits

        # Set width to the minimal and maximal value
        microscope.optics.energy_filter.slit.width.value = limits.min
        microscope.optics.energy_filter.slit.width.value = limits.max
