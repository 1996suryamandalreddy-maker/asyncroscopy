optics.deflectors.image_tilt

The tilt of the image as a 2D vector in radians (x, y), equivalent to diffraction shift.

Data type

Get: Point
Set: Point | list | tuple

Remarks

The image tilt value is a two component vector of radians. The value represents diffraction pattern shift with respect to the origin and it is defined by alignment.

When setting the value, the coordinates of the given point can be None. That means the value of the unspecified axis will remain the same.

Examples


        # Set image tilt to a new value
        microscope.optics.deflectors.image_tilt = Point(4e-3, 5e-4)

        # Set image tilt to a new value with a list
        microscope.optics.deflectors.image_tilt = [4e-3, 5e-4]

        # Set image tilt to a new value with a tuple
        microscope.optics.deflectors.image_tilt = (4e-3, 5e-4)

        # Set image tilt to a new y value, leave x value as is
        microscope.optics.deflectors.image_tilt = Point(y=4.35e-4)
