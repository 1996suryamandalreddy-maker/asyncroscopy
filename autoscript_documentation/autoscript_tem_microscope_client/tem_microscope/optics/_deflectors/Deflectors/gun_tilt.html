optics.deflectors.gun_tilt

The tilt beam in the gun as a unitless 2D vector (x, y). Ranges from -1 to +1 for both x and y.

Data type

Get: Point
Set: Point | list | tuple

Remarks

The beam blanker changes the gun tilt. Therefore changing the gun tilt is blocked as long as the beam is blanked. Function unblank() can be used to unblank the beam.

When setting the value, the coordinates of the given point can be None. That means the value of this axis will remain the same.

Examples


        # Set gun tilt to a new value
        microscope.optics.deflectors.gun_tilt = Point(0.4, 0.3)

        # Set gun tilt to a new value with a list
        microscope.optics.deflectors.gun_tilt = [0.4, 0.3]

        # Set gun tilt to a new value with a tuple
        microscope.optics.deflectors.gun_tilt = (0.4, 0.3)

        # Set gun tilt to a new y value, leave x value as is
        microscope.optics.deflectors.gun_tilt = Point(y=0.33)
