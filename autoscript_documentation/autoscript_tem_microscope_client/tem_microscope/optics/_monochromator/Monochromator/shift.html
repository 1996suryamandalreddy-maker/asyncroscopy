optics.monochromator.shift

The shift of the monochromator as a 2D vector in scaled units (x, y).

Data type

Get: Point
Set: Point | list | tuple

Remarks

The monochromator shift is a low level value used to control X and Y coils. The shift is relative to the value stored in the alignments.

When setting the value, the coordinates of the given point can be None. That means the value of this axis will remain the same.

Examples


        # Set monochromator shift to a new value
        microscope.optics.monochromator.shift = Point(0.5, -0.1)

        # Set monochromator shift to a new value with a list
        microscope.optics.monochromator.shift = [0.5, -0.1]

        # Set monocchromator shift to a new value with a tuple
        microscope.optics.monochromator.shift = (0.5, -0.1)

        # Set monochromator shift to a new y value, leave x value as is
        microscope.optics.monochromator.shift = Point(y=-0.1)
