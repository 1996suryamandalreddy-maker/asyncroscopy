optics.blanker.unblank

Unblanks the beam. When unblanked, the sample is exposed to the beam.

Syntax

unblank()

Remarks

Manually unblanking the beam using the unblank function overrides the auto beam blank mode setting and will interfere with the automated blanking process. To restore automatic blanking the auto beam blank mode should be set again explicitly. 

optics.blanker.auto_beam_blank.mode.value = AutoBeamBlankMode.BLANK_WHEN_IDLE
