optics.blanker.blank

Blanks the beam. When blanked, the sample is not exposed to the beam.

Syntax

blank()

Remarks

Manually blanking the beam using the blank function overrides the auto beam blank mode setting. To restore automatic beam blanking the beam blank mode should be explicitly set again.

optics.blanker.auto_beam_blank.mode.value = AutoBeamBlankMode.BLANK_WHEN_IDLE
