optics.aperture_mechanisms.bi_prisms.get_available_bi_prisms

Retrieves a list of names of currently available bi-prisms. Uses values from ApertureMechanismType.

Syntax

get_available_bi_prisms()

Return value

List[str] List of available bi-prism names.

Examples


        for bi_prism in get_available_bi_prisms():
            # Enable the bi-prism
            bi_prism.enable()
            print(f"{bi_prism.name} is enabled: {bi_prism.is_enabled()}")

            # Disable the bi-prism
            bi_prism.disable()
            print(f"{bi_prism.name} is enabled: {bi_prism.is_enabled()}")
