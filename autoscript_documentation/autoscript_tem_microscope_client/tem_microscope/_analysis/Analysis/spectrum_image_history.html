analysis.spectrum_image_history

Returns a list of previously acquired spectrum images.

Data type

[Read only] List[AdornedSpectrumImage]

Remarks

The files are ordered by acquisition time, with the most recent acquisition first. Faulted acquisitions are also included in the list and can be downloaded. This means that incomplete acquisition results can also be downloaded and analyzed.

The files are only valid for a limited time. During the AutoScript startup sequence, the acquired files are checked, and all files older than 24 hours are deleted.

Examples

The example shows how the last acquired spectrum image can be downloaded:


        # Get the last acquired spectrum image from the history
        last_spectrum_image = analysis.spectrum_image_history[0]

        # Download and save the spectrum image to a file
        last_spectrum_image.save(r"C:\users\factory\desktop\last_spectrum_image.emd")
