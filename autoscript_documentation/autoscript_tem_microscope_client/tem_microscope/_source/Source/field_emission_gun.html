source.field_emission_gun

Provides access to filed emission gun.

Properties

is_flashing_advised [Read only]

Methods

flash() Performs low temperature flashing unless high temperature is advised and sets flashing post action.

Examples

Field emission gun flashing using default flashing post action.


        if microscope.source.field_emission_gun.is_flashing_advised:
            microscope.source.field_emission_gun.flash()

Field emission gun flashing and restoring optimal beam current as post action.


        if microscope.source.field_emission_gun.is_flashing_advised:
            microscope.source.field_emission_gun.flash(FlashingPostAction.RESTORE_OPTIMAL_BEAM_CURRENT)
