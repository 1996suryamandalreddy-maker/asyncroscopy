optics.paused_scan_beam_position

The position of the beam in the full scan area when the scan is paused. Range: -1 to +1 for x and y.

Data type

Get: Point
Set: Point | list | tuple

Remarks

This property is only available on systems with STEM.

When setting the value, the coordinates of the given point can be None. That means the value of this axis will remain the same.

Examples


        microscope.optics.paused_scan_beam_position = Point(0.2, 0.2)
        microscope.optics.paused_scan_beam_position = [0.5, 0.5]        # List is used to modify the position
        microscope.optics.paused_scan_beam_position = (1, 0)            # Tuple is used to modify the position
        microscope.optics.paused_scan_beam_position = Point(1.1, 0.5)   # Error: value out of range!
        microscope.optics.paused_scan_beam_position = Point(0.5)        # Set X to 0.5, keep Y as is
        microscope.optics.paused_scan_beam_position = Point(y=0.7)      # Set Y to 0.7, keep X as is
