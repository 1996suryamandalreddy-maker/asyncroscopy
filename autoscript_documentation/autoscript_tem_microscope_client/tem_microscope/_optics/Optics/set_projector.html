optics.set_projector

Sets the projector to magnification or camera length using information encoded in a text label.

Syntax

set_projector(label)

Parameters

label str

Remarks

This function allows you to set several optical parameters at once. It can change the magnification, camera length, objective lens mode and/or projector mode. Properties available_values on classes camera_length and magnification can be used to get the list of valid values.

Label must contain three parts separated by spaces:

  • Projector mode identification:
    • Magnification modes: Mh, Mi, SA, LM, Ltz SA, Ltz Mh.
    • Camera length modes: D, LAD, Ltz D.
  • Camera length value / magnification value: nominal values are rounded. See MagnificationData.label or CameraLengthData.label.
  • Unit:
    • Camera length units: m (meter) for values > 1. For other values mm.
    • Magnification units: Mx for values >= 1,000,000. kx for values >= 100,000. For other values x.

Examples

Example of switching between different objective lens modes and magnifications:


        # Set objective lens mode to HM and magnification to 4400 x
        microscope.optics.set_projector("SA 4400 x")
        
        # Set objective lens mode to LM and magnification to 320 x
        microscope.optics.set_projector("LM 320 x")
        
        # Set objective lens mode to Lorentz and magnification to 28500 x
        microscope.optics.set_projector("Ltz Mh 28500 x")

Example of switching between different projector modes:


        # Set projector mode to imaging
        microscope.optics.set_projector("Mh 1.05 Mx")
        
        # Set projector mode to diffraction
        microscope.optics.set_projector("D 470 mm")

Example of using magnification and camera length available values as parameters:


        # Set projector to the first available magnification
        first_magnification = self.optics.magnification.available_values[0].label
        microscope.optics.set_projector(first_magnification)
        
        # Set projector to the first available camera length
        first_camera_length = self.optics.camera_length.available_values[0].label
        microscope.optics.set_projector(first_camera_length)
