optics.beam_diameter

The beam diameter in meters. Only available on systems with three condenser lenses.

Data type

float

Remarks

Accessing beam diameter is only available in PARALLEL illumination_mode.

Examples


        # Set illumination mode to parallel
        microscope.optics.illumination_mode = IlluminationMode.PARALLEL

        # Set convergence angle to 3e-8 meters
        microscope.optics.beam_diameter = 3e-8
