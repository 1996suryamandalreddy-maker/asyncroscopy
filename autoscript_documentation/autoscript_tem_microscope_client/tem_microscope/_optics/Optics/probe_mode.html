optics.probe_mode

The probe mode of the condenser system. Maps to ProbeMode enumeration.

Data type

str

Remarks

This mode has nearly no effect in low magnifications.

Examples


        # Set probe mode to nano probe
        microscope.optics.probe_mode = ProbeMode.NANO_PROBE
        
        # Set probe mode back to micro probe
        microscope.optics.probe_mode = ProbeMode.MICRO_PROBE
