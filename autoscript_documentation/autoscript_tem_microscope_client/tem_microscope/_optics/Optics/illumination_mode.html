optics.illumination_mode

The illumination mode of the condenser system. Maps to IlluminationMode enumeration. Only available on systems with three condenser lenses.

Data type

str

Remarks

Some Titan systems have only two condenser lenses and this makes the illumination system similar to Talos. On systems with three condenser lenses, lenses C2 or C3 can be switched off.

Examples


        # Switch illumination condenser mode into two lens and turn C3 lens off
        microscope.optics.illumination_mode = IlluminationMode.C3_OFF
        
        # Switch illumination condenser mode into probe and angle range to large
        microscope.optics.illumination_mode = IlluminationMode.PROBE_LARGE_ANGLE
        
        # Switch illumination condenser mode back to parallel
        microscope.optics.illumination_mode = IlluminationMode.PARALLEL
