optics.aperture_mechanisms

The collection of aperture mechanisms on the microscope.

Classes

bi_prisms The bi-prism control.

Properties

C1 [Read only] The Condenser 1 aperture mechanism.
C2 [Read only] The Condenser 2 aperture mechanism.
C3 [Read only] The Condenser 3 aperture mechanism (if present).
objective [Read only] The Objective aperture mechanism.
SA [Read only] The Selected Area aperture mechanism.

Methods

get_available() Retrieves a list of IDs of currently available aperture mechanisms. Uses values from ApertureMechanismType.
get_mechanism() Retrieves an aperture mechanism by its id.

Remarks

Only motorized aperture mechanisms are accessible. GetMechanism can be used to retrieve all supported aperture mechanisms on a given system, however the most widely used mechanisms are available in a form of property (the ones present on most systems). The GetMechanism function is intended for more rare mechanisms to keep the API clean and it also provides a way to enumerate and perform an action on all available mechanisms.
