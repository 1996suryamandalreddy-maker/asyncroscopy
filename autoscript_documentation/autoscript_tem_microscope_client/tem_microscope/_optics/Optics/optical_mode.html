optics.optical_mode

The optical mode of the microscope. Maps to OpticalMode enumeration.

Data type

str

Remarks

Optical mode change triggers setting a few other optical parameters to predefined default states:

  • Setting optical mode to TEM will also change projector_mode to IMAGING.
  • Setting optical mode to STEM will also change projector_mode to DIFFRACTION and probe_mode to NANO_PROBE.

Warning: predefined default states may have side effects on the optical subsystem. Best practice is to set up the optical mode in advance and be aware that changing the optical mode may cause other optical parameters (e.g. magnification) to change as well.

Examples

The following code shows how to change optical mode to STEM and then back to TEM:


        microscope.optics.optical_mode = OpticalMode.STEM
        microscope.optics.optical_mode = OpticalMode.TEM
