optics.scan_rotation

The rotation of the scan in radians. Applicable only in STEM mode.

Data type

float

Remarks

The property is only available in STEM optical_mode.

Examples


        import math
        
        # Set scan rotation to 1.1 degrees
        microscope.optics.scan_rotation = math.radians(1.1)
