optics.is_eftem_on

The EFTEM (Energy Filtered TEM) mode.

Data type

bool

Remarks

Enabling the EFTEM mode raises an exception if the system does not have an energy filter.

Examples


        # Get EFTEM mode state
        microscope.optics.is_eftem_on
        
        # Enable EFTEM mode
        microscope.optics.is_eftem_on = True
