optics.convergence_angle

The beam convergence angle in radians. Only available on systems with three condenser lenses.

Data type

float

Remarks

Accessing convergence angle is only available in PROBE_LARGE_ANGLE illumination_mode.

Examples


        # Set illumination mode to probe large angle
        microscope.optics.illumination_mode = IlluminationMode.PROBE_LARGE_ANGLE

        # Set convergence angle to 5 micrometers
        microscope.optics.convergence_angle = 5e-6
