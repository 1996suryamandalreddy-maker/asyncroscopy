optics.focus

Focus value of the main imaging lens (in meters).
[The property is deprecated and will be removed in version 1.17]

Data type

float

Remarks

Focus value of the current optical_mode. Changing focus will also change defocus and vice versa.

Examples


        # Set focus value to 5nm
        microscope.optics.focus = 5e-9

See also

optics.defocus
