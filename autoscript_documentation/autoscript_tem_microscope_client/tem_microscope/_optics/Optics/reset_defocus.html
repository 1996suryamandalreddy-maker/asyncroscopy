optics.reset_defocus

[The method is deprecated and will be removed in version 1.17]

Syntax

reset_defocus()

Remarks

Resets the current defocus to zero. This does not change the focus value (the focusing lens current). Use it when the image is properly focused to adjust the defocus scale.

Examples


        # Set defocus value to 8nm
        microscope.optics.defocus = 8e-9
        
        # Reset defocus to zero
        microscope.reset_defocus()
        
        # Defocus value is now zero
        print(microscope.optics.defocus)
