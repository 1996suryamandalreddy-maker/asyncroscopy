optics.deflectors

Provides access to deflector lens.

Properties

beam_shift The shift of the beam as a 2D vector in meters (x, y).
beam_tilt The tilt of the beam as a unitless 2D vector (x, y). Ranges from -1 to +1 for both x and y.
gun_shift The shift of the beam as a unitless 2D vector (x, y). Ranges from -1 to +1 for both x and y.
gun_tilt The tilt beam in the gun as a unitless 2D vector (x, y). Ranges from -1 to +1 for both x and y.
image_shift The shift of the image as a 2D vector in meters (x, y).
image_tilt The tilt of the image as a 2D vector in radians (x, y), equivalent to diffraction shift.

Methods

get_available_deflectors() Lists all available deflector types.
get_deflector_value() Retrieves the excitation of a DeflectorType if it is supported. The value's unit is dependent on the deflector type.
set_deflector_value() Sets the excitation of a DeflectorType to a given value.
