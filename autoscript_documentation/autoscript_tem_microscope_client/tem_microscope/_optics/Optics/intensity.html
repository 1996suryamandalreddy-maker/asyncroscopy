optics.intensity

Intensity value (not available in STEM mode).

Data type

float

Remarks

Intensity value is only available in TEM optical_mode. The value roughly ranges from 0 to 1, on some systems the minimum value may be negative. Value is clipped when out of range.

Examples


        # setting intensity to 0.1
        microscope.optics.intensity = 0.1
        print(microscope.optics.intensity)
        
        # setting intensity to 0.3
        microscope.optics.intensity = 0.3
        print(microscope.optics.intensity)
