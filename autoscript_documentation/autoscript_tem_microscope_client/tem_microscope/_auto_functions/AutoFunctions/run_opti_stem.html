auto_functions.run_opti_stem

Runs OptiStem corrections with the specified parameters. Only available in STEM mode.

Syntax

run_opti_stem()
run_opti_stem(settings)

Parameters

settings RunOptiStemSettings Settings for the OptiStem routine.

Return value

RunOptiStemResults A structure capturing results of the autofunction run.

Remarks

OptiStem (C1A1 and C1 corrections) is only available on the Titan TEM server v3.15 or Talos TEM server v2.15 and newer. OptiStem+ (B2A2 corrections) is only available on the Titan TEM server v3.19 or Talos TEM server v2.19 and newer.

The autofunction can currently be run in three different modes depending on the method member of the settings parameter:

  • C1A1 - correction of defocus + two-fold condenser astigmatism.
  • C1 - correction of defocus.
  • B2A2 - correcting coma B2 and three fold astigmatism A2.

All correction algorithms typically take less than a minute to complete.

All input parameters are optional, and if any are left out, the default values are used instead. Default values are dependent on the system. The cutoff_in_pixels and cutoff_spatial_frequency can't be set in combination with B2A2 method.

Preconditions:

  • Optical_mode must be STEM.
  • Column valves must be open.
  • FEG must be in operation.
  • HT must be on.
  • Condenser 1 and Condenser 2 must be enabled. For B2A2 method, Condenser 3 motorized, Objective and SA must be also enabled.
  • Diffraction must be on.
  • Nano probe must be selected.
  • Sample must be at eucentric height.
  • Microscope must be close to the ideal focus and condenser stigmatism.
  • (Titan only)The illumination mode must be set to probe.

Examples

The following example shows how to run the OptiSTEM routine with the default parameters:


        microscope.auto_functions.run_opti_stem()

The following example shows how to run the OptiSTEM routine with optional parameters:


        settings = RunOptiStemSettings(method=OptiStemMethod.C1, dwell_time=2e-06, cutoff_in_pixels=5)
        microscope.auto_functions.run_opti_stem(settings)
