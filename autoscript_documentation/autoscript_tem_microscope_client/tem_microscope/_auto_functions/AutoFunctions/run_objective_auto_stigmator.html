auto_functions.run_objective_auto_stigmator

Performs automatic defocus and objective astigmatism correction. Available only in TEM mode.

Syntax

run_objective_auto_stigmator(settings)

Parameters

settings RunObjectiveAutoStigmatorSettings Settings for the objective auto stigmator routine.

Return value

RunObjectiveAutoStigmatorResults A structure capturing results of the autofunction run.

Remarks

Most of the parameters in RunObjectiveAutoStigmatorSettings have default values and can be omitted. The first three parameters are linked to image acquisition and have the same meaning as the arguments to the acquire_camera_image() function. The default values are documented on the RunObjectiveAutoStigmatorSettings page.

For information on supported camera types see members of the CameraType enumeration. Running this autofunction with the FLUCAM camera type is not supported.

The screen doesn't have to be retracted before calling this auto function; the acquisition part of the task manages it automatically. The chosen detector is inserted automatically as well.

Preconditions and recommendations:

  • optical_mode must be TEM.
  • projector_mode must be IMAGING.
  • The microscope is properly aligned. Check the Direct Alignments (especially rotation center) and roughly correct objective astigmatism.
  • EPU / Auto Functions / Autofocus calibration has been performed.
  • The magnification calibrations have been performed.
  • A specimen is loaded which enables the display of Thon rings in the power spectrum of the TEM image. This is typically a thin amorphous carbon film, such as Quantifoil, Lacey Carbon, or a Cross Grating sample.
  • Spot size is 3 or higher.
  • Beam edge is not visible on Flucam.
  • Defocus is between -1 µm and -3 µm.
  • Magnification is approximately 150kX.
  • The beam diameter can be narrowed a bit to increase signal strength.
  • Accurately set the specimen to eucentric height, using the Stage Alpha tilt wobbler.

If the astigmatism cannot be corrected within a specified number of iterations, the optical settings are returned to the state before the start of the correction procedure.

Examples

The following example shows how to run objective auto stigmator routine with default parameters:


        # Save settings into a variable with all optional settings left as defaults
        settings = RunObjectiveAutoStigmatorSettings(CameraType.BM_CETA)
        # Run the objective auto stigmator routine with created settings
        result = microscope.auto_functions.run_objective_auto_stigmator(settings)

Optional parameters can be added to the structure. Parameter names need not to be specified if the order is kept.


        settings = RunObjectiveAutoStigmatorSettings(CameraType.BM_CETA, 1024, 2, 5, tile_size=1024)
        microscope.auto_functions.run_objective_auto_stigmator(settings)
