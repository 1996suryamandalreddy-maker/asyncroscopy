auto_functions.run_stem_auto_focus

Runs a routine to automatically focus in STEM optical mode.

Syntax

run_stem_auto_focus(settings)

Parameters

settings RunStemAutoFocusSettings Settings for the automatic focus routine.

Return value

RunStemAutoFocusResults A structure capturing results of the autofunction run.

Remarks

Most of the parameters in RunStemAutoFocusSettings have default values and can be omitted. The first three parameters are linked to image acquisition and have the same meaning as the arguments to the acquire_stem_image() function. The default values are documented on the RunStemAutoFocusSettings page.

The screen doesn't have to be retracted before calling this auto function; the acquisition part of the task manages it automatically. The chosen detector is inserted automatically as well.

Preconditions and recommendations:

  • optical_mode must be STEM. To run autofocus in TEM mode, use run_beam_tilt_auto_focus() function.
  • projector_mode must be DIFFRACTION.
  • A specimen is loaded.
  • Magnification is around 56kX.

Examples

The following example shows how to run the autofocus routine with default parameters:


        # Switch to STEM optical mode
        microscope.optics.optical_mode = OpticalMode.STEM
        # Save settings into a variable with all optional settings left as defaults
        settings = RunStemAutoFocusSettings(DetectorType.HAADF)
        # Run the auto focus routine with the created settings
        result = microscope.auto_functions.run_stem_auto_focus(settings)

Optional parameters can be added to the structure. Parameter names need not to be specified if the order is kept.


        settings = RunStemAutoFocusSettings(DetectorType.HAADF, 1024, 1e-5, true, reset_beam_blanker_afterwards=True)
        microscope.auto_functions.run_stem_auto_focus(settings)
