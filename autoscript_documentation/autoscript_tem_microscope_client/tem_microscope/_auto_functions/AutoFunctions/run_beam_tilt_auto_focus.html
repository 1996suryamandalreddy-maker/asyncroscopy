auto_functions.run_beam_tilt_auto_focus

Runs a routine to automatically focus using the beam tilt method. Available only in TEM mode.

Syntax

run_beam_tilt_auto_focus(settings)

Parameters

settings RunBeamTiltAutoFocusSettings Settings for the automatic focus routine.

Return value

RunBeamTiltAutoFocusResults A structure capturing results of the autofunction run.

Remarks

Most of the parameters in RunBeamTiltAutoFocusSettings have default values and can be omitted. The first three parameters are linked to image acquisition and have the same meaning as the arguments to the acquire_camera_image() function. The default values are documented on the RunBeamTiltAutoFocusSettings page.

For information on supported camera types, see the members of the CameraType enumeration. Running this autofunction with the FLUCAM camera type is not supported.

The screen doesn't have to be retracted before calling this auto function; the acquisition part of the task manages it automatically. The chosen detector is inserted automatically as well.

Preconditions and recommendations:

  • optical_mode must be TEM. To run auto focus in STEM mode, use run_stem_auto_focus().
  • projector_mode must be IMAGING.
  • Calibration of the auto function must be performed on the microscope beforehand ("Focus TEM Stigmator" calibration).
  • A sample must be present and positioned at eucentric height.
  • If the auto function results with an error 'Auto Focus did not converge.', try setting the maximum_iterations parameter to a higher value.

Examples

The following example shows how to run the autofocus routine with default parameters:


        # Save settings into a variable with all optional settings left as defaults
        settings = RunBeamTiltAutoFocusSettings(CameraType.BM_CETA)
        # Run the auto focus routine with the created settings
        result = microscope.auto_functions.run_beam_tilt_auto_focus(settings)

Optional parameters can be added to the structure. Parameter names need not to be specified if the order is kept.


        settings = RunBeamTiltAutoFocusSettings(CameraType.BM_CETA, 2, FocusMethod.STAGE_Z, maximum_iterations=10)
        microscope.auto_functions.run_beam_tilt_auto_focus(settings)
