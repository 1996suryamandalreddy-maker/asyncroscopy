vacuum.column_valves.open

Attempts to open the column valves.

Syntax

open()

Remarks

Changing the state of the column valves is not possible if the column valves are in NOT_ALLOWED state.

Examples


        # Open column valves
        microscope.vacuum.column_valves.open()

        # Print the column valves state
        print(microscope.vacuum.column_valves.state)
