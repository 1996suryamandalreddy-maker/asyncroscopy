service.offline.reset_image

Resets the image source in the offline simulator back to the default image.

Syntax

reset_image()

Examples


        # Load an image from disk and set it as a new simulation source
        microscope.service.offline.set_image(AdornedImage.load(r"C:\image.tiff"))

        # Subsequent acquisitions will return the loaded image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 1024, 1.0)

        # Reset the image back to defaults
        microscope.service.offline.reset_image()

        # Subsequent acquisitions will return the default simulated image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 1024, 1.0)
