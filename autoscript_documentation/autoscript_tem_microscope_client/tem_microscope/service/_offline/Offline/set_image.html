service.offline.set_image

Uploads an image into the offline simulator. AutoScript will return this image in all subsequent image acquisitions.

Syntax

set_image(image)

Parameters

image AdornedImage The image to set as the new source for simulated image acquisitions.

Remarks

The image will be resized to match the size specified in acquisition function parameters, but image metadata will not reflect this resizing.

Examples


        # Load an image from disk, check its size and upload it to the simulator 
        loaded_image = AdornedImage.load(r"C:\image.tiff")
        print(loaded_image.width)  # 1024
        print(loaded_image.height)  # 1024
        microscope.service.offline.set_image(loaded_image)

        # Required size is the same, the simulator will provide the original image
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 1024, 1.0)

        # Required size differs, the simulator will resize the image to 2048x2048
        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 2048, 1.0)
