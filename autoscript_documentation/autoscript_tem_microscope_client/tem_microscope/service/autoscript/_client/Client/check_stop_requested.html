service.autoscript.client.check_stop_requested

Checks whether a graceful stop of the currently executing script has been requested. If so, raises KeyboardInterrupt exception.

Syntax

check_stop_requested()

Remarks

This function allows the user to specify the exact time and conditions that must be met for gracefully stopping the script execution.

The stop request can be created in two places - AutoScript Runner or PyCharm editor

If the stop request is made before the script executes check_stop_requested(), the execution of the script is stopped.

If check_stop_requested() is called but no stop request is created the script will not stop on this line.
