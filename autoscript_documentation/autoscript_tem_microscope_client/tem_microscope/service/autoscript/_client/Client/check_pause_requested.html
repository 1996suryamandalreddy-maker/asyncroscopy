service.autoscript.client.check_pause_requested

Checks whether a pause of the currently executing script has been requested. If so, waits until the resume is requested.

Syntax

check_pause_requested()

Remarks

The pause request can be created in two places - AutoScript Runner or PyCharm editor

Once the script execution is paused it is possible to resume it by creating a resume request.

If check_pause_requested() is called but no pause request is created, the script will execute without pausing.
