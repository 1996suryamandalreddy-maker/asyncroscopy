specimen.piezo_stage

Provides piezo stage manipulation functionality.

Properties

is_enabled [Read only] Returns true if the piezo stage is enabled, false otherwise.
is_moving [Read only] Returns true if the piezo stage is currently moving (jogging velocity is non-zero), false otherwise.
jogging_velocity [Read only] Returns the current jogging velocity X, Y, Z in meters per second, A and B in radians per second.
position [Read only] Returns the piezo stage position X, Y, Z in meters, A and B in radians.

Methods

disable() Switches the piezo stage off.
enable() Switches the piezo stage on.
get_axis_limits() Returns limits of a piezo stage axis in meters or radians.
relative_move() Moves the piezo stage relatively to the current position.
reset_position() Resets the piezo stage to the specified position while compensating the stage position at the same time.
start_jogging() Starts jogging at a given velocity. This function is asynchronous and returns immediately.
stop() Stops all movement on all axes.
