specimen.stage

Provides access to the microscope stage (CompuStage or SmartStage).

Properties

is_enabled [Read only] Returns true if the stage is ready, false otherwise.
is_moving [Read only] Returns true if the stage is currently moving, jogging, or wobbling. False otherwise.
position [Read only] Returns the stage position (X, Y, Z) in meters, A and B in radians.

Methods

absolute_move() Moves the stage to a new position.
absolute_move_safe() Moves the stage to a new position. To avoid pole touches the tilt is first set to zero.
get_axis_limits() Returns limits of a stage axis in meters or radians.
get_holder_type() Returns the type of holder that is inserted.
move_to_pixel() Centers the specified image point in the field of view.
relative_move() Moves the stage relatively to the current stage position.
relative_move_safe() Moves the stage relatively to the current stage position. To avoid pole touches the tilt is first set to zero.
start_jogging() Starts jogging at a given velocity. This function is asynchronous and returns immediately.
stop_jogging() Stops jogging on all axes.
