acquisition.acquire_stem_images_advanced

Acquires images with advanced settings using specified detectors or segments.

Syntax

acquire_stem_images_advanced(settings)

Parameters

settings StemAcquisitionSettings Advanced STEM settings structure.

Return value

List[AdornedImage] List of STEM images. Order in the list corresponds to the order of segments in the settings.

Remarks

The images are returned as a list. The returned image sequence maintains the order given by detector types and segments in the settings. In case both segments and detector types are specified, all detector images are returned first. When using segment groups, (such as All) the images are ordered as follows: inner-1, inner-2, ..., outer-1, outer-2, ..., outer-4. For more information about about supported settings see StemAcquisitionSettings.

See this page for information about the acquisition overhead.

Examples

The following example shows an advanced acquisition from selected detector segments and detector types. Dwell time is set to 1 µs, the size of the image is 1024x1024 pixels, and the region of interest is set to a Rectangle in the upper left corner of the sensor with relative size 0.2 by 0.8. The resulting image will be cropped to the size of the given rectangle (205x819). 


        # Select segments from segmented detectors
        bf_s = microscope.detectors.get_scanning_detector(DetectorType.BF_S)
        bf_s.set_enabled_segments(DetectorSegmentType.INNER_RING)

        df_s = microscope.detectors.get_scanning_detector(DetectorType.DF_S)
        df_s.set_enabled_segments(DetectorSegmentType.OUTER_RING)

        # Combine enabled segments
        segments = df_s.get_enabled_segments() + bf_s.get_enabled_segments()

        # Select detector
        detector_types = [DetectorType.HAADF]

        # Create settings and acquire images
        settings = StemAcquisitionSettings(1e-6,
                                           detector_types=detector_types,
                                           detector_segments=segments,
                                           size=1024,
                                           region=Region(RegionCoordinateSystem.RELATIVE, Rectangle(0, 0, 0.2, 0.8)),
                                           auto_beam_blank=False)

        images = microscope.acquisition.acquire_stem_images_advanced(settings)
