acquisition.acquire_camera_image_advanced

Acquires an image from a camera with advanced settings.

Syntax

acquire_camera_image_advanced(settings)

Parameters

settings CameraAcquisitionSettings Advanced settings structure.

Return value

AdornedImage The camera image.

Remarks

For information about supported image sizes see get_supported_image_sizes() method.

The screen does not have to be retracted before calling this function; the acquisition manages it automatically. The selected detector is also inserted automatically.

Examples

The following code shows how a 1024 x 1024 image is acquired by the Ceta camera with an exposure time of 1s. Readout area is set to half and frame combining (Ceta specific setting) is set to 6. 


        settings = CameraAcquisitionSettings(CameraType.BM_CETA, ImageSize.PRESET_1024, 0.5,
            fixed_readout_area=FixedReadoutArea.HALF, frame_combining=6)

        image = microscope.acquisition.acquire_camera_image_advanced(settings)
