acquisition.acquire_camera_series

Acquires a series of images from a camera.

Syntax

acquire_camera_series(camera_detector, size, exposure_time, length)

Parameters

camera_detector str The name of the detector. Enumeration CameraType can be used.
size int Square image size in pixels. Enumeration ImageSize can be used.
exposure_time float The exposure time in seconds.
length int The number of images to acquire.

Return value

List[AdornedImage] The image series.

Remarks

The entire series of images must fit into memory. The maximum number of images returned in one call is 64. To take and process a longer stream of images, you must call the function in a loop and process the images in batches. See the example below.

For information about supported image sizes see get_supported_image_sizes() method.

The screen does not have to be retracted before calling this function; the acquisition manages it automatically. The selected detector is also inserted automatically.

Examples

Take 5 images with Flucam and save them to disk with increasing index in the filename.


        images = microscope.acquisition.acquire_camera_series(CameraType.FLUCAM, ImageSize.PRESET_1024, 0.5, 5)
        
        index = 1
        for image in images:
            image.save(f"c:\\image{index}.tiff")
            index += 1

Acquire 100 images with Ceta camera and save them with increasing index in the filename. Note that we need to implement a processing loop to overcome the limit of 64 images in a single acquisition call. 


        index = 1
        for loops in range(1, 10):
            images = microscope.acquisition.acquire_camera_series(CameraType.BM_CETA, 1024, 1.0, 10)
            for image in images:
                print("Saving image", index)
                image.save(f"c:\\image{index}.tiff")
                index += 1
