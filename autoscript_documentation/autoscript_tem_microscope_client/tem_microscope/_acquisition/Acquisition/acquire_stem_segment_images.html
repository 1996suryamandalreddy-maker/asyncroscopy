acquisition.acquire_stem_segment_images

Acquires images from scanning detector segments.

Syntax

acquire_stem_segment_images(detector_segments, size, dwell_time)

Parameters

detector_segments List[DetectorSegment] The list of detector segments.
size int Square image size in pixels. Enumeration ImageSize can be used.
dwell_time float The dwell time in seconds.

Return value

List[AdornedImage] The list of images. The order of the images in the list corresponds to the order of the segments in the input array.

Remarks

  • As a basic use case, the function takes a list of segments from one detector, simultaneously acquires images on those that are enabled, and returns the resulting images as a list of AdornedImage objects, maintaining their input order.
  • The input array may also include segments from multiple detectors. Images are acquired simultaneously on all such detectors and their segments. Non-segmented detectors, such as HAADF, can also participate in the acquisition. For such detectors, include a virtual segment named "Single" in the input array.
  • Typically, detector segments are configured by accessing the detector object before calling this function. Gain, offset and enable/disable state can be set for each segment.
  • The name of the segment from which the image originates is stored in the custom section of the image metadata under the key "StemSegment". An example of how to retrieve this information can be found in the Examples section.
  • The screen doesn't have to be retracted before calling this function; the acquisition manages it automatically. The chosen detector is inserted automatically as well.
  • The Panther STEM detectors (DF-S and BF-S) consist of eight segments: 4 quadrants on a inner and outer ring, resulting in 16 segments in total. However, only 12 segments can be active at once as the outer 4 quadrants of the BF-S detector are shadowed by the DF-S detector when both are inserted. When combined with the HAADF detector the maximum of 13 signals can be recorded (see the examples).

See this page for information about the acquisition overhead.

Examples

The following example shows acquisition on two segments of the BF-S detector: 


        # Get object representing the BF-S detector
        detector = microscope.detectors.get_scanning_detector(DetectorType.BF_S)

        # Enable first two segments of the inner ring and disable all other segments
        detector.set_enabled_segments([DetectorSegmentType.INNER1, DetectorSegmentType.INNER2])

        # Get list of enabled segments for the acquisition method
        segments = detector.get_enabled_segments()

        # Acquire the images
        images = microscope.acquisition.acquire_stem_segment_images(segments, size=1024, dwell_time=200e-9)

        # images[0] holds the image from "Inner1" segment
        # images[1] holds the image from "Inner2" segment

DF4 detectors also contain segments, but these are not divided into rings and different identifiers must be used: 


        # Get object representing the DF4 detector
        detector = microscope.detectors.get_scanning_detector(DetectorType.DF4)

        # Enable first two segments of the detector
        detector.set_enabled_segments([DetectorSegmentType.SEGMENT1, DetectorSegmentType.SEGMENT2])

        # Acquire the images
        images = microscope.acquisition.acquire_stem_segment_images(detector.get_enabled_segments(), size=1024,
                                                                    dwell_time=200e-9)

        # images[0] holds the image from "Segment1" segment
        # images[1] holds the image from "Segment2" segment

The function can also acquire images from multiple detectors at the same time. Detectors without segments are assumed to have a virtual segment "Single". 


        # Get detector objects
        bf_s = microscope.detectors.get_scanning_detector(DetectorType.BF_S)
        haadf = microscope.detectors.get_scanning_detector(DetectorType.HAADF)

        # Enable all four segments of BF-S outer ring and disable all others
        bf_s.set_enabled_segments(DetectorSegmentType.OUTER_RING)

        # Enable SINGLE segment on HAADF
        haadf.set_enabled_segments(DetectorSegmentType.SINGLE)

        # Join the lists together
        segments = bf_s.get_enabled_segments() + haadf.get_enabled_segments()

        # Acquire one image per segment, a total of five images in this case
        images = microscope.acquisition.acquire_stem_segment_images(segments, size=1024, dwell_time=200e-9)

Acquire 13 images from BF-S, DF-S and HAADF detectors simultaneously. 


        # Get detector objects
        bf_s = microscope.detectors.get_scanning_detector(DetectorType.BF_S)
        df_s = microscope.detectors.get_scanning_detector(DetectorType.DF_S)
        haadf = microscope.detectors.get_scanning_detector(DetectorType.HAADF)

        # Enable the inner segments on the BF-S detector
        bf_s.set_enabled_segments(DetectorSegmentType.INNER_RING)

        # Enable all segments on the DF-S detector
        df_s.set_enabled_segments(DetectorSegmentType.ALL)
        
        # Enable a single segment on HAADF
        haadf.set_enabled_segments(DetectorSegmentType.SINGLE)

        # Join the lists together
        segments = bf_s.get_enabled_segments() + df_s.get_enabled_segments() + haadf.get_enabled_segments()

        # Acquire one image per segment, a total of 13 images in this case
        images = microscope.acquisition.acquire_stem_segment_images(segments, size=256, dwell_time=1e-6)

Retrieve segment information from metadata:


        # Get object representing the BF-S detector
        detector = microscope.detectors.get_scanning_detector(DetectorType.BF_S)

        # Enable first two segments of the inner ring and disable all other segments
        detector.set_enabled_segments([DetectorSegmentType.INNER1, DetectorSegmentType.INNER2])

        # Acquire the images
        images = microscope.acquisition.acquire_stem_segment_images(detector.get_enabled_segments(), size=1024,
                                                                    dwell_time=200e-9)

        # Retrieve segment information from the metadata
        for image in images:
            # Find first metadata custom property collection with an empty scope
            custom_properties = [property_list for property_list in image.metadata.custom_properties
                                 if property_list.scope == ""][0].properties

            # Find first custom property by the name "StemSegment"
            segment_custom_property = [custom_property for custom_property in custom_properties
                                       if custom_property.name == "StemSegment"][0]

            # Print segment info
            print(segment_custom_property.value)
