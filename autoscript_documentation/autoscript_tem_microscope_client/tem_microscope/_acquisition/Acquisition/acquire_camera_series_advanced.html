acquisition.acquire_camera_series_advanced

Acquires a series of images from a camera with advanced settings.

Syntax

acquire_camera_series_advanced(settings, length)

Parameters

settings CameraAcquisitionSettings Advanced settings structure.
length int

Return value

List[AdornedImage] The image series.

Remarks

The entire series of images must fit into memory. The maximum number of images returned in one call is 64. To take and process a longer stream of images, you must call the function in a loop and process the images in batches. See the example in acquire_camera_series().

For information about supported image sizes see get_supported_image_sizes() method.

The screen does not have to be retracted before calling this function; the acquisition manages it automatically. The selected detector is also inserted automatically.

Examples

Take 5 images with Ceta with advanced acquisition settings and save them to disk with increasing index in the filename.


        settings = CameraAcquisitionSettings(CameraType.BM_CETA, ImageSize.PRESET_1024, 0.5,
            fixed_readout_area=FixedReadoutArea.HALF, frame_combining=6)
        images = microscope.acquisition.acquire_camera_series_advanced(settings, 5)
        
        index = 1
        for image in images:
            image.save(f"c:\\image{index}.tiff")
            index += 1
