acquisition.acquire_camera_image

Acquires an image from a camera.

Syntax

acquire_camera_image(camera_detector, size, exposure_time)

Parameters

camera_detector str The name of the camera detector. Enumeration CameraType can be used.
size int Square image size in pixels. Enumeration ImageSize can be used.
exposure_time float The exposure time in seconds.

Return value

AdornedImage The camera image.

Remarks

For information about supported image sizes see get_supported_image_sizes() method.

The screen does not have to be retracted before calling this function; the acquisition manages it automatically. The selected detector is also inserted automatically.

Examples

The following code shows how a 1024 x 1024 image is acquired by the Ceta camera with an exposure time of 1s. Note that you need to specify the correct camera name, either as a string or (better) a value from from the CameraType enumeration. You can find out names of installed cameras by calling the detectors.camera_detectors() function. 


        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, 1024, 1.0)

Enumeration ImageSize can be used to select acquisition size:


        image = microscope.acquisition.acquire_camera_image(CameraType.BM_CETA, ImageSize.PRESET_1024, 1.0)
