acquisition.acquire_stem_image

Acquires an image using a scanning detector.

Syntax

acquire_stem_image(scanning_detector, size, dwell_time)

Parameters

scanning_detector str The name of the detector. Enumeration DetectorType can be used.
size int Square image size in pixels. Enumeration ImageSize can be used.
dwell_time float The dwell time in seconds.

Return value

AdornedImage The STEM image.

Remarks

If a multisegment detector (DF-S or BF-S) is specified, the image is acquired from all of its segments, regardless of which segments may have been disabled by calling ScanningDetector.set_enabled_segments(). The function does not permanently reset the segment configuration, but simply ignores it.

If you want to acquire images from individual segments, use acquisition.acquire_stem_segment_images().

The screen doesn't have to be retracted before calling this function; the acquisition manages it automatically. The chosen detector is inserted automatically as well.

Examples

Acquire a STEM image of 512 x 512 pixels from the HAADF detector with 1 µs dwell time:


        image = microscope.acquisition.acquire_stem_image("HAADF", ImageSize.PRESET_512, 1e-6)

Acquire a STEM image of 1024 x 1024 pixels from the BF detector with 2 µs dwell time and save it to disk in TIFF format:


        image = microscope.acquisition.acquire_stem_image(DetectorType.BF, ImageSize.PRESET_1024, 2e-6)
        image.save("c:\\result.tiff")
