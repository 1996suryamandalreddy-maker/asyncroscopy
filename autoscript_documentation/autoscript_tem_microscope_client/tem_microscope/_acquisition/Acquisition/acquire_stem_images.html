acquisition.acquire_stem_images

Acquires images using multiple scanning detectors.

Syntax

acquire_stem_images(scanning_detectors, size, dwell_time)

Parameters

scanning_detectors List[str] The list of scanning detectors to use in acquisition. Enumeration DetectorType can be used.
size int Square image size in pixels. Enumeration ImageSize can be used.
dwell_time float The dwell time in seconds.

Return value

List[AdornedImage] The list of images from the detectors.

Remarks

The screen doesn't have to be retracted before calling this function; the acquisition manages it automatically. The chosen detector is inserted automatically as well.

See this page for information about the acquisition overhead.

Examples

Take image with each STEM detector installed on the microscope and save it as TIFF with detector name noted in the file name: 


        detectors = microscope.detectors.scanning_detectors
        print("Available detectors:", detectors)
        
        # Take one image with each detector; resolution will be 1k and dwell time 1us
        images = microscope.acquisition.acquire_stem_images(detectors, 1024, 1e-6)
        print(f"{len(images)} images taken.")
        
        # Save the images
        for i in range(0, len(images)):
            print("Saving image from", detectors[i])
            images[i].save(f"c:\\image_{detectors[i]}.tiff")
