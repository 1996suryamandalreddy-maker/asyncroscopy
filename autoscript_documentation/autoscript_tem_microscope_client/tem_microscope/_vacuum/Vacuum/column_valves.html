vacuum.column_valves

Exposes column valves functionality.

Properties

state [Read only] The state of column valves. Values from enumeration ColumnValvesState are used.

Methods

close() Closes the column valves.
open() Attempts to open the column valves.

Examples

The following example shows how to close, open and get the status of the column valves:


        microscope.vacuum.column_valves.close()
        print(microscope.vacuum.column_valves.state) # CLOSED
        microscope.vacuum.column_valves.open()
        print(microscope.vacuum.column_valves.state) # OPENED
