Enumerations

Enumerations are simple static objects consisting of constant integer or string values. They do not inherit from the Enum class from the Python standard library, nor do the behave like types. Enumerations are often used to denote all or the most frequently used values in various API calls. All possible uses of enumeration values are documented with the phrase "Enumeration X can be used" or similar.
