VacuumState

State of the microscope vacuum.

Values

BUFFER_CYCLE "BufferCycle" The buffer cycle is busy: emptying the buffer for the projection chamber. Some noise or vibrations may occur.
BUSY "Busy" The vacuum system is busy transitioning to another state.
CRYO_CYCLE_CONDITION_COLUMN "CryoCycleConditionColumn" The cryo cycle is active: warming up the column and pumping away the water vapor that is released.
READY "Ready" The vacuum system is ready and operational.
UNKNOWN "Unknown" The vacuum system is in an unknown state.
VACUUM_OFF "VacuumOff" The vacuum system is switched off.
