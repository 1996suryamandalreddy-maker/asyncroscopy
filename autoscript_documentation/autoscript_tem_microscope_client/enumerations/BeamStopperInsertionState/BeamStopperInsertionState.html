BeamStopperInsertionState

Possible insertion states of the beam stopper.

Values

BUSY "Busy" Beam stopper is busy.
HALF_INSERTED "HalfInserted" Beam stopper is half way inserted.
INSERTED "Inserted" Beam stopper is inserted.
RETRACTED "Retracted" Beam stopper is retracted.
UNKNOWN "Unknown" Beam stopper state is unknown.
