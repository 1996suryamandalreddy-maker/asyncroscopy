AutoBeamBlankMode

Defines auto beam blank modes.

Values

BLANK_WHEN_IDLE "BlankWhenIdle" Auto beam blank enabled.
BLANK_WHEN_IDLE_TEM "BlankWhenIdle_TEM" Blanks in TEM only.
OFF "Off" Auto beam blank disabled.

Remarks

  • OFF:
    Automatic beam blanking is completely disabled. No blanking or unblanking takes place without manual user interventions.

  • BLANK_WHEN_IDLETEM:
    When selected the blanking is automatically triggered under these circumstances:

    • When no cameras are in the beam's path anymore the beam is blanked.
    • When the fluorescent screen is inserted into the beam's path the beam unblanks.
    • When a camera is inserted into the beam's path the beam is blanked.
    • When the system exits TEM mode the beam is unblanked.

  • BLANK_WHEN_IDLE:
    When selected the system automatically unblanks before an acquisition starts and blanks when the last acquisition finishes. This mode is only available for systems with the NanoPulser hardware.
