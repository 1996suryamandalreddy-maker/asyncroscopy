CameraType

Specifies camera types.

Values

BM_CETA "BM-Ceta" Bottom mount Ceta camera.
BM_EMPAD "BM-Empad" Bottom mount EMPAD camera.
BM_FALCON "BM-Falcon" Bottom mount Falcon camera.
EF_CCD "EF-CCD" Electron filter CCD camera.
EF_FALCON "EF-Falcon" Electron filter Falcon camera.
FLUCAM "Flucam" Flucam / SmartCam camera.
SH_EMPAD "SH-Empad" Service hatch mount EMPAD camera.

Remarks

AutoScript is using values from this enumeration as camera identifiers. For displaying names to the end user, use property display_name. Cameras that are not listed in this enumeration are not supported.
