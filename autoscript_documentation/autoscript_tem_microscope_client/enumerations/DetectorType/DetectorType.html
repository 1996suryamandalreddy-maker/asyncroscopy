DetectorType

Specifies scanning detector types.

Values

BF "BF" Bright Field detector.
BF_S "BF-S" Segmented bright field detector.
DF2 "DF2" Dark Field detector.
DF4 "DF4" Outer dark field retractable detector.
DF_S "DF-S" Segmented dark field detector.
HAADF "HAADF" High Angle Annular Dark Field detector.

Remarks

AutoScript is using values from this enumeration as detector identifiers. For displaying names to the end user, use property display_name. Scanning detectors that are not listed in this enumeration are not supported.
