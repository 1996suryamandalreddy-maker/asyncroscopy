Microscope API

AutoScript TEM is a comprehensive software package from Thermo Fisher Scientific designed to facilitate Transmission Electron Microscope scripting and related data processing utilities in a Python 3 language environment.

The most important class in this package is TemMicroscopeClient, which after instantiation and connection to an AutoScript server provides a wide range of functions through its members (child objects, properties and functions) organized in a static tree.

The package contains also other classes. Data structures make parameters and return values of static tree methods more comprehensive, while enumerations can save some time and help avoid typos when used instead of string or integer values. Dynamic objects are objects whose lifetime is expected to be shorter than the lifetime of TemMicroscopeClient and which often have to be created explicitly during script execution. Detectors may serve as a good example of dynamic objects.
