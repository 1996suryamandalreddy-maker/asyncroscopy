About AutoScript TEM

AutoScript TEM is a comprehensive software package from Thermo Fisher Scientific designed to facilitate transmission electron microscope control and related data processing tasks within a Python 3 language environment.

The package encompasses the Python 3 distribution with an interpreter and a selection of widely used packages like opencv, numpy or scipy. Most notably, it incorporates a set of packages specifically tailored for Thermo Fisher Scientific electron microscope control.

These include autoscript_tem_microscope_client with microscope control functions, and autoscript_tem_toolkit with domain-specific data processing functions such as fine-tuned algorithms for computer vision.

Another part of the installation is PyCharm; a Python integrated development environment with AutoScript customizations to enhance the ease of writing and debugging scripts that leverage the AutoScript API.

AutoScript TEM adopts a client-server architecture, with communication handled over TCP/IP. This enables clients to connect both locally and remotely, supporting simultaneous connections for multiple clients.
